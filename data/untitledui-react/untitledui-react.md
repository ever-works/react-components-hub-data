## Overview

Untitled UI React is a comprehensive component library with 300+ pre-built UI elements and sections. It provides ready-to-use components for both application interfaces and marketing websites.

## Key Features

- 300+ pre-built components (modals, forms, cards, etc.)
- Built with React Aria and TypeScript
- 180+ marketing page sections
- Fully customizable with Tailwind CSS
- 4.5K GitHub stars

## Installation

```bash
$ npm install @untitledui/react
``
