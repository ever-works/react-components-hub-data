## Overview

Flickity is a touch-friendly carousel library that focuses on smooth physics-based scrolling, making it a flexible option for different projects. It provides momentum-based interactions that feel native and responsive.

## Key Features

- Momentum-based scrolling for smooth, natural interactions
- Drag and flick feature with precise physics-based motion
- Lazy loading support for performance optimization
- Wrap-around (infinite) scrolling
- Supports slideshow components, galleries, and scrolling lists
- Custom slide positioning (center, left, right)
- Adaptive height adjustment based on slide content
- Used by over 14.1k developers
- 7.6k GitHub stars

## When to Use

- Projects requiring a mobile-like scrolling experience
- Image galleries, card-based layouts, and horizontal scrolling sections
- Applications where gesture-driven content navigation is a priority
