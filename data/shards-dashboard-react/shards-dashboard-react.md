## Overview

Shards Dashboard takes a content management and editorial-focused approach to admin dashboards, feeling more like a CMS backend than a generic admin panel.

## Features

- Blog/content management focused layout
- Content-centric navigation
- Publishing-oriented widgets
- Clean typography optimized for readability
- Modern design system emphasizing content hierarchy
- User profile pages
- Form components
- Data tables
