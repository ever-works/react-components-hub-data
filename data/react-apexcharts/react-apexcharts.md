## Overview

React ApexCharts is a React wrapper for ApexCharts, a modern JavaScript charting library with over 1.3K stars on GitHub. It features responsive designs, mobile-friendly charts, and built-in support for real-time data updates.

## Features

- 30+ chart types
- Real-time data updates
- Responsive and mobile-friendly
- Interactive charts with zoom, pan, and selection
- Candlestick charts for financial data
- Range charts
- Annotations and markers
- Multiple Y-axes
- Mixed chart types
- Data labels customization
- Legend customization
- Tooltip customization
- Export to PNG/SVG/CSV
- Dark mode support
- TypeScript definitions

## Chart Types

- Line, Area, Bar, Column charts
- Pie, Donut, Radial charts
- Scatter, Bubble charts
- Heatmap, Treemap
- Candlestick, Box plot
- Radar, Polar charts
- Range area, Range bar
- Timeline charts
