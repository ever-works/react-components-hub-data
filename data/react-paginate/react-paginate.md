## Overview

A ReactJS component to build pagination. Fully customizable with flexible styling options.

## Key Features

- Easy to integrate
- Customizable styling
- Multiple pages support
- Page range display
- Previous/next buttons
- Break labels
- Margin pages display
- Click handlers
- Active page highlighting
- Disabled state support

## Configuration

- pageCount: Total number of pages
- pageRangeDisplayed: Pages shown in range
- marginPagesDisplayed: Pages shown at edges
- onPageChange: Callback on page change
- initialPage: Starting page
