## Overview

React-accessible-treeview is a production-ready React component that implements the treeview pattern as described by the WAI-ARIA Authoring Practices. It provides comprehensive accessibility out of the box while remaining highly customizable.

## Features

- Full WAI-ARIA compliance for accessibility
- Customizable through render props pattern
- Prop getters for flexible composition
- Keyboard navigation support
- Screen reader optimization
- Expandable and collapsible nodes
- Single and multiple selection modes
- TypeScript support
- Custom node rendering

## Accessibility

- Proper ARIA roles and attributes
- Keyboard support (arrow keys, Enter, Space)
- Screen reader announcements
- Focus management
- Semantic HTML structure
