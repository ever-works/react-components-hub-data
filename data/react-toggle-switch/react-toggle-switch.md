## Overview

React Toggle Switch provides a flexible and accessible toggle component for React applications. It supports custom colors, sizes, and keyboard interactions.

## Features

- Customizable on/off colors and thumb size
- Keyboard accessibility (space/enter)
- Controlled and uncontrolled modes
- Theme-aware styling
- RTL (right-to-left) support

## Installation

`npm install react-toggle-switch`
