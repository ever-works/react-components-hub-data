## Overview

A syntax highlighting component for React. Uses Prism or Highlight.js under the hood with dynamic virtual DOM updates.

## Key Features

- Prism.js support
- Highlight.js support
- Multiple language support
- 30+ themes
- Inline styles
- Line highlighting
- Line numbers
- Custom styling
- Copy button
- Performance optimized
- TypeScript support

## Supported Languages

- JavaScript, Python, Java, C++
- HTML, CSS, SQL
- Markdown, JSON, XML
- And 100+ more
