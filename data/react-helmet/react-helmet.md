## Overview

React Helmet is the original library for managing document head in React applications. While still functional, React Helmet Async is recommended for new projects due to better SSR support and thread safety.

## Status

**Note:** For new projects, use React Helmet Async instead. This library is in maintenance mode.

## Features

- Manage document head
- Dynamic title
- Meta tags
- Link tags
- Script tags
- Style tags
- HTML attributes
- Body attributes
- Server-side rendering
- Nested components
- Declarative API
