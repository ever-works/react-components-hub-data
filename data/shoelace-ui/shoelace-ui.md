## Overview

Shoelace UI provides a collection of React components that prioritize simplicity, accessibility, and flexibility. It's designed to be framework-agnostic, allowing developers to use it with any UI framework.

## Key Features

- **Accessible by Default**: Compliant with W3C accessibility standards (WCAG AA/AAA).
- **Component Library**: Includes buttons, forms, modals, navigation menus, and more.
- **Custom Themes**: Easily inject custom themes using CSS variables.
- **Lightweight**: Minimal bundle size with tree-shakable components.
- **Community-Driven**: Actively maintained by a growing community.
