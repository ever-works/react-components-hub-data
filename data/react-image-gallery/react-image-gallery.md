## Overview

React Image Gallery is a flexible image gallery component with built-in lightbox support, thumbnail navigation, and customizable controls for creating professional image galleries.

## Key Features

- **Lightbox Support**: Full-screen image view
- **Thumbnail Navigation**: Click thumbnails to view
- **Keyboard Navigation**: Arrow keys to navigate
- **Slide Show**: Auto-play mode
- **Touch Support**: Swipe on mobile
- **Responsive**: Adaptive sizing
- **Custom Rendering**: Override components
- **TypeScript**: Type definitions

## Features

- Image carousel
- Thumbnail strip
- Lightbox mode
- Auto-play/slide show
- Keyboard navigation
- Touch/swipe support
- Customizable controls
- Loading indicators
- Error handling
- EXIF data support

## Configuration

- items: Image array
- autoPlay: Auto-play images
- lazyLoad: Lazy load images
- thumbnailPosition: Thumbnail location
- showPlayButton: Show/hide play button
- showThumbnails: Show/hide thumbnails
- isOpen: Lightbox open state
- showBullets: Show/hide indicators
