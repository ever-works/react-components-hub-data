## Overview

Rough.js is a lightweight graphics library that lets you draw in a sketchy, hand-drawn-like style. Built on HTML5 Canvas and SVG, it provides functions for drawing lines, curves, arcs, polygons, circles, and ellipses with a unique artistic look. It serves as the foundation for sketchy-styled charting libraries and is widely used for creating visually distinctive data visualizations.

## Features

- Hand-drawn, sketchy appearance for shapes
- Supports lines, curves, arcs, polygons, circles, and ellipses
- Works with both Canvas and SVG rendering
- Lightweight with no dependencies
- TypeScript definitions included
- Integrates with React, Vue, and vanilla JS/TS
- Used by many sketchy charting libraries (e.g., roughViz.js)
