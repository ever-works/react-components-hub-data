## Overview

Linaria is a zero-runtime CSS-in-JS library for React that compiles styles during the build process rather than at runtime. It enables developers to write CSS using JavaScript syntax while producing regular CSS output, eliminating the runtime overhead and hydration issues associated with traditional CSS-in-JS libraries like Styled Components.

## Features

- Zero-runtime CSS compilation
- Standard CSS syntax support within JavaScript
- Full server-side rendering compatibility
- Dynamic styling with CSS variables
- Theming support
- Works with Webpack, Rollup, Metro, and esbuild
- No hydration mismatches
- Compatible with React and other frameworks
