## Overview

react-remove-scroll is a component that removes page scroll while preserving the scrollbar gap to prevent layout shifts. It's perfect for modal dialogs and overlays that need to prevent background scrolling.

## Features

- **Scroll Lock**: Prevent background scrolling
- **No Layout Shift**: Preserve scrollbar width
- **iOS Support**: Works on iOS Safari
- **Nested Locks**: Multiple scroll locks
- **Touch Support**: Prevent touch scrolling
- **Accessibility**: Screen reader compatible
- **TypeScript**: Full type definitions
- **Lightweight**: Small bundle size

## Use Cases

- Modal dialogs
- Side drawers and panels
- Full-screen overlays
- Lightboxes and galleries
- Mobile menus
- Any component requiring scroll lock
