## Overview

The final form state management library for React. A subscription-based form library.

## Key Features

- Subscription-based updates
- Minimal re-renders
- Small bundle size
- Field-level subscriptions
- Form-level subscriptions
- Validation support
- Async validation
- TypeScript support
- Field-level errors
- Decorators support

## Core Concepts

- Form: Main component
- Field: Field subscription
- Subscription: Fine-grained updates
- Validation: Synchronous/asynchronous
