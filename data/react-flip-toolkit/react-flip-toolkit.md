## Overview

React FLIP Toolkit provides declarative animations for React using the FLIP (First, Last, Invert, Play) technique. It enables smooth, performant animations for layout changes, shared element transitions, and complex UI transformations.

## Features

- FLIP animations
- Shared element transitions
- Layout animations
- Enter/exit animations
- Staggered animations
- Spring physics
- Scale animations
- Route transitions
- Declarative API
- TypeScript support
- Minimal bundle size
- GPU-accelerated

## Core Components

- Flipper - animation container
- Flipped - animated element

## FLIP Technique

- First - measure initial state
- Last - measure final state
- Invert - calculate difference
- Play - animate transformation
