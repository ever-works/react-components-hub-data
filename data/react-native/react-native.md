## Overview

React Native is the go-to solution for bringing React from the web to mobile platforms. It allows developers to build native mobile applications for iOS and Android using React's declarative component model and JavaScript/TypeScript.

## Features

- Write once, run on both iOS and Android
- Native rendering using platform-specific components
- Fast Refresh for instant feedback during development
- Access to native APIs and modules
- Large ecosystem of community libraries
- Can be bridged with native code when needed
- Hot reloading and debugging tools
