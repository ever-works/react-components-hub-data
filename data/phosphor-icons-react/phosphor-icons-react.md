## Overview

Phosphor Icons is a flexible icon family designed for interfaces, diagrams, and presentations. With over 9,000 icons available in six different weight options, it provides unmatched versatility for establishing visual hierarchy and adapting to different design contexts.

## Features

- **Massive Library**: Over 9,000 icons across all categories
- **Six Weights**: Thin, Light, Regular, Bold, Fill, and Duotone
- **React Components**: Native React components for each icon
- **TypeScript**: Full TypeScript support
- **Tree-Shakeable**: Import only what you need
- **Customizable**: Size, color, and weight properties
- **Consistent Grid**: All icons designed on the same grid system
- **Context API**: Theme-based icon configuration

## Weight Options

1. **Thin**: Ultra-light, minimal aesthetic
2. **Light**: Subtle, refined appearance
3. **Regular**: Default, balanced weight
4. **Bold**: Strong, prominent presence
5. **Fill**: Solid, filled variants
6. **Duotone**: Two-tone depth and dimension

## Customization

- Size: Any pixel or rem value
- Color: CSS color values
- Weight: Switch between six weight variants
- Mirroring: Horizontal flip for RTL
