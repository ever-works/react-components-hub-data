## Overview

Baobab React provides React integration for Baobab, a persistent and immutable data structure. It enables components to access and react to changes in the state tree.

## Features

- Immutable persistent data structure
- Tree-based state management
- React component integration with cursors
- Automatic re-rendering on state changes
- Type checking support
- Historical state tracking (time-travel debugging)
