## Overview

Effector is a multi-store state manager with support for React bindings. It provides a reactive approach to managing application state with a focus on type safety and developer experience.

## Features

- Multi-store architecture for organized state management
- Built-in React hooks for consuming stores
- Type-safe API with TypeScript support
- Reactive primitives: stores, events, effects
- Easy testing and debugging
- SSR support
- No boilerplate or reducers required
