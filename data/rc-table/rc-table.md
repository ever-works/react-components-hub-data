## Overview

rc-table is the foundational table component maintained by the react-component organization, serving as the underlying table engine for Ant Design's Table component. It provides a robust, extensible table implementation for React.

## Features

- Virtual scrolling for large datasets
- Column grouping and nested headers
- Sticky columns and sticky headers
- Expandable rows with custom content
- Row selection with checkbox
- Column resizing and reordering
- Custom cell renderers
- Fixed columns support
- Summary/footer rows
- Empty text customization
- Row and cell event handling
- Accessibility support
- Tree data display
- CSS-in-JS theming support
