## Overview

Vega-Lite is a high-level visualization grammar that provides a concise JSON syntax for creating interactive visualizations. Built on top of Vega, it defaults to best practices for visual encoding and interaction, making it easier to create effective charts without detailed specifications.

## Features

- Concise JSON grammar for common chart types
- Automatic visual encoding and configuration
- Built-in support for: bar, line, point, area, circle, text, rect, boxplot, histogram, and more
- Composite views: layer, facet, repeat, and concat
- Selections for interactive filtering and highlighting
- Data transformation shortcuts (bin, aggregate, calculate, etc.)
- Compiles to Vega specifications
- Works with React and vanilla JS/TS
- TypeScript definitions included
