## Overview

React Masonry Component is a React wrapper for Masonry, the popular Cascading grid layout library by David DeSandro.

## Features

- Wraps the original Masonry library for React usage
- Cascading grid layout with column-based positioning
- Automatically positions elements based on available vertical space
- Supports imagesLoaded for proper layout after image loading
- Responsive and configurable column widths
- Compatible with Masonry's options API
