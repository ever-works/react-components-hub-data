## Overview

A feature-rich and customizable data grid React component. Fully featured and tree-shakable.

## Key Features

- Column resizing
- Column reordering
- Row selection
- Multi-select
- Frozen columns
- Frozen rows
- Inline editing
- Custom cell renderers
- Keyboard navigation
- Copy/paste support
- Grouping
- Aggregation
- Performance optimized

## Customization

- Custom cell formats
- Custom editors
- Custom sorters
- Custom filters
- Theming support
