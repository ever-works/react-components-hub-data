## Overview

SVAR React DataGrid is a free, open-source data grid component that balances simplicity with enterprise-grade features. Built for modern React applications, it provides virtual scrolling, editing capabilities, and a rich API.

## Features

- Virtual scrolling for rows and columns
- Inline and form-based editing
- Multi-column sorting
- Column filtering
- Column resizing and reordering
- Row selection (single and multi)
- Fixed columns
- Custom cell rendering
- Data export (CSV, Excel)
- Keyboard navigation
- Context menu support
- Grouping and aggregation
- Tree data support
- TypeScript support
- Responsive design
- Theme customization
- Rich API for programmatic control

## Performance

- Handles large datasets efficiently
- Virtual scrolling for optimal rendering
- Minimal re-renders
- Smooth scrolling experience

## Editing Features

- Inline cell editing
- External form editing
- Validation support
- Batch updates
- Undo/redo functionality

## Data Operations

- Client-side and server-side data
- Lazy loading support
- CRUD operations
- Real-time data updates
- Custom data adapters

## Use Cases

- Admin dashboards
- Data management applications
- Report builders
- CRM systems
- Analytics platforms
- Inventory management
