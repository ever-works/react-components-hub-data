## Overview

React Docgen is a CLI and toolbox to help extract information from React components and generate documentation. It parses component files to extract props, types, and comments.

## Features

- **PropTypes Extraction**: Automatically extract component props
- **TypeScript Support**: Parse TypeScript component definitions
- **JSDoc Integration**: Extract documentation from comments
- **Flow Support**: Works with Flow type annotations
- **Default Props**: Captures defaultProps definitions
- **Custom Handlers**: Extensible with custom parsers
- **Multiple Components**: Handle multiple components per file

## Use Cases

- Automatic documentation generation
- Component library documentation
- API reference generation
- Storybook integration
- Design system documentation
