## Overview

React Rich Text Editor is a feature-rich WYSIWYG editor for React applications. It supports markdown formatting, collaborative editing, and real-time preview capabilities.

## Features

- Drag-and-drop file insertion
- Real-time collaborative editing
- Markdown and HTML export
- Customizable toolbar
- Offline storage support
- Accessible keyboard shortcuts

## Installation

`npm install react-rich-text-editor`
