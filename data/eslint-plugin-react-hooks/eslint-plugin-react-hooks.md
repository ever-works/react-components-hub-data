## Overview

ESLint Plugin React Hooks is the official ESLint plugin that enforces the Rules of Hooks. It catches common mistakes in hook usage and ensures hooks are called correctly.

## Features

- **Rules of Hooks**: Enforce hook calling rules
- **Exhaustive Deps**: Check useEffect/useMemo dependencies
- **Auto-fix**: Automatic dependency fixing
- **Custom Hooks**: Support for custom hook patterns
- **TypeScript**: Works with TypeScript
- **Official**: Maintained by React team
- **Essential**: Recommended for all hook usage

## Use Cases

- Preventing hook-related bugs
- Enforcing correct dependency arrays
- Maintaining code quality
- Teaching proper hook usage
- Required for production React apps
