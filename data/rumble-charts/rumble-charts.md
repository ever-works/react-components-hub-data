## Overview

Rumble-Charts is a lightweight, flexible, and easy-to-use chart library for React applications based on D3.js. It provides simple, customizable chart components with a declarative API.

## Features

- Built on top of D3.js for powerful data visualization capabilities
- Lightweight and easy to integrate into React projects
- Flexible component-based API for creating custom charts
- Supports common chart types including bars, pies, lines, and dots
- Declarative approach to chart building in React
