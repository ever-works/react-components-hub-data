## Overview

react-docgen-typescript is a tool for extracting documentation from TypeScript React components. It provides full TypeScript support and generates detailed component documentation including props, types, and comments.

## Features

- **Full TypeScript Support**: Native TypeScript parsing
- **Interface & Type Extraction**: Document all TypeScript types
- **JSDoc Comments**: Include inline documentation
- **Generic Components**: Support for generic types
- **Union Types**: Handle complex type definitions
- **Storybook Integration**: Official Storybook addon
- **Custom Props**: Filter and customize output

## Use Cases

- TypeScript component documentation
- Storybook documentation addon
- Design system documentation
- API reference for component libraries
- Type-safe prop documentation
