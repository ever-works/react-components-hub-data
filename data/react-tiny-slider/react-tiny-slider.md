## Overview

React Tiny Slider is a React wrapper around Tiny Slider, providing a lightweight carousel component ideal for building image galleries and content sliders.

## Key Features

- **Lightweight**: Minimal bundle size
- **Responsive**: Mobile-friendly design
- **Touch Support**: Swipe gestures
- **Keyboard Navigation**: Arrow key support
- **Lazy Loading**: Load images on demand
- **Multiple Items**: Show multiple items per slide
- **Auto Play**: Automatic slide advancement
- **Callbacks**: Slide change events
- **TypeScript**: Type support

## Configuration Options

- items: Number of visible items
- speed: Slide transition speed
- autoplay: Enable auto-play
- autoplayTimeout: Auto-play delay
- responsive: Mobile breakpoints
- controls: Show/hide navigation
- nav: Show/hide indicators
- touch: Enable touch support
- lazyload: Lazy load images
