## Overview

UnoCSS is an instant, highly customizable atomic CSS engine that is inspired by Tailwind CSS and Windi CSS. It provides a fast, on-demand alternative to traditional CSS frameworks, generating only the CSS needed for each project.

## Features

- On-demand CSS generation for minimal bundle sizes
- Highly customizable rules and presets
- Compatible with Tailwind CSS utilities
- Works with Vite, Webpack, and other build tools
- Custom presets for design system consistency
- Fast build times
- Supports React, Vue, and other frameworks
