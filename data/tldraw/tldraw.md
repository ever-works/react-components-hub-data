## Overview

tldraw is a free, open-source collaborative whiteboard built for developers and designers. It supports real-time collaboration and provides a flexible canvas for creating diagrams, wireframes, and sketches. It offers a React SDK for embedding the whiteboard directly into applications.

## Features

- Collaborative whiteboarding with real-time multiplayer
- Embeddable React component via @tldraw/tldraw package
- Shape tools including rectangles, ellipses, arrows, and freehand drawing
- Sticky notes and text annotations
- Export to PNG and SVG formats
- Auto-layout tools for organizing content
- Persistent canvas state with multi-user sessions
- Open-source SDK for custom shape and tool creation

## Pricing

Free and open-source under the tldraw license. Cloud-hosted multiplayer requires a subscription for production use.
