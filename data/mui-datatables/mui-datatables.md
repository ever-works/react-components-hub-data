## Overview

MUI-datatables is a data tables component built on Material-UI v4/v5 for React. It provides features such as filtering, searching, sorting, and export with a clean, customizable interface.

## Features

- Server-side processing support
- Column visibility toggle
- CSV and print data export
- Responsive layout (stacked, simple, full width)
- Customizable search and filter behavior
- Row selection with checkbox
- Custom column rendering
- Pagination with customizable rows per page
- Download and print functionality
- Theme customization via Material-UI
- Fixed header and column options
- Expandable rows with custom content
- Custom toolbars and actions
