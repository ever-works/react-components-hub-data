## Overview

Split Pane React provides resizable split panes for React applications. It supports both horizontal and vertical layouts with draggable dividers, allowing users to resize panels to their preference.

## Features

- Horizontal and vertical split layouts
- Draggable resize dividers
- Multiple panel support
- Nested split panes
- Minimum and maximum size constraints
- Customizable divider styles
- Different themes available
- Controlled component mode
- Fluid pane support
- Touch-friendly
- Keyboard accessible
- TypeScript support
- React 16.8+ compatible
- Live demo available
- Responsive behavior

## Layout Options

- Vertical split (left/right)
- Horizontal split (top/bottom)
- Combined layouts with nesting
- Multiple panels in sequence

## Use Cases

- Code editors (editor + preview)
- File explorers (tree + content)
- Dashboard layouts
- Admin panels
- Data comparison views
- Multi-panel applications
- IDE-like interfaces
