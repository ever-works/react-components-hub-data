## Overview

Vivus is a lightweight JavaScript class that animates SVG elements to give the appearance of being drawn in real-time. It supports multiple animation types and provides fine-grained control over the drawing process.

## Features

- Three animation types: delayed (sequential path drawing), async (all paths drawn simultaneously), and oneByOne (each path completes before the next begins)
- Path timing control — paths are animated based on their length
- Custom easing functions including Ease, Ease-in, Ease-out, and Ease-in-out
- Callback support for animation completion
- Lightweight — minimal footprint
- Works with inline SVGs, object tags, and external SVG files
- Force rendering option for precise control

## Best Use Case

Creating drawing/hand-drawn style animations for SVG logos, icons, and illustrations.
