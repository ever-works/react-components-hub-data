## Overview

Zenith takes an ultra-minimal achromatic approach to dashboard design, proving that restraint can be a feature. The black-and-white palette creates a dashboard that adapts to any brand without visual conflict.

## Features

- 50+ routes
- Six dashboard variants: Overview, Analytics, eCommerce, CRM, SaaS, Finance
- 34 shadcn/ui components
- AI Chat page
- Kanban board
- Calendar integration
- Wizard component
- TipTap 3 editor
- Leaflet maps
- CRUD pages
- Project management pages
- Three density levels for data-heavy or spacious layouts
- Storybook documentation for component exploration
- React + Next.js + shadcn/ui
