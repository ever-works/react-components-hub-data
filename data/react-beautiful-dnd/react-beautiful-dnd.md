## Overview

React Beautiful DnD is a beautiful drag-and-drop library built by Atlassian that makes it easy to create delightful drag-and-drop experiences with accessibility built-in from the ground up.

## Key Features

- **Beautiful Animations**: Smooth, performant animations
- **Accessible**: Built-in keyboard navigation and ARIA support
- **Touch Support**: Works on mobile and touch devices
- **Performance**: Optimized rendering with minimal DOM updates
- **Flexible API**: Covers simple to complex use cases
- **Automatic Scrolling**: Intelligent scroll behavior
- **Multi-Select Support**: Select multiple items
- **TypeScript**: Full type definitions

## Available Components

- **Droppable**: Defines drop zones
- **Draggable**: Defines draggable items
- **DragDropContext**: Root context for drag-drop

## Features

- Drag Between Lists
- Drag Within List
- Multi-List Dragging
- Virtualization Support
- Custom Animations
- Scroll Containers
- Snap to Grid

## Accessibility

- Full keyboard navigation
- Screen reader announcements
- ARIA attributes
- Focus management
- Tested with popular screen readers

## Use Cases

- Todo List Sorting
- Kanban Boards
- Priority Lists
- File Organization
- Menu Organization
- Timeline Events
