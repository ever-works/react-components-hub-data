## Overview

Waku is a lightweight React framework that provides first-class support for React Server Components (RSC). Created by the same developer behind Zustand, it follows a minimalist philosophy while enabling server-first React development.

## Features

- First-class React Server Components (RSC) support
- Minimal configuration and setup
- File-based routing
- Server functions support
- Lightweight compared to Next.js
- Created by the Zustand developer
