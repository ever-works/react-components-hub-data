## Overview

Downshift is a tiny, primitive React hook that abstracts away the logic of selecting a single item, managing state, handling keyboard navigation, and managing focus for dropdown, select, and autocomplete components.

## Key Features

- **Unstyled**: No styling opinions, full customization
- **Accessible**: Follows ARIA patterns for dropdowns
- **Keyboard Navigation**: Full keyboard support
- **Small Bundle Size**: Minimal overhead
- **Headless**: Logic without UI
- **TypeScript Support**: Fully typed
- **Testing Utilities**: Helpers for testing

## Use Cases

- Select Components: Single item selection
- Autocomplete: Search with filtered suggestions
- Combobox: Combined input and list
- Command Palette: Searchable command interface
- Tag Input: Multiple item selection

## Hook API

- useSelect: For select/dropdown
- useCombobox: For combobox/autocomplete
- useMultipleSelection: For multi-select

## Advanced Features

- Item highlighting and selection
- Keyboard navigation (arrow keys, enter, escape)
- Dynamic items list
- Custom item rendering
- Event callbacks for all interactions
