## Overview

React Fast Marquee provides a smooth, performant marquee component with true infinite scrolling. It's perfect for logo displays, news tickers, and any continuously scrolling content.

## Features

- Smooth infinite scroll
- No layout shift
- Pause on hover
- Pause on click
- Gradient edges
- Configurable speed
- Direction control
- Loop behavior
- Clone children for seamless loop
- Responsive
- TypeScript support
- Lightweight
- GPU accelerated

## Configuration

- speed - scroll speed (px/s)
- delay - initial delay
- direction - left or right
- loop - number of loops (0 = infinite)
- gradient - show gradient edges
- gradientColor - gradient color
- gradientWidth - gradient width
- pauseOnHover - pause on mouse hover
- pauseOnClick - pause on click
- play - play/pause control
- autoFill - auto-fill with clones
