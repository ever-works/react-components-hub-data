## Overview

Shadcn Admin Kit is a React admin dashboard toolkit built on top of shadcn/ui components and the React Admin framework. It provides a solid foundation for building admin interfaces with a Kanban board, CRUD operations, and configurable data providers.

## Features

- Combines shadcn/ui components with React Admin (ra-core) framework
- Pre-built admin layout with navigation and theming
- Supports CRUD operations for resources
- Kanban board view with drag-and-drop using @hello-pangea/dnd
- Compatible with ra-data-fakerest for mock data during development
- Custom hooks for organizing data (e.g., useTodosByStatus for grouping records)
- Modal-based create and edit forms
- Optimistic UI updates with drag-and-drop reordering
- TypeScript support with path aliases
- Tailwind CSS for component styling
- Dark mode support
- Vite-based project setup with fast development and optimized builds
