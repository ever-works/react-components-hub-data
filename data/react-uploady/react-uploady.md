## Overview

Modern file uploading - components & hooks for React. A set of components and hooks for handling file uploads in React applications.

## Key Features

- Multiple upload components
- Drag and drop support
- Progress tracking
- File preview
- Chunk upload
- Cancellable uploads
- TypeScript support
- Hooks-based API
- Accessible
- Mobile-friendly

## Main Components

- Uploady: Main provider
- UploadButton: File input
- UploadDropZone: Drag-drop area
- UploadPreview: File preview
- useUploady: Hook for upload logic
