## Overview

Radix Context Menu is a headless, accessible context menu component that displays at the pointer location when triggered by right-click or long press. It provides all the functionality without any styling opinions.

## Features

- Triggered by right-click or long press
- Fully accessible with ARIA
- Keyboard navigation support
- Submenus with hover delays
- Checkbox menu items
- Radio group menu items
- Optional portal rendering
- Custom item components
- Collision detection
- Focus management
- Typeahead support
- Disabled items
- Separator support
- Completely unstyled
- TypeScript support

## Composition

- Root, Trigger
- Portal, Content
- Item, CheckboxItem, RadioItem
- Sub, SubTrigger, SubContent
- Separator, Label

## Use Cases

- File manager actions
- Text editor menus
- Image manipulation
- Table row actions
- Canvas element menus
- Custom application menus
