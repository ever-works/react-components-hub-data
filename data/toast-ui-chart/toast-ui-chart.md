## Overview

TOAST UI Chart is a beautiful charting library developed by NHN for data visualization. It provides a comprehensive set of interactive, responsive charts built with SVG for crisp rendering. The library features a clean design and is easy to integrate into web applications.

## Features

- Wide variety of chart types: line, area, bar, column, pie, bubble, heatmap, treemap, radial, boxplot
- SVG-based rendering for sharp visuals
- Responsive and mobile-friendly
- Interactive tooltips and hover effects
- Animation support for chart transitions
- Export chart as image
- Multiple axes support
- Data grouping options
- Works with React, Vue, and vanilla JS
