## Overview

React Markdown is a component for rendering markdown content using React. It's built on top of remark, a popular markdown processor, and allows for flexible customization of how markdown is rendered.

## Key Features

- **Safe by Default**: Sanitizes HTML by default
- **Plugin Support**: Extend with remark plugins
- **Custom Renderers**: Override how elements are rendered
- **No DOM Dependencies**: Pure React component
- **TypeScript Support**: Fully typed
- **CommonMark Spec**: Follows markdown standard
- **Syntax Highlighting**: Support for code blocks
- **Table Support**: Markdown tables

## Custom Renderers

- heading: Custom heading rendering
- paragraph: Custom paragraph rendering
- code: Custom code block rendering
- link: Custom link rendering
- image: Custom image rendering
- list: Custom list rendering
- listItem: Custom list item rendering
- table: Custom table rendering
