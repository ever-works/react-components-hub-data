## Overview

RevoGrid is a fast and lightweight data grid designed to handle millions of rows efficiently through virtualization, offering spreadsheet-like features for modern web applications.

## Features

- Virtualization for handling millions of rows
- Custom cell templates
- Editable cells
- High-performance rendering
- Lightweight footprint
- Spreadsheet-like interactions
- Supports large datasets
