## Overview

Base UI is an unstyled React component library from MUI that provides the essential building blocks for creating accessible, performant web interfaces. It focuses on providing the logic and accessibility patterns while leaving styling to you.

## Features

- Fully unstyled for maximum customization
- WCAG 2.1 Level AA compliant
- Built-in focus management and keyboard navigation
- Full TypeScript support
- Hooks-based API for headless development
- Small bundle size
- No CSS dependencies
- Customizable slot system
- Extensive documentation

## Components Included

- Accordion
- Alert
- Autocomplete
- Button
- Dialog
- Dropdown
- Form Controls
- Menu
- Modal
- Popper
- Select
- Slider
- Tabs
- Textarea
- And more

## Installation

```bash
npm install @mui/base
```
