## Overview

RedwoodJS is a full-stack React framework that brings together the best practices and tools for modern web development. It uses GraphQL for the API layer, Prisma for database management, and is designed for serverless deployment on platforms like Netlify and Vercel.

## Features

- Full-stack React framework with GraphQL API
- Prisma integration for database management
- Serverless-ready architecture
- Built-in Jest and Storybook support
- File-based routing with cells for data fetching
- Role-based authentication
- TypeScript first-class support
- Automatic API schema generation
