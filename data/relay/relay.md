## Overview

Relay is a JavaScript framework for building data-driven React applications, developed by Meta. It provides a sophisticated approach to fetching and managing GraphQL data with features optimized for large-scale applications including automatic caching, garbage collection, and pagination.

## Features

- GraphQL-first data fetching
- Automatic query coalescing and deduplication
- Built-in pagination patterns
- Data normalization and garbage collection
- Strong type system with generated Flow/TypeScript types
- Automatic caching and stale-while-revalidate patterns
- Mutations with optimistic updates
- Built for Facebook-scale applications

## Pricing

Free and open-source, developed by Meta.
