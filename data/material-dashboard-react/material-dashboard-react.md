## Overview

Material Dashboard React is Creative Tim's flagship free React dashboard and one of the most downloaded admin templates. It delivers a clean Material Design aesthetic throughout.

## Features

- 70+ frontend elements including buttons, inputs, navbars, cards, and alerts
- Seven example pages
- Dynamic chart components
- Sass-based theming for customizing colors and typography through variables
- Material Design aesthetic throughout
- Thorough documentation
- Proven production track record
