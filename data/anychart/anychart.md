## Overview

AnyChart is a flexible JavaScript charting library that works with React, Angular, Vue, Svelte, and vanilla TypeScript or JavaScript. It provides a comprehensive set of chart types and customization options for interactive data visualization.

## Features

- Supports React, Angular, Vue, Svelte, and vanilla JS/TS
- Wide variety of chart types including line, bar, pie, scatter, and more
- Interactive tooltips, crosshairs, and zoom controls
- Built-in data export capabilities
- Responsive design for different screen sizes
- TypeScript definitions included
