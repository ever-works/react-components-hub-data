## Overview

@monaco-editor/react is a React wrapper for the Monaco Editor, the code editor that powers VS Code. It provides a rich editing experience with syntax highlighting, IntelliSense, and customization without complex build configuration.

## Features

- IntelliSense (code completion)
- Syntax highlighting for 100+ languages
- Code folding and minimap
- Find and replace
- Multi-cursor editing
- Diff editor mode
- Customizable themes
- TypeScript support
- Real-time validation
- Bracket matching
- Code formatting
- No webpack configuration needed
- SSR compatible
- Diff viewer
- Custom language support

## Editor Modes

- Standard editor
- Diff editor (side-by-side comparison)
- Read-only mode

## Use Cases

- Code playgrounds
- Online IDEs
- Documentation with live code
- Code review tools
- Configuration editors
- SQL/query builders
- JSON/YAML editors
