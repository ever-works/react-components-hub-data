## Overview

Pure React Carousel is a fully controlled React carousel component that provides maximum flexibility in component assembly and styling. It's a pure React solution with no external JavaScript library dependencies.

## Features

- **100% Test Coverage**: Thoroughly tested and reliable
- **Fully Controlled**: Complete control over carousel state
- **Flexible DOM Order**: Assemble components in any order
- **Pure React**: No jQuery or external libraries
- **Responsive**: Adapts to container size
- **Customizable**: Add any classes, props, or styles
- **TypeScript**: Full TypeScript support
- **Accessible**: Keyboard navigation and ARIA support

## Core Components

- **CarouselProvider**: Wrapper component with required props
- **Slider**: Contains all slide components
- **Slide**: Individual slide component
- **ButtonBack**: Previous slide button
- **ButtonNext**: Next slide button
- **Dot**: Pagination dot component
- **DotGroup**: Container for pagination dots
- **Image**: Lazy-loading image component

## Required Props

- `totalSlides`: Number of slides in carousel
- `naturalSlideWidth`: Natural width of slides
- `naturalSlideHeight`: Natural height of slides

## Advanced Features

- Touch and swipe support
- Keyboard navigation
- Infinite loop mode
- Vertical orientation
- Variable slide widths
- Custom step intervals
- Drag enabled/disabled
- Play direction control

## Performance

Optimized for performance with minimal re-renders and efficient state management. Works well with large numbers of slides.
