## Overview

TanStack Query is a powerful asynchronous state management, server-state utilities and data fetching library. It works with TypeScript/JavaScript, React, Solid, Svelte and Vue.

## Features

- Server-state synchronization
- Automatic caching and deduplication
- Optimistic updates
- Mutation support
- Pagination and infinite scroll
- Query invalidation
- DevTools integration
- SSR support
- Automatic refetching
- Retry logic
- TypeScript support
