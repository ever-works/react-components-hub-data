## Overview

react-drag-drop-files is a lightweight React library for handling file uploads through drag-and-drop or traditional file selection. It uses styled-components for styling and offers flexible configuration options.

## Features

- Drag and drop file upload
- Click to browse files
- File type validation
- File size limits
- Multiple file support
- Custom styling with styled-components
- Disabled state
- Custom labels and messages
- File preview support
- Error handling
- TypeScript support
- Lightweight package
- Touch-friendly
- Accessible design

## Validation Options

- Accepted file types
- Maximum file size
- Multiple file restrictions
- Custom validation functions

## Use Cases

- Image uploads
- Document uploads
- Media file handling
- Attachment features
- Profile picture uploads
- Bulk file uploads
- Form file inputs
