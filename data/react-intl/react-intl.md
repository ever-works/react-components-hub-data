## Overview

React Intl is part of FormatJS, providing React components and an API to internationalize React applications. It handles formatting of dates, numbers, and strings, including pluralization and translations.

## Features

- **ICU Message Syntax**: Powerful message formatting
- **Number/Date Formatting**: Locale-aware formatting
- **Pluralization**: Handle plural forms correctly
- **Components**: Declarative formatting components
- **Hooks API**: Modern hooks-based API
- **TypeScript**: Full type support
- **Message Extraction**: CLI tools for extraction
- **200+ Locales**: Extensive locale support

## Use Cases

- Multi-language applications
- Global products
- Date and number localization
- Currency formatting
- Plural forms handling
- RTL language support
