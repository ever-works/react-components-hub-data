## Overview

Rockpack is a zero-configuration solution for creating React applications with server-side rendering, bundling, linting, and testing.

## Features

- Zero-configuration setup
- SSR support out of the box
- Development server with HMR
- Production builds with code splitting
- Built-in linting
- Ready-made testing setup
- Webpack optimizations for React
