## Overview

Volt brings Bootstrap 5 to React with over 100 custom-styled components, providing a familiar foundation for Bootstrap teams with React component abstractions.

## Features

- 100+ custom-styled Bootstrap 5 components
- Date pickers
- Modal components
- Tabs
- Navigation bars
- Prebuilt dashboard pages
- Chart widgets
- Notification system
- Sass variables and mixins for customization
- Professional theming over base Bootstrap
- Consistent design quality throughout
