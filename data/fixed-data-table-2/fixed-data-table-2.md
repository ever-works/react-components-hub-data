## Overview

fixed-data-table-2 is an active community fork of the original Facebook fixed-data-table. It uses virtual rendering to efficiently display large datasets while maintaining a consistent set of DOM nodes for performance.

## Features

- Virtual row rendering for large datasets
- Fixed headers and columns
- Column resizing and reordering
- Scrollable data regions
- Custom cell renderers
- Row heights customization
- Multi-header support
- RTL layout support
- Keyboard navigation
- Frozen columns (left/right)
- Performance optimized for thousands of rows
