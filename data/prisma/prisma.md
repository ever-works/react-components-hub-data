## Overview

Prisma is a next-generation ORM that provides a type-safe database client for popular databases including PostgreSQL, MySQL, SQLite, and MongoDB. It is the most popular ORM choice for full-stack React applications.

## Features

- Declarative data modeling with Prisma schema
- Auto-generated TypeScript types from schema
- Intuitive, type-safe query builder
- Built-in migration system
- Support for PostgreSQL, MySQL, SQLite, MongoDB, SQL Server
- Prisma Studio for database browsing
- Works well with Next.js and other React frameworks

## Pricing

Open-source core is free. Prisma Data Platform (cloud offering) has a free tier and paid plans.
