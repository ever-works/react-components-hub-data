## Overview

Mailing is a framework for building and sending emails using React components. It provides a development workflow with local previewing, automated email screenshot testing, and seamless integration for building production-ready email templates.

## Features

- Build emails with React components
- Local previews and dev server
- Automated screenshot testing
- Template management
- Easy integration with any email service
- Hot reloading during development
