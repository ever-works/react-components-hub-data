## Overview

Zustand provides an intuitive API for managing global state in React, using plain JavaScript objects and functions.

## Key Features

- **No Context Required**: Manages state directly without React Context API.
- **Middlewares**: Supports middleware extensions like persist, devtools, and hydration.
- **Selectors**: Optimized re-renders with computed selectors.
- **TypeScript Support**: Built-in type declarations for type-safe state management.
