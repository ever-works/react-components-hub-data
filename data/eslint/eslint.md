## Overview

ESLint is the standard linting tool for JavaScript and TypeScript projects. In React projects, it is typically configured with the ESLint Plugin React and ESLint Plugin React Hooks to enforce React-specific rules and patterns. It integrates with IDEs to highlight issues in real-time during development.

## Features

- Enforces consistent code style across a codebase
- Follows popular style guides like Standard, AirBnB, and more
- React-specific linting rules via eslint-plugin-react
- Integration with IDEs for real-time feedback
- Auto-fixes many common issues automatically
- Extensible via custom plugins and rules
- Works with TypeScript via @typescript-eslint
