## Overview

Bytescale Upload Widget is a complete file upload solution that includes cloud storage out of the box. It handles everything from upload to storage to delivery, making it perfect for production applications.

## Features

- Built-in cloud storage (storage included)
- Image cropping and editing
- Video previews and thumbnails
- Document previews (PDF, Office files)
- Drag and drop upload
- Multi-file uploads
- Direct browser-to-cloud uploads
- Image optimization and resizing
- Automatic format conversion (WebP, AVIF)
- CDN delivery included
- File validation and virus scanning
- Upload progress tracking
- Folder organization
- Public and private files
- Expiring URLs
- Custom branding
- Mobile optimized
- Accessibility compliant

## Supported File Types

- Images (JPEG, PNG, GIF, WebP, SVG, etc.)
- Videos (MP4, MOV, AVI, WebM, etc.)
- Documents (PDF, DOC, XLS, PPT, etc.)
- Audio files
- Archives (ZIP, RAR, etc.)
- Raw files
