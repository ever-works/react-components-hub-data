## Overview

Motion-Primitives is an open-source UI kit that provides beautiful, pre-built animated components for React and Next.js applications. It's designed to help developers create stunning animations faster with minimal effort.

## Features

- **Pre-built Components**: Ready-to-use animated UI components
- **React & Next.js**: Built specifically for modern React frameworks
- **Tailwind CSS**: Seamless integration with Tailwind
- **Copy & Paste**: Simply copy components into your project
- **Customizable**: Easy to modify and adapt to your design
- **TypeScript**: Full TypeScript support
- **Regular Updates**: Actively maintained and updated
- **Performance**: Optimized animations using best practices

## Component Categories

- Animated cards and containers
- Hero sections with motion
- Background effects
- Text animations
- Button interactions
- Loading states
- Transition effects
- Scroll-triggered animations

## Animation Features

- Smooth transitions
- Gesture-based interactions
- Scroll-linked animations
- Entrance and exit animations
- Hover effects
- Layout animations
- Spring physics

## Use Cases

- Landing pages
- Marketing websites
- SaaS applications
- Portfolio sites
- Product showcases
- Interactive dashboards
- Onboarding flows

## Integration

Components are designed to be copied directly into your codebase, allowing full customization and ownership of the code. Works seamlessly with Framer Motion under the hood.
