## Overview

Day.js is a minimalist JavaScript library that provides a fast and efficient API for parsing, validating, manipulating, and displaying dates and times. With a tiny 2 KB bundle size, it is a popular choice for React applications that need date handling without the bloat of Moment.js.

## Features

- Tiny bundle size (~2 KB)
- Immutable API for predictable date operations
- Extensible via a plugin system
- Chained date manipulation methods
- Relative time formatting and duration support
- Locale support for internationalization
- Drop-in replacement for Moment.js in most cases
- Zero dependencies
