## Overview

React Responsive Carousel is a powerful, responsive carousel component for React applications. It offers dynamic adaptation to different screen sizes and excellent performance with lazy loading and hardware acceleration.

## Features

- **Fully Responsive**: Adapts seamlessly to all screen sizes
- **Lazy Loading**: Improved performance with large galleries
- **Hardware Accelerated**: Smooth animations using CSS transforms
- **Touch Support**: Swipe gestures on mobile devices
- **Keyboard Navigation**: Arrow key support
- **Thumbnails**: Optional thumbnail navigation
- **Auto-play**: Built-in autoplay functionality
- **Customizable**: Extensive styling options

## Events & Callbacks

- onChange: Fired when slide changes
- onClickItem: When item is clicked
- onClickThumb: When thumbnail is clicked
- onSwipeStart: When swipe begins
- onSwipeEnd: When swipe ends
- onSwipeMove: During swipe movement

## Performance

Hardware-accelerated transitions ensure smooth 60fps animations. Lazy loading reduces initial load time and improves performance with large image sets.
