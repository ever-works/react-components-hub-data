## Overview

React Draft Wysiwyg is a WYSIWYG editor built using ReactJS and DraftJS libraries, providing a rich text editing experience with standard formatting capabilities.

## Features

- Built on ReactJS and DraftJS
- Inline formatting options (bold, italic, underline, strikethrough, monospace)
- Block-level formatting (headings, lists, blockquotes, code blocks)
- Text alignment options
- Font size and font family selectors
- Color pickers for text and background
- Emoji support
- Image upload and embedding
- Link insertion
- History support (undo/redo)
- Configurable toolbar
- Customizable styling
- HTML export
