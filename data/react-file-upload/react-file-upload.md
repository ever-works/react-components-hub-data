## Overview

React File Upload is a modern file upload component for React applications that supports drag-and-drop, multiple file selection, and real-time progress tracking.

## Features

- Drag-and-drop file selection with visual feedback
- Multiple file upload support with individual progress
- File type and size validation
- Customizable upload endpoints
- Thumbnail previews for images
- Server-side upload handling
