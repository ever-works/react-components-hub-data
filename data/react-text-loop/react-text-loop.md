## Overview

React Text Loop provides an animated component that cycles through an array of text strings with smooth transitions. It's ideal for creating dynamic headlines, rotating testimonials, or any rotating text content.

## Features

- Text rotation
- Smooth transitions
- Configurable speed
- Multiple transition effects
- Pause on hover
- Custom delay
- Spring physics
- Mask overflow
- No fade effect option
- Height adjustment
- Custom elements
- TypeScript support

## Transition Effects

- Slide up
- Slide down
- Fade
- Custom transitions
- Spring animations

## Configuration

- children - text array
- interval - rotation interval
- delay - initial delay
- springConfig - spring physics
- fade - fade effect
- mask - mask overflow
- noWrap - prevent wrapping
- adjustingSpeed - height adjustment speed
