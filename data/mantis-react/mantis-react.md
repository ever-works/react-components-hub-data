## Overview

Mantis takes an unusual approach by blending Material UI components with Ant Design's design philosophy, offering the best visual elements from both systems. It is built by CodedThemes, the same team behind Berry.

## Features

- Material UI components with Ant Design design philosophy
- Redux Toolkit state management
- ApexCharts for data visualization
- Customizable color presets for brand alignment
- Thorough TypeScript support
- Clean component architecture for scalability
- Enterprise-focused design direction
- Unique visual direction combining MUI and Ant Design aesthetics
