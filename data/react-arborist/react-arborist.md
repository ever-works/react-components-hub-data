## Overview

React Arborist is the complete tree view component for React. It provides a production-ready solution for displaying hierarchical data structures with full support for interactions and customization.

## Features

- Complete tree view implementation with parent-child relationships
- Drag-and-drop support for reordering nodes
- Multi-select and single-select modes
- Efficient rendering of thousands of items with virtual scrolling
- Keyboard navigation and accessibility features
- Customizable node rendering
- Open/close folder states
- Inline editing capabilities
- Search and filter functionality
- TypeScript support

## Use Cases

- File browsers and explorers
- Directory structures
- Navigation sidebars
- Layer panels for design tools
- Menu hierarchies
- Organizational charts
