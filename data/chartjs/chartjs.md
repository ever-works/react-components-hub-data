## Overview

Chart.js provides a set of frequently used chart types, plugins, and customization options. In addition to built-in chart types, community-maintained chart types are available for extended functionality.

## Features

- Multiple built-in chart types (line, bar, radar, pie, doughnut, polar area, bubble, scatter, and more)
- Highly customizable with custom plugins
- Annotation, zoom, and drag-and-drop functionality via plugins
- Community-maintained chart types for additional variety
- Responsive canvas-based rendering
- HTML5 Canvas rendering
- Mix and match bar and line charts to provide a clear visual distinction between datasets
