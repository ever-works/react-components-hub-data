## Overview

NextAuth.js provides a simple authentication solution for Next.js applications with support for social logins, email authentication, and database integration.

## Key Features

- **OAuth Providers**: Pre-configured support for Google, GitHub, Facebook, and 100+ other providers.
- **Email Sign In**: Passwordless email-based authentication with verification tokens.
- **Database Adapters**: Connect to MySQL, PostgreSQL, MongoDB, or custom databases.
- **Session Management**: Secure cookie-based session storage with automatic renewal.
- **TypeScript Support**: Full type definitions for authentication state and user profiles.
