## Overview

TanStack Form is a modern form library that brings the same level of innovation to form handling that TanStack Query brought to data fetching. It provides a headless, framework-agnostic approach to building forms with first-class TypeScript support.

## Features

- Headless UI - bring your own components
- First-class TypeScript support with full type inference
- Framework agnostic (React, Vue, Solid, Angular)
- Built-in validation with adapters for Zod, Yup, Valibot
- Async validation support
- Field-level and form-level validation
- Dependent field validation
- Array and nested object support
- Optimized re-renders - only affected fields update
- Transform and parse field values
- Custom validation messages
- Submission handling with loading states
- Dirty state tracking
- Touch/blur tracking
- Reset functionality

## Key Advantages

- Zero dependencies
- Tiny bundle size (~15kb)
- Excellent performance with minimal re-renders
- Works with any UI library
- Comprehensive TypeScript support
- Active development by TanStack team
