## Overview

React SaaS Template provides both a dashboard and a complete marketing site in a single project, eliminating the need to match the landing page to the dashboard.

## Features

- Marketing landing page with feature sections
- Blog system
- User account management
- Subscription and pricing pages
- Admin dashboard
- Single project sharing the same Material UI design system
- Image cropping component
- Emoji picker
- Drag-and-drop components
- Mature codebase used in real projects
