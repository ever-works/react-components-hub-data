## Overview

React Highlight Words is a React component that highlights specified words or phrases in text, making it perfect for search result highlighting, text matching, and content emphasis.

## Key Features

- **Flexible Matching**: Highlight multiple words/phrases
- **Case Sensitive**: Optional case-sensitive matching
- **Whole Words Only**: Match whole words only option
- **Custom Styling**: Style highlighted text via className or style
- **Active Highlighting**: Highlight active search term differently
- **Performance**: Efficient text parsing
- **TypeScript**: Type-safe component

## Props

- autoEscape: Auto-escape special regex characters
- caseSensitive: Case-sensitive matching
- textToHighlight: Text to search within
- searchWords: Words/phrases to highlight
- highlightClassName: CSS class for highlights
- highlightStyle: Inline styles for highlights
- activeClassName: Class for active highlights
- activeStyle: Styles for active highlights
- activeIndex: Index of active highlight

## Use Cases

- Search Result Highlighting
- Text Search in Content
- Keyword Emphasis
- Text Matching
- Search Highlighting
- Content Discovery

## Performance

- Efficient string parsing
- Minimal re-renders
- Large text support
