## Overview

Next.js is a popular React meta-framework that extends React with features for building production-grade web applications. It supports multiple rendering patterns including server-side rendering (SSR), static site generation (SSG), incremental static regeneration (ISR), and React Server Components (RSC).

## Features

- Server-side rendering and static site generation out of the box
- File-based routing system
- API routes for building backend functionality
- React Server Components (RSC) and React Server Functions (RSF) since 2023
- Image optimization and font optimization
- Built-in CSS support and Sass support
- Middleware support for edge execution
- App Router and Pages Router architectures
- Internationalization (i18n) routing
