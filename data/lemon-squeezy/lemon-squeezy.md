## Overview

Lemon Squeezy is a merchant of record that combines the power of Stripe with simplified tax handling, global payments, and subscription management. It has been acquired by Stripe and provides an easier alternative for React developers who need payment processing without dealing directly with Stripe's complexities.

## Features

- Built on Stripe infrastructure
- Automatic tax collection and remittance
- Subscription and one-time payment support
- License key management
- Affiliate system
- Global payment methods
- React integration via Stripe-compatible APIs
- Merchant of record (handles tax compliance)

## Pricing

No monthly fees. Transaction-based pricing with a percentage fee per successful sale, varying by plan tier.
