## Overview

React Local Toast is a notification component designed for React applications, known for its simplicity and efficiency. It provides a lightweight solution for managing notifications without compromising on essential features, making it ideal for projects that require a lean notification system.

## Features

- Easy and quick integration into React projects
- Lightweight and minimal footprint
- Minimalist design approach
- Essential notification features without bloat
- Supports local toast notifications
- Responsive and accessible
- Straightforward API

## Installation

```bash
npm install react-local-toast
```
