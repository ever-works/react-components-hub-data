## Overview

Prettier is an opinionated code formatter that enforces a consistent code style by parsing code and reprinting it with its own rules. It removes all original styling and ensures code looks consistent.

## Features

- **Opinionated**: Minimal configuration needed
- **Multi-Language**: JS, TS, JSX, CSS, HTML, and more
- **Editor Integration**: VS Code, WebStorm, Vim, etc.
- **Pre-commit Hooks**: Format before commits
- **CI Integration**: Enforce formatting in CI/CD
- **ESLint Integration**: Works with ESLint
- **Customizable**: Some configuration options
- **Fast**: Efficient formatting

## Use Cases

- Consistent code formatting across teams
- Eliminating style debates
- Automatic code cleanup
- Pre-commit formatting
- CI/CD code quality checks
