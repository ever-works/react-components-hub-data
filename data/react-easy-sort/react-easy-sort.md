## Overview

React Easy Sort allows sorting elements with drag and drop functionality. It's mobile-friendly by default and doesn't block scrolling when swiping inside sortable areas, providing a smooth user experience on all devices.

## Features

- Mobile and touch device support
- Doesn't block page scrolling
- Smooth drag animations
- Vertical and horizontal sorting
- Auto-scroll when dragging near edges
- Customizable drag handle
- Custom drag item styling
- Grid layout support
- TypeScript support
- Minimal dependencies
- Small bundle size
- Works with lists and grids
- Accessibility considerations

## Components

- SortableList: Wrapper for sortable items
- SortableItem: Individual sortable element
- SortableKnob: Optional drag handle

## Callbacks

- onSortEnd: Called when sorting completes
- Receives old and new indices

## Use Cases

- Todo list reordering
- Image gallery sorting
- Playlist management
- Dashboard widget arrangement
- Priority lists
- Menu item ordering
