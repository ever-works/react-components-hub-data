## Overview

simpleParallax.js is a JavaScript library that adds a parallax effect to images. It is designed to be the easiest way to implement parallax scrolling with just a few lines of code.

## Features

- Easy parallax effect for images
- Lightweight and performant
- Works with React through ref integration
- Configurable scale, orientation, and delay
- No dependencies
- Works on both horizontal and vertical scrolling
- Customizable max transition value
