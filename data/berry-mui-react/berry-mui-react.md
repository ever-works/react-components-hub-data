## Overview

Berry is a Material UI dashboard by CodedThemes that stands out with its distinctive color palette and visual personality. It is featured on the MUI Store.

## Features

- Dark and light modes
- Authentication pages with Auth0, Firebase, and JWT support
- ApexCharts integration
- Redux state management
- Nine demo pages
- Customizable color presets
- Bold color palette that diverges from default Material Design
- Refined component styling with visual warmth
