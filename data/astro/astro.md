## Overview

Astro is a framework-agnostic web development tool optimized for building fast, content-focused websites. It integrates seamlessly with React components while using an islands architecture that ships only HTML and CSS by default, loading JavaScript only when interactivity is required.

## Features

- Zero JavaScript shipped by default
- Islands architecture for selective hydration
- Framework-agnostic: works with React, Vue, Svelte, and more
- Static site generation optimized for performance
- Built-in support for Markdown and MDX
- Image optimization and asset handling
- Content Collections for organizing content
