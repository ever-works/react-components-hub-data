## Overview

React-Charts provides flexible, composable chart components for React with support for multiple chart types and data formats.

## Features

- Support for line, bar, pie, donut, radar, and scatter charts
- Responsive design with automatic resizing
- Customizable color palettes and themes
- Interactive tooltips and legends
- Animation support for data transitions
- Multiple data format support (JSON, CSV, arrays)
- Combination chart types

## Installation

`npm install react-charts`
