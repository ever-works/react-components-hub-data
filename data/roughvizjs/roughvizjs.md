## Overview

roughViz.js is a JavaScript library for creating sketchy, hand-drawn styled charts. It builds on D3.js, Rough.js, and Handy to produce unique artistic-looking data visualizations. The library provides a simple HTML-based API and is available for React and Vue integration.

## Features

- Hand-drawn, sketchy chart appearance
- Chart types: bar, stacked bar, donut, pie, horizontal bar, line, scatter
- Built on D3.js, Rough.js, and Handy
- Simple HTML-based API
- Customizable title, subtitle, axis labels, and colors
- Responsive and interactive charts
- Works with React, Vue, and vanilla JS
