## Overview

React Firebase Hooks provides a set of reusable React Hooks for Firebase. It simplifies Firebase integration by handling subscriptions, loading states, and errors automatically.

## Features

- **Auth Hooks**: Authentication state management
- **Firestore Hooks**: Real-time document/collection hooks
- **Storage Hooks**: File upload/download hooks
- **Database Hooks**: Realtime Database hooks
- **Loading States**: Built-in loading indicators
- **Error Handling**: Automatic error states
- **TypeScript**: Full type support
- **Auto Cleanup**: Automatic subscription cleanup

## Use Cases

- Firebase authentication
- Real-time data synchronization
- File storage and uploads
- User authentication state
- Firestore queries
- Firebase-powered React apps
