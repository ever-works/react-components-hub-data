## Overview

SurveyJS is an advanced Survey and Form library that provides a comprehensive toolkit for creating dynamic, customizable surveys and forms in React and other JavaScript frameworks.

## Features

- JSON-based survey definition
- Multiple question types
- Conditional logic and branching
- Data validation
- Customizable themes
- Data export (JSON, CSV, Excel)
- Responsive design
- Localization support
- Real-time preview
- Data persistence
