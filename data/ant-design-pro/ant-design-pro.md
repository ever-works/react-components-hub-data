## Overview

Ant Design Pro is the most starred React admin template, backed by the Ant Design team at Alibaba. It is a production-grade admin framework used by Fortune 500 companies.

## Features

- 20+ business-ready pages out of the box
- ProComponents library for rapid form and table development
- Built-in i18n supporting 15+ languages
- Route-level access control
- Full TypeScript throughout
- Integration with UmiJS build tool
- Enterprise design quality refined at massive scale
