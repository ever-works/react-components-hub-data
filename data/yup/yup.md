## Overview

Yup is a schema builder for runtime value parsing and validation. It provides an expressive API for building complex validation schemas and is the most popular validation library for React forms.

## Features

- Expressive schema building
- Async validation support
- Custom validation methods
- Extensible with custom types
- Composable schema building
- Transform values during validation
- Conditional validation (when)
- Array and object validation
- Lazy validation for recursive schemas
- Custom error messages
- Localization support
- TypeScript support
- Integration with Formik
- Works with React Hook Form

## Validation Methods

- String validation (email, url, uuid, matches, etc.)
- Number validation (min, max, positive, integer, etc.)
- Date validation (min, max)
- Boolean validation
- Array validation (min, max, of)
- Object validation (shape)
- Mixed type validation (oneOf, notOneOf)
- Nullable and optional fields

## Schema Composition

- Conditional requirements with when()
- Cross-field validation
- Transform values with transform()
- Test custom conditions with test()
- Lazy evaluation for recursive types

## Integration

- Formik (built-in support)
- React Hook Form with @hookform/resolvers
- Final Form
- Custom form solutions

## Use Cases

- Form validation in React applications
- API request/response validation
- Configuration validation
- User input sanitization
