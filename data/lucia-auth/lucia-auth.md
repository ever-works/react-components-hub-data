## Overview

Lucia Auth is a lightweight authentication library that provides the building blocks for implementing authentication in JavaScript applications. It is part of a broader ecosystem that includes Oslo for authentication and cryptography primitives, and Arctic for OAuth 2.0 support with 50+ providers.

## Features

- Oslo: Authentication and cryptography primitives
- Arctic: OAuth 2.0 with support for 50+ providers
- Framework-agnostic design
- Works with any database adapter
- Session-based authentication
- Strong typing with TypeScript
- Lightweight with minimal dependencies
