## Overview

Twin is a tool that fuses Tailwind's utility classes with CSS-in-JS power, compiling them at build time. Write familiar Tailwind classes directly in JSX, and Twin magically transforms them into performant, scoped styles with no runtime cost.

## Features

- Combines Tailwind utility classes with CSS-in-JS
- Compiles styles at build time
- Zero runtime cost
- Works directly in JSX
- Produces performant, scoped styles
- Integrates with popular CSS-in-JS libraries
