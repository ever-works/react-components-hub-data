## Overview

Reakit is a toolkit for building accessible UIs with React. It provides low-level interactive components and hooks that handle complex accessibility patterns, allowing you to build accessible interfaces with confidence.

## Features

- WCAG 2.1 compliant components
- Comprehensive keyboard navigation support
- Screen reader friendly
- Full TypeScript support
- Composable hooks-based API
- No styling dependencies (bring your own styles)
- Focus management
- Dialog, menu, tooltip, popover, tabs, and more
- Extensive component collection

## Components Included

- Composite (toolbar, menubar)
- Disclosure (accordion, tabs)
- Dialog (modal, alert, popover)
- Menu (dropdown, context menu)
- Form (checkbox, radio, input, select)
- Tooltip and more

## Installation

```bash
npm install reakit
```
