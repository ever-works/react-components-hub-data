## Overview

React Stonecutter is a highly configurable animated grid layout component for React. It provides a masonry-like layout where items can enter, leave, and update with smooth spring animations.

## Features

- Animated grid layouts with spring physics
- Masonry-style layout for items of varying sizes
- Configurable column count and gutter
- Enter, leave, and update animations
- Spring-based animation via react-motion
- Responsive layout support
- Supports any content within grid cells
