## Overview

htmltojsx is part of React Magic — a tool that automatically converts plain HTML markup into React JSX components, streamlining the migration of existing websites to React.

## Features

- Automatic HTML-to-JSX syntax conversion
- Handles inline style object generation from CSS
- Converts event handlers to React synthetic events
- CLI for batch processing
- Handles className and htmlFor conversions
- Supports inline SVG conversion
