## Overview

React Multistep is a flexible wizard component for building multi-step forms and sequential data collection flows in React. Version 6+ is completely headless, separating logic from presentation.

## Features

- Headless architecture - complete UI control
- State management for steps
- Navigation between steps
- Step validation support
- Progress tracking
- Conditional step rendering
- Step skipping logic
- Data persistence across steps
- Custom navigation controls
- Step lifecycle hooks
- TypeScript support
- Lightweight and performant
- No styling opinions
- Compatible with any UI framework
- Accessible by design

## Architecture

- MultiStep component manages state and logic
- Developers control all UI rendering
- Hook-based API for step management
- Flexible step configuration

## Use Cases

- Multi-step registration forms
- Onboarding flows
- Checkout processes
- Survey and questionnaire builders
- Configuration wizards
- Data collection forms
