## Overview

react-use-form-state is a tiny React Hook that helps manage form state with minimal effort. It provides a clean API for handling form inputs, validation, and submission without the overhead of larger form libraries.

## Features

- **Simple API**: Easy to learn and use
- **Input Binding**: Automatic input value and onChange binding
- **Type Safety**: Full TypeScript support
- **Validation**: Built-in validation support
- **Form State**: Track pristine, touched, and validity states
- **Custom Inputs**: Works with custom input components
- **Tiny Bundle**: Minimal bundle size impact

## Use Cases

- Simple contact forms
- Login and registration forms
- Search forms and filters
- Settings and preferences
- Small to medium complexity forms
