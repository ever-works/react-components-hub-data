## Overview

next-intl is an internationalization library specifically built for Next.js applications. It provides comprehensive locale support including translations, locale negotiation, routing, and date/number formatting, designed to work seamlessly with both the Next.js App Router and Pages Router.

## Features

- Built specifically for Next.js
- Support for both App Router and Pages Router
- Locale negotiation and routing
- Message formatting with ICU syntax
- Date, number, and time formatting
- Type-safe translations
- Server and client component support
- Lightweight bundle size
