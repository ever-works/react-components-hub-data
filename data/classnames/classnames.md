## Overview

classnames is a simple utility for conditionally joining CSS class names together. It's one of the most popular utilities in the React ecosystem for handling dynamic class names.

## Features

- **Conditional Classes**: Add classes based on conditions
- **Object Syntax**: Clean conditional syntax
- **Array Support**: Combine multiple sources
- **Falsy Values**: Automatically filters falsy values
- **TypeScript**: Type definitions included
- **Tiny**: Only 1KB minified
- **Fast**: Highly optimized
- **Dedupe**: Optionally deduplicate classes

## Use Cases

- Conditional CSS classes
- Component state styling
- Theme switching
- Responsive classes
- BEM methodology
- Any dynamic class name needs
