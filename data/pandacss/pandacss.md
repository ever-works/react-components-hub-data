## Overview

PandaCSS is a modern CSS-in-JS engine that combines the developer experience of CSS-in-JS with zero-runtime overhead. It compiles to atomic CSS at build time, providing type-safe utility-first styling with advanced features like conditional styling, recipes, and design token support.

## Features

- Zero-runtime CSS compilation
- Type-safe styling with TypeScript
- Design token system
- Recipe-based component styling
- Conditional and responsive utilities
- Works with any framework including React
- No hydration issues
- Generated style sheet for optimal performance
