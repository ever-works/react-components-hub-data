## Overview

React GSAP Enhancer bridges React and GSAP, allowing developers to use GSAP's animation features (timelines, tweens, plugins) within React's component lifecycle while maintaining React's rendering patterns.

## Features

- Full GSAP API access in React components
- Timeline and tween management
- Integration with React lifecycle methods
- Support for GSAP plugins (ScrollTrigger, MorphSVG, etc.)
- Reusable animation patterns
