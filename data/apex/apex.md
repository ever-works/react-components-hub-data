## Overview

Apex is the most comprehensive React admin dashboard template, delivering 125+ static routes across five dashboard variants built on Next.js 16 with 35 shadcn/ui primitives.

## Features

- Five dashboard variants: Overview, Analytics, eCommerce, CRM, SaaS
- 125+ static routes covering every admin use case
- 20+ app pages included
- TanStack Table v8 for data tables
- Recharts 3 for charting and visualizations
- Authentication with 2FA
- Command Palette for quick navigation
- i18n support for internationalization
- Live theme customizer for real-time branding preview
- 35 shadcn/ui component primitives
- Dark/light theme support
