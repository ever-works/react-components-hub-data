## Overview

Zag (Zero-API Grind) is a library of framework-agnostic UI component pattern primitives. Built using finite state machines (XState), it provides the logic and accessibility for common UI patterns like accordions, dialogs, menus, and tabs. It works with React, Vue, and Solid.

## Features

- Framework-agnostic: works with React, Vue, and Solid
- Built on finite state machines using XState
- Headless component logic with built-in accessibility
- Consistent API across frameworks
- Supports 30+ UI patterns
- TypeScript support
- Used as the foundation for Ark UI
