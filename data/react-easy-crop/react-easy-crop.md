## Overview

React Easy Crop is a React component to crop images with easy interactions and minimal bundle size. It's actively maintained and designed for modern React with hooks support.

## Features

- Lightweight bundle size
- Hook-based API
- Zoom functionality
- Pan/drag positioning
- Touch gesture support
- Fixed or free aspect ratios
- Minimum/maximum zoom
- Crop area customization
- TypeScript support
- Responsive design
- Rotation support
- Easy integration
- Custom media (images/videos)
- Callback on crop completion
- Style customization

## Cropping Options

- Fixed aspect ratio (1:1, 4:3, 16:9, etc.)
- Free-form cropping
- Custom aspect ratios
- Circular crops for avatars

## Use Cases

- Profile picture cropping
- Avatar editors
- Image uploads
- Photo editing apps
- Social media features
- Product images
