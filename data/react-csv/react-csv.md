## Overview

React CSV provides simple components for generating and downloading CSV files from data in React applications. It includes CSVLink and CSVDownload components for flexible CSV generation.

## Features

- CSVLink component (download on click)
- CSVDownload component (auto-download)
- Array of arrays support
- Array of objects support
- String data support
- Custom headers
- Custom separators
- Custom filenames
- UTF-8 BOM support
- Event callbacks (onClick)
- Asynchronous data loading
- TypeScript definitions
- Lightweight package
- Easy integration

## Components

- `<CSVLink>`: Link element for downloading CSV
- `<CSVDownload>`: Auto-trigger download

## Use Cases

- Export table data
- Report generation
- Data backup
- Analytics export
- Contact list downloads
- Inventory exports
- User data downloads
