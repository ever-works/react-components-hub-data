## Overview

Minimal UI blends a clean, minimalist design with powerful analytics features like ApexCharts, Vite 5, and Iconify React. It is designed for startups looking for a performance-optimized, semantic UI.

## Features

- Fully responsive and easily customizable
- Smooth animations for polished user experience
- Lightweight yet feature-packed
- ApexCharts for data visualization
- Vite 5 for fast builds
- Iconify React for icons
- Tech stack: React 18, TypeScript, MUI, Vite 5
