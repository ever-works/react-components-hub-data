## Overview

FlyonUI is an open-source Tailwind CSS component library that provides a comprehensive set of customizable UI components. It includes seamlessly integrated toast notification components with multiple variations suitable for modern web applications.

## Features

- Open-source Tailwind CSS component library
- Multiple toast notification variations and styles
- Seamless integration with React via Tailwind CSS
- Customizable notification components
- Notyf integration for toast notifications
- Framework-agnostic with React support guide
