## Overview

Tremor is an open-source library of React components specifically designed for creating dashboards and data visualizations. It provides copy-and-paste Tailwind CSS UI components for charts and dashboards that are both beautiful and functional.

## Features

- 35+ fully open-source, accessible components
- Built with React, Tailwind CSS, and Radix UI
- Optimized for dashboard and data-heavy applications
- Copy-paste philosophy - you own the code
- Modern, clean design system
- Comprehensive chart components (Area, Bar, Line, Donut, etc.)
- Data display components (Cards, Tables, Metrics)
- Input components for filtering and interaction
- Full TypeScript support
- Responsive and mobile-friendly
- Built-in accessibility features
