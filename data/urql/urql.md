## Overview

urql (Universal React Query Library) is a lightweight GraphQL client developed by Formidable. It provides a simple yet powerful API for consuming GraphQL APIs in React applications, with a smaller bundle size than Apollo Client while still offering comprehensive features for caching, subscriptions, and query management.

## Features

- Lightweight GraphQL client with minimal bundle size
- Built on top of normalized caching
- Support for GraphQL subscriptions
- Customizable exchange system for middleware
- React hooks-based API
- DevTools integration
- SSR support
- Works with any GraphQL API
