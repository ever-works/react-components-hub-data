## Overview

Luckysheet is an open-source online spreadsheet built with pure JavaScript that provides a Google Sheets-like experience directly in the browser.

## Features

- Collaborative editing
- Full formula engine
- JSON-based data storage
- Built with pure JavaScript (no framework dependencies)
- Familiar spreadsheet UI similar to Google Sheets
- Cell formatting and styling

## Pricing

Free and open-source under the MIT license.
