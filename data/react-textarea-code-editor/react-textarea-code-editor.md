## Overview

A simple code editor built on HTML textarea with syntax highlighting. Provides a lightweight solution for code editing without unnecessary complexity.

## Features

- Built on standard HTML textarea
- Syntax highlighting for multiple languages
- Lightweight implementation
- Easy to integrate
- Customizable styling
- Line numbers support
- Copy to clipboard functionality
- Dark mode support
- Mobile friendly

## Languages Supported

- JavaScript/TypeScript
- Python
- HTML/CSS
- SQL
- JSON
- Shell/Bash
- And many more via Prism.js

## Use Cases

- Code submission forms
- Documentation examples
- Configuration file editors
- Quick code snippets
- Form-based code input
