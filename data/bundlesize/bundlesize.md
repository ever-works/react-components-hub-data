## Overview

bundlesize helps you keep your bundle sizes in check by setting size limits and validating them in CI. It prevents bundle size regressions and keeps your React application performant.

## Features

- **Size Budgets**: Set maximum bundle sizes
- **CI Integration**: Fail builds on size increase
- **Multiple Bundles**: Check multiple files
- **Compression**: Check gzipped sizes
- **GitHub Integration**: Status checks on PRs
- **Custom Globs**: Match file patterns
- **Flexible Config**: Configure thresholds
- **Performance**: Prevent bundle bloat

## Use Cases

- Bundle size monitoring
- Performance budgets
- CI/CD quality gates
- Preventing regressions
- Optimizing bundle sizes
- Team performance standards
