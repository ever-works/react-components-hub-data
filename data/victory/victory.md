## Overview

Victory is a collection of composable React components for building interactive data visualizations. Each module can be used independently.

## Key Features

- Modular components
- Composable architecture
- Accessibility built-in
- ARIA labels
- Animation support
- Responsive design
- Theming
- TypeScript support
- Touch-friendly
- Server-side rendering

## Component Types

- Area, Bar, Line, Scatter, Pie
- VictoryChart for composition
- Axis, Grid, Legend
- Tooltip, Voronoi
- Container components
