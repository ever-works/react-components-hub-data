## Overview

Signal is a terminal-inspired DevOps dashboard built for engineering teams, designed to speak the visual language of developers who live in terminals.

## Features

- JetBrains Mono typography for terminal aesthetic
- 57+ pages included
- 13 infrastructure-specific views: Server Fleet, Containers, Deployments, Incidents, Log Explorer, Uptime Monitor, Pipelines, Databases, Security Audit, API Monitoring, Kubernetes, Cloud Costs, Status Page
- Six color presets for team customization
- Three density modes for personal preference
- Built with React + Next.js + shadcn/ui
- Dark/light theme support
- Every page designed for engineering operations from the start
