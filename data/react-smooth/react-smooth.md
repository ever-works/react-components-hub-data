## Overview

react-smooth is a React component that makes it easy to add smooth animations to your components. It's particularly useful for animating data visualizations and charts.

## Features

- **Spring Physics**: Natural-feeling animations
- **Easing Functions**: Multiple easing options
- **Duration Control**: Time-based animations
- **From/To Values**: Simple animation definition
- **Customizable**: Flexible configuration
- **TypeScript**: Type definitions included
- **Lightweight**: Small bundle size

## Use Cases

- Chart animations (used by Recharts)
- Data visualization transitions
- Number counters and progress bars
- UI element animations
- Smooth state transitions
