## Overview

React Focus Lock is a component that traps focus inside a specified container. It's essential for creating accessible modal dialogs, popups, and overlays that comply with WCAG accessibility guidelines.

## Features

- **Focus Trapping**: Keep focus within component
- **Auto Focus**: Automatically focus first element
- **Return Focus**: Return focus on unmount
- **Nested Locks**: Support for nested focus locks
- **Custom Focus**: Control focus order
- **Accessibility**: ARIA compliant
- **TypeScript**: Full type support
- **SSR Compatible**: Works with server-side rendering

## Use Cases

- Modal dialogs and overlays
- Dropdown menus
- Navigation drawers
- Alert dialogs
- Accessible form wizards
- Keyboard navigation
