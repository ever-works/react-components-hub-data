## Overview

react-md-editor is a simple yet powerful Markdown editor component for React applications. It allows users to easily create and edit Markdown content within a user-friendly interface, providing a split view with live preview.

## Features

- Live Markdown preview
- Syntax highlighting for code blocks
- Customizable toolbar
- Support for standard Markdown syntax and extensions
- Split editor/view mode
- Full-screen editing mode
- Support for GitHub-flavored Markdown (GFM)
- Custom theme support
- Keyboard shortcuts
- Content height customization
- Export functionality
