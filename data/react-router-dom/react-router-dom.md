## Overview

React Router DOM provides the web-specific components and hooks for React Router, enabling full-featured client-side routing for single-page applications.

## Key Features

- **Route Matching**: Dynamic route matching
- **Nested Routes**: Support for nested routing
- **URL Params**: Dynamic path parameters
- **Search Params**: Query string handling
- **Navigation**: Programmatic navigation
- **Links**: Client-side links
- **Outlets**: Route rendering outlets
- **Loaders**: Data loading before render
- **Actions**: Form submission handling
- **TypeScript**: Full type support

## Components

- BrowserRouter: Router provider
- Routes: Route definitions
- Route: Single route
- Link: Navigation links
- NavLink: Active-aware links
- Outlet: Nested route display
- Navigate: Redirect component

## Hooks

- useNavigate: Navigate programmatically
- useParams: Get URL parameters
- useSearchParams: Query string access
- useLocation: Get current location
- useMatch: Match routes
- useResolvedPath: Resolve paths
- useOutlet: Access nested outlet
