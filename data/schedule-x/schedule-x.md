## Overview

Schedule-X is a modern event calendar and scheduler component for React applications. It provides multiple view types and is built with extensive customization capabilities for various scheduling requirements.

## Features

- Multiple calendar views (day, week, month)
- Event management
- Drag and drop events
- Resize events
- Customizable appearance
- Plugin system
- Recurring events support
- Time zone handling
- Resource scheduling
- Custom event rendering
- Responsive design
- TypeScript support
- Accessibility features
- Modern architecture
- Framework integrations

## View Types

- Day view
- Week view
- Month view
- Agenda view
- Custom views

## Use Cases

- Appointment booking
- Meeting schedulers
- Event calendars
- Resource management
- Class schedules
- Team calendars
- Booking systems
