## Overview

React Redux is the official React UI bindings layer for Redux. It lets your React components read data from a Redux store and dispatch actions to update the store.

## Features

- **React Hooks**: useSelector and useDispatch hooks
- **Performance Optimized**: Automatic render optimization
- **TypeScript Support**: Full type inference
- **Connect API**: Classic connect() HOC for class components
- **Provider Component**: Easy Redux store integration
- **Batch Updates**: Batched subscription updates
- **DevTools**: Redux DevTools Extension integration

## Use Cases

- Complex application state management
- Global state shared across components
- Predictable state updates
- Time-travel debugging
- State persistence and hydration
