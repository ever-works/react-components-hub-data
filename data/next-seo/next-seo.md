## Overview

Next SEO is a plugin that makes managing SEO in Next.js applications easier. It provides simple components for setting meta tags, Open Graph tags, Twitter Cards, and JSON-LD structured data.

## Features

- **Meta Tags**: Easy meta tag management
- **Open Graph**: Facebook sharing optimization
- **Twitter Cards**: Twitter sharing configuration
- **JSON-LD**: Structured data support
- **TypeScript**: Full type definitions
- **Default SEO**: Set default SEO config
- **Per-page SEO**: Override defaults per page
- **Canonical URLs**: Duplicate content prevention

## Use Cases

- Next.js SEO optimization
- Social media sharing
- Search engine optimization
- Rich search results
- Blog and content sites
- E-commerce SEO
