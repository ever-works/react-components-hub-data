## Overview

A widely-used JavaScript library for creating reorderable drag-and-drop lists. Works across multiple frameworks and platforms.

## Features

- Drag and drop reordering within and between lists
- Touch-friendly interface
- Works with any HTML element
- Multi-framework support (React, Angular, Vue, etc.)
- Auto-scrolling during drag operations
- Customizable drag handles and ghost elements
- Smooth animations
- No external dependencies
