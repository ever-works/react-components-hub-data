## Overview

React-elastic-carousel is a robust React carousel component library designed to create carousels or sliders in React applications. With its flexibility and customization options, you can effortlessly showcase a collection of content items in a smooth sliding manner, whether horizontally or vertically.

## Features

- Smooth elastic drag effects for playful UI
- Supports breakpoints and custom item widths
- Simple API and responsive design
- Horizontal and vertical sliding support
- Flexible customization options for unique requirements
