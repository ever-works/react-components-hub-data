## Overview

RTK Query is the official data fetching and caching tool built into Redux Toolkit. It provides a powerful approach to data fetching that eliminates the need to write boilerplate fetching logic while automatically managing caching, invalidation, and refetching. It is ideal for applications already using Redux.

## Features

- Built into Redux Toolkit with zero additional dependencies
- Automatic caching and deduplication
- Automatic re-fetching on refocus and refetch intervals
- Optimistic updates and mutations
- Generated React hooks for each endpoint
- TypeScript-first design with full type inference
- Automatic normalization and pagination support
- Integrated with Redux DevTools
