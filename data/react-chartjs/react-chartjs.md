## Overview

React ChartJS provides React components for Chart.js, making it easy to integrate Chart.js charts into React applications. It offers a declarative API while maintaining access to all Chart.js features.

## Features

- **Chart.js 3+**: Latest Chart.js version support
- **All Chart Types**: Line, Bar, Pie, Doughnut, Radar, and more
- **TypeScript**: Full type safety
- **Hooks API**: Access chart instances directly
- **Plugins**: Support for Chart.js plugins
- **Responsive**: Mobile-friendly charts
- **Animations**: Smooth chart animations
- **Tree Shaking**: Import only what you need

## Use Cases

- Business analytics dashboards
- Data visualization
- Progress tracking
- Statistical reports
- Real-time data monitoring
