## Overview

React Cookie is a small library that manages cookies in React applications. It offers an API that makes it easy to work with cookies in React, streamlining state management. It includes functions for setting, getting, and deleting cookies, as well as utility functions for checking the existence of cookies and getting all cookies.

## Features

- Easy-to-use API for handling cookies in React
- Works on both client and server sides
- Provides functions for essential CRUD (Create, Read, Update, Delete) operations on cookies
- Utility functions to check the existence of cookies and retrieve the cookie list
- Strong typing support using TypeScript
- Easily integrated with other React applications
- Designed to be lightweight
- Actively maintained with updates and bug fixes
