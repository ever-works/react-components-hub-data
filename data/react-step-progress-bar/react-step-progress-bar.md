## Overview

react-step-progress-bar is a React library for creating visually appealing progress bars with defined steps. It's perfect for visualizing multi-step processes like forms, wizards, tutorials, and onboarding flows.

## Features

- **Customizable Steps**: Define any number of steps with custom positioning
- **Flexible Layout**: Steps can be spaced linearly or positioned manually
- **Custom Components**: Use your own components for steps and progress bar
- **Progress Tracking**: Visual indication of completed, current, and upcoming steps
- **Responsive**: Adapts to different screen sizes
- **Styling Freedom**: Full control over colors, sizes, and appearance
- **Animation Support**: Smooth transitions between steps

## Configuration

- **Steps Positioning**: By default, steps are spaced linearly on the ProgressBar, but you can override this by specifying positions in percentages
- **Custom Rendering**: Replace default step and progress bar components with your own
- **Progress Percentage**: Control the current progress programmatically
- **Step States**: Distinguish between accomplished, current, and pending steps

## Common Use Cases

- Multi-step registration forms
- Checkout processes
- Onboarding tutorials
- Survey and questionnaire flows
- Installation wizards
- Order tracking interfaces
- Course or lesson progress

## Component Architecture

The library provides a ProgressBar component that accepts Step components as children. Each step can be customized with transitions, labels, and custom rendering logic.
