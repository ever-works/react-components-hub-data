## Overview

Semiotic is a data visualization framework for React created by the nteract team. It wraps D3 functionality into React components, making it easier to create custom visualizations like network charts, sunburst diagrams, Voronoi treemaps, and area charts. It provides a composable API for building complex visualizations that leverage React's declarative paradigm.

## Features

- React-based declarative visualization API
- Built on D3.js for powerful rendering
- Visualization types: XYFrame, OrdinalFrame, DivFrame, Voronoi, Network, Sunburst, Sankey
- Responsive charts with dynamic sizing
- Annotation and labeling support
- Customizable axes, legends, and tooltips
- Animation and transition support
- TypeScript definitions included
- React-only framework
