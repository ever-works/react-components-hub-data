## Overview

React Photo Album is a responsive photo gallery component designed to work seamlessly with modern lightbox libraries like yet-another-react-lightbox.

## Features

- **Responsive Layouts**: Multiple layout algorithms
- **Lazy Loading**: Performance-optimized image loading
- **Lightbox Integration**: Works with popular lightbox libraries
- **Touch Support**: Mobile-friendly interactions
- **TypeScript**: Full TypeScript support
- **SSR Compatible**: Server-side rendering support
- **Customizable**: Extensive customization options

## Layout Types

- Rows layout: Traditional gallery rows
- Columns layout: Masonry-style columns
- Masonry layout: Pinterest-style masonry
- Custom layouts: Build your own

## Image Handling

- Responsive image sources
- Srcset and sizes attributes
- Custom aspect ratios
- Photo metadata support
- Alt text and titles
- Custom photo rendering

## Performance

- Lazy loading built-in
- Optimized rendering
- Minimal re-renders
- Efficient layout calculations
- ResizeObserver support

## Customization

- Custom photo components
- Custom container components
- Spacing configuration
- Padding settings
- Breakpoint configuration
- Click handlers

## Use Cases

- Portfolio galleries
- Image showcases
- Photography websites
- Product galleries
- Media libraries
- Documentation images
