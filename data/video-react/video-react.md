## Overview

Video-React is a web video player built specifically for React and HTML5. It provides a complete video player component with customizable controls, keyboard shortcuts, and accessibility features.

## Features

- Full HTML5 video support
- Customizable player controls
- Keyboard shortcuts
- Fullscreen support
- Responsive design
- Progressive loading
- Subtitles/captions support
- Volume and playback rate control
- Playback quality selection
- Error handling
- Accessibility optimized
- TypeScript support

## Components Included

- Player (main container)
- PlaybackRateMenuButton
- VolumeMenuButton
- Shortcut menu
- Progress control
- ClosedCaptions

## Installation

```bash
npm install video-react
```

## Styling

Video-React includes default CSS styling but is designed for customization. You can override styles using CSS or styled-components.
