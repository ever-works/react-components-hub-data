## Overview

Vault is a Robinhood-inspired investment dashboard with Plus Jakarta Sans for readability and Geist Mono for financial data precision, built for investment and fintech use cases.

## Features

- 57+ pages
- 13 investment modules: Portfolio, Watchlist, Market Overview, Stock Screener, Market Movers, Earnings Calendar, Dividends, Order History, Price Alerts, Analytics, Tax Center, Stock Detail
- Dark/light themes
- 6 color presets
- 10+ chart types
- Plus Jakarta Sans typography
- Geist Mono for financial data precision
- React + Next.js + Tailwind CSS
