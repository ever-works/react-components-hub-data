## Overview

ApexCharts is a modern JavaScript charting library for building interactive data visualizations. It provides a wide range of chart types with responsive design and easy integration across multiple frameworks including React, Angular, Vue, Svelte, and vanilla JS/TS.

## Features

- 14+ chart types: line, area, bar, column, pie, donut, scatter, bubble, heatmap, treemap, radial bar, radar, polar area, candlestick, box plot
- Interactive tooltips, zoom, pan, and selection
- Responsive design with automatic resizing
- Dynamic chart updates with real-time data
- Export charts as PNG, SVG, or CSV
- Customizable themes and styling
- TypeScript definitions included
- Works with React, Angular, Vue, Svelte, and vanilla JS/TS
