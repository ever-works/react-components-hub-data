## Overview

A React component for choosing dates. Built with date-fns for small package size and native date handling.

## Key Features

- Date and time selection
- Custom date formats
- Keyboard navigation
- Range selection
- Localization support
- Filter dates
- Inline mode
- Fixed height
- Excluded dates
- Min/max date constraints
- Touch-friendly

## Integration

- Easy integration with forms
- Hook-based API support
- Works with React Hook Form
- Works with Formik
- Custom input components
