## Overview

Vanilla Extract is a type-safe CSS-in-TS library that generates all styles at build time with zero runtime overhead. It provides a modern approach to co-locating styles with components while avoiding the performance and hydration issues associated with runtime CSS-in-JS solutions.

## Features

- Zero-runtime CSS generation at compile time
- Type-safe CSS with TypeScript
- Built-in theming system
- Responsive utility functions
- Recipe patterns for component variants
- Static style composition
- Works with Vite, Webpack, esbuild, and more
- Works with React, preact, solid-js, and more
