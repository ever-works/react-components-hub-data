## Overview

Excalidraw is a virtual collaborative whiteboard tool with a hand-drawn visual style. It is excellent for rough yet lightweight sketches, rapid prototyping, and planning React component layouts or UI/UX concepts.

## Features

- Hand-drawn visual style
- Real-time collaborative editing
- End-to-end encryption
- Shape library and templates
- Easy-to-use drawing and text tools
- Embeddable as a component in web applications
- Open-source
