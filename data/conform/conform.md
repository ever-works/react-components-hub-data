## Overview

Conform is an emerging form validation library for React that integrates more naturally with full-stack applications. Unlike traditional form libraries, it works progressively enhanced, providing server-side validation that works even without JavaScript on the client.

## Features

- Server-first form validation
- Progressive enhancement support
- Type-safe form handling with TypeScript
- Integrates with Remix, Next.js, and other full-stack frameworks
- Works without JavaScript (graceful degradation)
- Schema-based validation with Zod
