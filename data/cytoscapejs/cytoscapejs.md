## Overview

Cytoscape.js is a fully featured graph theory library written in JavaScript. It provides an easy-to-use interface for interactive graph visualisation and analysis, with optional user interaction and graph animation. Built with React, Angular, and Vue support.

## Features

- Graph theory (a.k.a. network) representation and analysis
- Automatic node placement and edge routing with multiple layout algorithms
- Interactive user gestures for graph manipulation
- Customisable styling with CSS-like selectors
- SVG and HTML/CSS node rendering options
- Extension ecosystem for additional functionality
- TypeScript definitions included
- Supports React, Angular, Vue, and vanilla JS
