## Overview

untitledui Pagination is a collection of free and open-source React pagination components designed for modern applications and websites. The components are built on React Aria for robust accessibility and styled with Tailwind CSS for easy customization.

## Features

- Built with React Aria for accessible keyboard and screen reader navigation
- Styled with Tailwind CSS for seamless integration in Tailwind projects
- Installable via CLI (`npx untitledui@latest add pagination`)
- Source code available on GitHub

## Component Variations

- **Page Default** — Standard pagination with numbered page links
- **Page Minimal Center** — Minimalist centered page navigation
- **Card Default** — Card-style pagination with full-width container
- **Card Minimal Right Aligned** — Minimal card layout, right-aligned
- **Card Minimal Center Aligned** — Minimal card layout, center-aligned
- **Card Minimal Left Aligned** — Minimal card layout, left-aligned
- **Button Group Right Aligned** — Button-style pagination, right-aligned
- **Button Group Center Aligned** — Button-style pagination, center-aligned
- **Button Group Left Aligned** — Button-style pagination, left-aligned
- **Card Advanced** — Advanced card pagination with additional controls (showing item ranges, per-page selectors)
- **Card Advanced Center** — Advanced card pagination, center-aligned
- **Pagination Dot** — Dot-based pagination for carousels or sliders
- **Pagination Line** — Line-style pagination indicator
