## Overview

AG Grid is the industry standard for JavaScript data grids, offering both free and enterprise versions. The React integration provides seamless React component architecture while maintaining AG Grid's powerful features.

## Core Features (Free)

- Sorting (single and multi-column)
- Filtering (text, number, date)
- Pagination
- Row selection (single and multiple)
- Column resizing and auto-sizing
- Column pinning (left and right)
- Column reordering
- Cell rendering and editing
- Keyboard navigation
- Row height
- Header height
- CSV export
- Themes (Alpine, Balham, Material)
- Customizable
- TypeScript support
