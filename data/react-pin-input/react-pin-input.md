## Overview

React PIN Input provides a set of input fields optimized for entering PIN codes, OTP codes, and verification codes. It handles common patterns like auto-focus, paste handling, and keyboard navigation.

## Features

- Multiple input fields
- Auto-focus on mount
- Auto-focus next field
- Auto-focus previous on delete
- Paste support
- Copy support
- Keyboard navigation
- Numeric input
- Alphanumeric input
- Custom input types
- Custom field count
- Secure/password mode
- Custom styling
- Validation
- Disabled state
- Auto-submit on complete
- Clear button

## Input Types

- Numeric (0-9)
- Alphanumeric
- Custom pattern
- Password/secure

## Behavior

- Auto-advance to next field
- Backspace to previous field
- Paste entire code
- Select on focus
- Clear on paste
- Auto-submit when complete

## Styling

- Custom container styles
- Focus states

- Success states
- Disabled states
- Custom separators

## Validation

## Accessibility

- ARIA labels
