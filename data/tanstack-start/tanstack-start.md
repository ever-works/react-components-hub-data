## Overview

TanStack Start is a full-stack React meta-framework that is part of the TanStack ecosystem. It integrates with TanStack Router for type-safe routing and TanStack Query for data management. Currently in beta, it aims to provide a TypeScript-first developer experience for building server-rendered React applications.

## Features

- Full-stack React framework with SSR support
- Tight integration with TanStack Router
- Built-in data loading patterns
- React Server Components support in development
- Type-safe from routes to data
- Part of the broader TanStack ecosystem
