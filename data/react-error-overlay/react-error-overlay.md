## Overview

react-error-overlay is the error overlay component used by Create React App. It displays runtime errors and warnings in a user-friendly format during development with clickable stack traces and code snippets.

## Features

- **Error Display**: Beautiful error overlays
- **Source Code**: Show error location in code
- **Stack Traces**: Clickable stack traces
- **Warnings**: Display build warnings
- **Hot Reload**: Updates on code changes
- **Dismissable**: Close overlay to continue
- **CRA Integration**: Built into Create React App
- **Customizable**: Can be used standalone

## Use Cases

- Development error handling
- Custom React tooling
- Improved developer experience
- Error debugging
- Alternative to console errors
