## Overview

React Modal is a fully accessible modal dialog component for React applications. It provides an intuitive API for creating modal dialog windows with customizable styling and behavior.

## Features
-s Fully accessible via keyboard navigation
-s Customizable style properties
-s Support for portals and overlays
-s Easy to integrate with accessibility features
-s Lightweight and battle-tested in production

## Installation

Install via npm:

```
$ npm install react-modal
```
