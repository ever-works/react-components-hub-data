## Overview

Tiptap is a headless, framework-agnostic and extendable rich text editor based on ProseMirror. It's completely customizable and can be extended with plugins.

## Key Features

- Framework-agnostic core
- React component available
- Extension system
- Collaborative editing
- Built-in extensions
- Custom extensions
- TypeScript support
- Full keyboard support
- Accessible
- Mobile-friendly

## Built-in Extensions

- Formatting: Bold, Italic, Underline
- Lists: Bullet, Ordered
- Blocks: Paragraph, Heading, Code
- Media: Image, Link, Video
- Tables, Comments, Mentions
