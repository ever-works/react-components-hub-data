## Overview

React Cosmos is a tool for sandboxing and developing React components in isolation. It provides a workspace where developers can iterate on components, view all possible states, and test them independently from the application context.

## Features

- Isolated component rendering for focused development
- Fixture-based testing for all component states and props
- Support for rendering component fixtures with different providers and contexts
- Hot module reloading for fast iteration
- Auto-detection of component fixtures
- Export of fixtures for sharing and documentation
- Plugin system for extending functionality
- Works alongside existing build tools (Webpack, Vite)
- Component state persisting between hot reloads
