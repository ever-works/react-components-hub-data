## Overview

Unlayer provides a drag-and-drop email editor as a React component. It enables users to design beautiful, responsive emails without coding through an intuitive visual interface.

## Features

- Drag-and-drop interface
- Pre-built content blocks
- Responsive design tools
- Custom block creation
- Template library
- Export to HTML/JSON
- Image upload and management
- Font customization
- Mobile preview
- Merge tags support
- Undo/redo functionality
- Custom tools and properties
- White-label options
- Translation support
- Dark mode

## Content Blocks

- Text blocks
- Image blocks
- Button blocks
- Divider blocks
- Social icons
- Video embeds
- HTML blocks
- Menu blocks
- Timer blocks
- Custom blocks

## Use Cases

- Email marketing platforms
- Newsletter builders
- Transactional email design
- Email automation tools
- Marketing automation

## Pricing

Free tier available with usage limits. Premium plans start at $99/month for additional features and higher limits.
