## Overview

Shadcn UI Kit compiles various admin dashboards, web app starters, and blocks built around shadcn/ui. It provides pre-assembled layouts, but much of the value sits behind a payment wall.

## Tech Stack

- Next.js 16+
- Tailwind CSS v4
- React
- TypeScript

## Features

- Consistent minimal design aligned with shadcn/ui's typography and spacing
- Includes around 11 admin dashboards, 10 web app starters, 30+ subpages, plus blocks and components
- Built on current stack with Next.js 16+ and Tailwind CSS v4

## Pros

- Easier entry point if already locked into shadcn/ui patterns
- Some customization potential for basic adjustments

## Cons

- No Figma file available, hindering design-dev alignment
- No matching icons library included
- Relies on shadcn/ui's Radix foundations, carrying migration overhead to Base UI
- Free version is heavily restricted with limited updates; full access requires a one-time payment
