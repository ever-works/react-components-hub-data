## Overview

React Range Slider Input is a lightweight React component wrapper for range-slider-input. It provides an efficient way to create customizable range sliders for selecting single values or ranges.

## Features

- Lightweight (~2kB gzipped)
- Single value or range selection
- One or two drag handles
- Customizable styling
- Touch-friendly
- Keyboard navigation
- Min/max value constraints
- Step increments
- Disabled state support
- Orientation support (horizontal/vertical)
- RTL support
- TypeScript definitions
- Accessibility features
- No dependencies

## Slider Types

- Single value slider (one handle)
- Range slider (two handles)

## Use Cases

- Price range filters
- Volume controls
- Progress indicators
- Settings and preferences
- Data filtering
- Value selection in forms
