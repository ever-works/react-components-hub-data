## Overview

react-image-magnifiers provides a collection of responsive image magnifying components for React that work with both mouse and touch interactions. Perfect for product images and detailed photo viewing.

## Features

- Multiple magnifier types
- Mouse and touch support
- Zoom by click/double-click/tap
- Drag to move while zoomed
- Responsive design
- Configurable zoom levels
- Smooth transitions
- Touch gestures
- No dependencies
- Lightweight components
- Easy integration
- TypeScript definitions
- Customizable styling

## Component Types

- **Magnifier**: Basic in-place magnification
- **GlassMagnifier**: Magnifying glass effect
- **SideBySideMagnifier**: Zoomed view beside image
- **PictureInPictureMagnifier**: Small zoom window
- **MagnifierContainer**: Links preview and zoom

## Use Cases

- E-commerce product images
- Photo galleries
- Stock photo sites
- Detailed image inspection
- Product catalogs
- Real estate photos
