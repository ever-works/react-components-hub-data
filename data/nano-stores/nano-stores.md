## Overview

Nano Stores is a minimalist state management library that provides an ultra-lightweight approach to managing application state. It is framework-agnostic with dedicated integration for React, offering a fast and predictable store pattern with a small bundle footprint.

## Features

- Ultra-lightweight (~1 KB)
- Framework-agnostic with React integration
- Computed stores for derived state
- Persistent stores for localStorage
- Atoms for simple values
- Maps for object state
- Clean and intuitive API
