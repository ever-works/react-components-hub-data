## Overview

reaflow is a React library designed for building workflow editors, flow charts, and diagrams. It provides a set of composable components that make it easy to create node-based visual interfaces with automatic layout, drag-and-drop, and extensive customization options.

## Features

- Node-based diagram editing
- Automatic layout algorithms (tree, layered, etc.)
- Drag and drop node creation and repositioning
- Customizable node and edge rendering
- Zoom and pan controls
- Selection and multi-selection support
- Port-based node connections
- Built with TypeScript
- React component composition
