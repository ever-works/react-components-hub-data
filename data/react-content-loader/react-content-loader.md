## Overview

react-content-loader is an SVG-based skeleton loading component library that provides presets and supports complex custom shapes. It's highly customizable and perfect for creating engaging loading experiences.

## Features

- SVG-based rendering
- Prebuilt presets (Facebook, Instagram, Code, List)
- Custom shape support
- Animation options (pulse, wave)
- Responsive design
- Customizable colors
- Speed control
- Unique key generation
- TypeScript support
- React Native support
- Style customization
- Gradient backgrounds
- RTL support
- Accessibility features

## Preset Templates

- Facebook-styled cards
- Instagram-styled feeds
- Code blocks
- List items
- Custom SVG shapes

## Use Cases

- Content placeholders
- Loading states
- Data fetching
- Image loading
- List rendering
- Feed displays
