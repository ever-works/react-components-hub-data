## Overview

React Spinners is a comprehensive collection of loading spinner components for React, providing 30+ different spinner styles with full customization options.

## Key Features

- **30+ Spinners**: Various spinner designs
- **Customizable**: Size, color, speed
- **CSS Animations**: Smooth animations
- **Simple Props**: Easy configuration
- **No Dependencies**: Pure CSS animations
- **TypeScript**: Type definitions
- **Accessibility**: ARIA labels

## Configuration

- color: Spinner color
- size: Spinner size
- speedMultiplier: Animation speed
- loading: Show/hide
- css: Custom CSS
- radius: Border radius
- margin: Margin around spinner
