## Overview

Fortress is a Bloomberg-inspired institutional finance dashboard built for trading desks, risk management teams, and compliance officers who need dense, data-heavy interfaces.

## Features

- 57+ pages
- 13 finance-specific pages covering multi-asset trading (Fixed Income, FX, Derivatives)
- Risk analytics dashboards
- Regulatory compliance monitoring
- Dense, data-heavy interface design
- Framer Motion animations
- Vitest + Playwright testing
- Storybook documentation
- React + Next.js + Tailwind CSS
