## Overview

Muuri React brings the powerful Muuri layout engine to React applications. It supports drag-and-drop, sorting, filtering, and responsive reflow with smooth position animations.

## Features

- Responsive grid layouts with automatic reflow
- Drag-and-drop item repositioning
- Sortable grid items with animated transitions
- Filterable grids with show/hide animations
- Smooth layout transitions powered by Web Animations API
- Declarative React API
- Works with virtualized and large grids
