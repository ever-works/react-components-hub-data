## Overview

Keen Slider is a modern, lightweight slider library focused on performance and smooth mobile interactions. It's optimized for speed and provides an excellent touch experience across all devices.

## Features

- **Lightweight**: Small bundle size with minimal overhead
- **Zero Dependencies**: No external library requirements
- **Touch Optimized**: Smooth, native-feeling touch interactions
- **Performance**: 60fps animations with hardware acceleration
- **React Support**: Official React integration
- **TypeScript**: Full TypeScript support
- **Plugin System**: Extend functionality with plugins
- **SSR Compatible**: Works with Next.js and other SSR frameworks
- **Responsive**: Built-in breakpoint support

## Slider Modes

- Free mode (snap to any position)
- Snap mode (snap to slides)
- Carousel mode (infinite loop)
- Vertical mode
- RTL support

## Key Capabilities

- Multiple slides per view
- Variable slide widths
- Centered slides
- Spacing between slides
- Custom animation easing
- Drag resistance
- Mouse and touch events
- Keyboard navigation

## Plugins

- Navigation (arrows)
- Dots/bullets
- Autoplay
- Zoom
- Mutations observer

## Performance Features

- Hardware-accelerated CSS transforms
- RequestAnimationFrame for smooth updates
- Optimized event listeners
- Minimal reflows and repaints
- Efficient memory usage

## Mobile Experience

Keen Slider excels at providing native-like touch interactions:
- Momentum scrolling
- Rubber band effect
- Touch feedback
- Gesture recognition

- Mobile-first applications

Official packages available for React, Vue, and Angular. Works seamlessly with modern build tools and frameworks.
