## Overview

React Bits is not a traditional library but a large collection of animated React components made to spice up web creations. It provides a comprehensive set of animations, components, backgrounds, and visual effects.

## Features

- 90+ total components and growing weekly
- Categories include text animations, general animations, interactive components, and animated backgrounds
- All components have minimal dependencies
- Highly customizable through props
- Free and open-source
