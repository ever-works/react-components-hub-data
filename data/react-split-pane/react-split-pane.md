## Overview

React Split Pane provides a component for creating resizable split views with draggable dividers. Users can adjust the size of panes by dragging the separator between them.

## Features

- Horizontal split
- Vertical split
- Resizable panes
- Draggable divider
- Min/max pane sizes
- Default pane size
- Nested split panes
- Controlled/uncontrolled
- Persist sizes
- Snap to positions
- Custom divider
- Step resizing
- Disable resizing

## Layout Options

- Vertical split (side-by-side)
- Horizontal split (top-bottom)
- Nested splits (complex layouts)
- Multiple panes
- Mixed orientations

## Size Control

- Pixel values
- Percentage values
- Min size constraints
- Max size constraints
- Default sizes
- Initial sizes

## Divider

- Customizable appearance
- Hover effects
- Dragging state
- Custom width/height
- Custom cursor
- Touch support

## Events

- onChange - size changes
- onDragStarted
- onDragFinished
- onResizerClick
- onResizerDoubleClick

## Persistence

- Save pane sizes
- LocalStorage integration
- Restore on mount
- Session persistence

## Customization

- Custom resizerStyle
- Custom paneStyle

- Styled components
- Theme support
