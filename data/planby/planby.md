## Overview

Planby is a React component designed to help create schedules, timelines, and electronic program guides (EPG) for online TV, video-on-demand services, music events, sporting events, and more. Built specifically for React with superior performance.

## Features

- Timeline and EPG layouts
- Customizable appearance
- Responsive design
- Event scheduling
- Program guide support
- Time-based layouts
- Drag and drop support
- Custom renderers
- Real-time updates
- Timezone support
- Lightweight and fast
- TypeScript support
- React Native compatible
- Easy integration
- Trusted by 100+ companies

## Use Cases

- TV program guides (EPG)
- Video-on-demand schedules
- Music event timelines
- Sports event calendars
- Broadcast scheduling
- Event planning
- Conference schedules
- Class timetables
