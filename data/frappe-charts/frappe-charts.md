## Overview

Frappe Charts is a lightweight, modern SVG charting library with zero dependencies. It provides GitHub-inspired simple and beautiful charts that are easy to integrate into web applications. Supports React, Vue, Svelte, and vanilla JavaScript frameworks.

## Features

- Zero dependencies
- SVG-based rendering for crisp visuals
- Clean, GitHub-inspired design aesthetic
- Supports line, area, bar, and heatmap charts
- Easy API for quick integration
- Responsive and adaptable to container sizes
- Works with React, Vue, Svelte, and vanilla JS
