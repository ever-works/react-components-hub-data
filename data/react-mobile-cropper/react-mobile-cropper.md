## Overview

A mobile-optimized image cropping component for React, providing an experience similar to popular Android image croppers.

## Features

- Android-inspired cropping interface
- Touch-optimized interactions
- Built on top of react-advanced-cropper
- Ready-to-use out of the box
- Responsive design for mobile screens
- Supports image zooming and panning within crop area
- Customizable output format and size

## Pricing

Free and open-source.
