## Overview

Univer is an open-source spreadsheet, document, and presentation framework built with TypeScript that enables seamless integration into modern web applications.

## Features

- Real-time collaborative editing
- High-performance rendering
- Full formula engine support
- Pivot tables
- Cross-document integration (spreadsheets, documents, presentations)
- Extensible plugin system
- Built with TypeScript for modern web frameworks
- Apache-2.0 license
