## Overview

react-infinite-scroll-hook is a React hook that provides infinite scroll functionality using the IntersectionObserver API. It offers a simple, performant way to load more content as users scroll.

## Features

- Uses IntersectionObserver for performance
- Works with window scroll
- Works with scrollable containers
- Hook-based API
- TypeScript support
- Loading state management
- Error handling support
- Ref-based sentinel element
- React 19 compatible
- Cleanup functions for refs
- Minimal configuration
- No dependencies
- Small bundle size

## How It Works

1. Create a sentinel element ref
2. Observer watches when sentinel enters viewport
3. Triggers data loading callback
4. Automatically handles loading states
