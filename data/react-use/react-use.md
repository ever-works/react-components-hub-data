## Overview

React Use is a collection of React hooks that covers most of the use cases developers encounter in modern React applications.

## Key Features

- **Lifecycle Hooks**: useEffect simplifications, useMount, useUnmount
- **State Hooks**: useState variants with persistence
- **Async Hooks**: useAsync, useFetch for data loading
- **Form Hooks**: useInput, useCheckbox for form state management
- **Side Effect Hooks**: useDebounce, useThrottle
- **DOM Hooks**: useLocalStorage, useSessionStorage
- **Media Hooks**: useMedia, useGeolocation
- **UI Hooks**: useWindowSize, useScroll
- **Full TypeScript Support**
- **SSR Compatible**

## Popular Hooks

- useAsync: Manage asynchronous operations
- useFetch: Simple data fetching
- useLocalStorage: Persist state to localStorage
- useMount: Run code on component mount
- useUpdateEffect: useEffect that skips initial render
