## Overview

Reaviz provides a comprehensive set of declarative chart components for React applications, combining D3.js's data manipulation with React's component model.

## Key Features

- **Chart Types**: Bar, line, area, scatter, pie, sankey, heatmap, treemap, and 14+ more.
- **Interactive Features**: Zooming, panning, tooltips, and gesture support.
- **Customization**: Full control over colors, animations, and axis configurations.
- **Accessibility**: ARIA-compliant components with keyboard navigation.
- **Performance**: Virtualized rendering for large datasets.
