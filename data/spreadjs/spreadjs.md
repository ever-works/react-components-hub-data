## Overview

SpreadJS (formerly GrapeCity SpreadJS) is a high-performance spreadsheet component designed for JavaScript applications, offering deep Excel compatibility and enterprise-grade features.

## Features

- Full Excel compatibility
- Advanced formula engine
- Custom functions support
- High-performance rendering
- Spreadsheet-like UI with cell editing
- Data binding and validation
- Import/export Excel files
