## Overview

ECharts for React is a React wrapper for Apache ECharts, a powerful, intuitive, and interactive charting and visualization well-suited for commercial presentations. It is written in JavaScript and uses the ZRender rendering engine that supports both Canvas and SVG.

## Features

- React wrapper for Apache ECharts
- ZRender rendering engine supporting Canvas and SVG
- ECharts 5 enhancements in storytelling and data appearance
- Augmented visualization capabilities
- Responsive design support
- Collaboration and performance enrichment
- Improved programmer experience
- Globalization support
- Powerful and intuitive interactive charting
- Well-suited for commercial presentations
