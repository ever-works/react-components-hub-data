## Overview

Muze is a free data visualization library by Charts.com that leverages WebAssembly for performance. It provides a composition layer for building interactive, cross-tabular data visualizations, enabling developers to create complex exploratory charts with connected cross-chart interactions.

## Features

- WebAssembly-powered rendering for performance
- Cross-tabular data visualization composition
- Connected cross-chart interactions (brushing, linking)
- Free and extensible architecture
- Support for exploratory data analysis workflows
- Works with React and vanilla JS
