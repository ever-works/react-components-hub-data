## Overview

React PDF Viewer is a React component to view a PDF document. It's written in TypeScript and powered by React hooks completely. The component is built on top of PDF.js with a focus on providing a rich viewing experience.

## Features

- Zoom in/out and fit to page
- Search text within document
- Thumbnail sidebar
- Bookmarks/outline navigation
- Page navigation controls
- Full-screen mode
- Rotation controls
- Print functionality
- Download PDF
- Dark mode support
- Annotation tools
- Customizable toolbar
- Plugins architecture
- Lazy page rendering
- Responsive design
