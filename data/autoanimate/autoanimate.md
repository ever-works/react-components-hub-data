## Overview

AutoAnimate is a zero-configuration animation library that automatically animates DOM changes. It detects additions, removals, and reordering of elements and smoothly animates them.

## Features

- Zero configuration required
- Automatic DOM change detection
- Add/remove/move animations
- React hook (useAutoAnimate)
- Lightweight (~2KB)
- TypeScript support
- Customizable duration
- Easing options
- No dependencies
- Framework agnostic
- Simple API
- Performance optimized
- Works with lists
- Smooth transitions

## React Integration

```javascript
import { useAutoAnimate } from '@formkit/auto-animate/react'
const [parent] = useAutoAnimate()
```

## Use Cases

- List animations
- Todo lists
- Shopping carts
- Search results
- Dynamic forms
- Card grids
- Any dynamic content
