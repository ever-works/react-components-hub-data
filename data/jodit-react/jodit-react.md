## Overview

Jodit is a lightweight, fast WYSIWYG HTML editor built with vanilla JavaScript and no dependencies. It offers a modern editing experience with extensive features while maintaining a small footprint.

## Features

- Lightweight and fast
- No dependencies
- Mobile-friendly
- Touch support
- Inline editing mode
- Iframe mode
- Source code editor with syntax highlighting
- Full screen mode
- Responsive design
- Image upload and handling
- Drag and drop images
- Paste handling
- Table editing
- Insert special characters
- Insert emoticons
- Link management
- Find and replace
- Spell checker integration
- Undo/redo
- Multiple language support
- RTL text support
