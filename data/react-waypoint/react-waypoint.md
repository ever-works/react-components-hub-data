## Overview

React Waypoint is a component that executes a callback function when you scroll to it. It's useful for lazy loading content, infinite scrolling, triggering animations, and tracking analytics events.

## Features

- **Enter/Exit Callbacks**: Fire callbacks on scroll events
- **Position Tracking**: Track element position in viewport
- **Configurable Offset**: Adjust trigger points
- **Horizontal Support**: Works for horizontal scrolling
- **Nested Waypoints**: Support multiple waypoints
- **Throttling**: Performance optimization
- **TypeScript**: Type definitions included

## Use Cases

- Infinite scrolling
- Lazy loading images and content
- Scroll-triggered animations
- Analytics event tracking
- Progress indicators
- Sticky navigation
