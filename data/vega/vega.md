## Overview

Vega is a visualization grammar, a declarative language for creating, saving, and sharing interactive visualization designs. Vega provides a JSON-based grammar for specifying visualizations, making it easy to describe complex charts and data transformations declaratively.

## Features

- Declarative JSON grammar for visualization design
- Data transformation pipeline (filter, aggregate, bin, etc.)
- Scales, axes, and legends with automatic configuration
- Interactive features: tooltips, brushes, selections, and event handling
- Marks: rect, symbol, line, area, text, image, arc, path
- Composable with Vega-Lite for simpler specifications
- Server-side rendering support with Vega Canvas
- Works with React and vanilla JS/TS
- TypeScript definitions included
