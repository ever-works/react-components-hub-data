## Overview

jsx-email is a React-based toolkit for building HTML emails. It allows developers to write email templates using familiar React components and JSX syntax, then compile them to standards-compliant HTML that renders correctly across email clients.

## Features

- Build emails with React components and JSX
- Component-based email templates for reusability
- Compiles to table-based HTML compatible with all major email clients
- Support for Tailwind CSS-like styling via utility classes
- Preview emails locally during development
- TypeScript support for type-safe templates
- CLI tools for building, previewing, and sending emails
- Works with providers like Resend for email delivery
