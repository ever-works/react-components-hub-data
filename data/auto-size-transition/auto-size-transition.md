## Overview

Auto Size Transition provides animated transitions for containers that dynamically resize based on their child content, without the layout jumps typically associated with height changes.

## Features

- Automatically measures child content dimensions
- Smooth transitions for height and width changes
- Uses requestAnimationFrame for performant animations
- No layout thrashing
- Configurable animation duration and easing
- Supports nested transitions
- Works with conditional rendering

## Use Cases

- Collapsible panels
- Expandable accordions
- Animated dropdowns
- Content disclosure patterns
- Dynamic content areas
