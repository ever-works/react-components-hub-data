## Overview

Flux brings a gradient-forward design language to the React admin dashboard space, purpose-built for SaaS and startup teams wanting visual flair.

## Features

- 64+ pages across five dashboard variants
- 7 specialty pages: Product Roadmap, Feature Flags, Deployment Log
- Over 300 color combinations
- Dark/light/system themes
- RTL support
- Command Palette for quick navigation
- i18n support for internationalization
- 35 shadcn/ui primitives
- Framer Motion for page transitions and micro-interactions
- Clear visual hierarchy across dense data layouts
