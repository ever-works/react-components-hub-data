## Overview

react-use-websocket provides a simple React Hook interface for WebSocket connections. It handles common WebSocket patterns like reconnection logic, message queuing, and connection state management.

## Features

- **Auto Reconnection**: Configurable reconnection strategies
- **Message Queue**: Queue messages when disconnected
- **Connection State**: Track connection status
- **TypeScript**: Full type safety
- **Event Callbacks**: onOpen, onClose, onMessage, onError
- **Shared Connections**: Share WebSocket across components
- **Filter Messages**: Built-in message filtering

## Use Cases

- Real-time chat applications
- Live data feeds and dashboards
- Collaborative editing tools
- Gaming applications
- Stock trading platforms
- IoT dashboards
