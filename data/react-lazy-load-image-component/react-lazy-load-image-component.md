## Overview

react-lazy-load-image-component provides React components for lazy loading images. It improves initial page load performance by only loading images when they enter the viewport.

## Features

- **Lazy Loading**: Load images on demand
- **Placeholder Support**: Show placeholders while loading
- **Effects**: Fade, blur, opacity effects
- **IntersectionObserver**: Modern API usage
- **Threshold**: Control when loading starts
- **SSR Support**: Server-side rendering compatible
- **TypeScript**: Full type definitions
- **Lightweight**: Small bundle size

## Use Cases

- Image galleries and portfolios
- Long-scrolling pages
- E-commerce product listings
- Blog posts with images
- Performance optimization
- Mobile optimization
