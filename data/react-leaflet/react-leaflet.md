## Overview

React Leaflet is a React wrapper around Leaflet, a popular open-source mapping library. It brings the power of Leaflet to React with a declarative component API, making it easy to build interactive maps.

## Features

- Declarative React components for maps
- Support for multiple tile providers (OpenStreetMap, Mapbox, etc.)
- Markers, popups, and custom overlays
- Layer management
- Geofencing and polygon drawing
- Zoom and pan controls
- Responsive design
- TypeScript support
- Full Leaflet plugin support
- Performance optimized

## Core Components

- MapContainer (main map component)
- TileLayer (background map tiles)
- Marker (point on map)
- Popup (informational overlay)
- Tooltip (hoverable information)
- Circle, Rectangle, Polygon
- LayerGroup, FeatureGroup
- ZoomControl, ScaleControl

## Installation

```bash
npm install react-leaflet leaflet
```
