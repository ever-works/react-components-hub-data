## Overview

React Redux Toastr is a notification component library designed for React applications, with built-in Redux integration for unified state management. It provides a robust and consistent approach to handling notifications throughout the application.

## Features

- Seamless Redux integration for state management
- Support for various notification types (success, error, warning, info)
- Consistent behavior across the application through unified state
- Toast-style notifications with customizable options
- Simple and straightforward API
- Easy integration with existing React/Redux projects
- Confirmation dialogs support

## Installation

```bash
npm install --save react-redux-toastr
```
