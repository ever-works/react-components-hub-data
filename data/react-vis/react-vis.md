## Overview

React Vis provides a collection of declarative visualization components for creating complex data visualizations in React.

## Key Features

- **Chart Types**: Line, bar, pie, radar, and over 15+ chart types.
- **Custom Scales**: Support for linear, log, time, and categorical scales.
- **Interactivity**: Built-in zooming, brushing, and animated transitions.
- **SVG Rendering**: High-quality SVG-based visualizations with responsive design.
- **Accessibility**: ARIA roles and keyboard navigation support.
