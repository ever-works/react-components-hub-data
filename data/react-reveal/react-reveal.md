## Overview

React Reveal is an animation framework for React that makes it easy to add scroll-triggered reveal animations. While still functional and popular, newer alternatives like React Awesome Reveal may be preferred for new projects.

## Status Note

**⚠️ Development has slowed.** Consider alternatives:
- React Awesome Reveal (Intersection Observer based)
- Framer Motion
- React Spring

## Features

- Scroll-based animations
- Multiple animation effects
- Cascade animations
- Custom delays
- Configurable duration
- Server-side rendering
- High performance
- TypeScript definitions available
- Flexible configuration

## Animation Components

- Fade
- Flip
- Rotate
- Zoom
- Bounce
- Slide
- Roll
- LightSpeed
- Reveal (custom)

## Configuration Options

- effect - animation type
- duration - animation duration
- delay - start delay
- count - repeat count
- forever - infinite loop
- cascade - stagger children
- wait - wait before animating
- fraction - visible fraction trigger
- refProp - custom ref prop

## Cascade Effect

- Stagger child animations
- Sequential reveals
- Configurable delay
- Works with lists
- Natural flow

## Trigger Configuration

- onReveal - reveal callback
- when - conditional trigger
- spy - re-trigger on change
- fraction - visibility threshold
