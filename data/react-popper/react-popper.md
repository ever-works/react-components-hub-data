## Overview

React Popper is a React wrapper around the Popper.js positioning engine, providing components and hooks for building positioned overlays like dropdowns, tooltips, and popovers.

## Key Features

- **Smart Positioning**: Auto-detect best position
- **Collision Detection**: Avoid viewport edges
- **Flip Strategy**: Flip when needed
- **Offset Control**: Custom spacing
- **Arrow Support**: Directional arrows
- **React Hooks**: Hook-based API
- **TypeScript**: Full type support
- **Modifiers**: Extensible modifier system

## Components & Hooks

- usePopper: Main positioning hook
- Popper: Positioned element component
- Reference: Reference element
- Arrow: Arrow element

## Features

## Modifiers

- offset: Add spacing
- flip: Flip placement
- preventOverflow: Contain in viewport
- arrow: Arrow positioning
- computeStyles: Style generation
- applyStyles: Apply to DOM
- hide: Hide when clipped
