## Overview

Re-Flex is a collection of React components that implement various types of flex layout containers with resizable panels.

## Features

- Flex-based layout containers with resizable panels
- Smooth and fast rendering using the browser's native flexbox
- ReflexContainer for primary and secondary layout regions
- ReflexSplitter for drag-to-resize functionality
- ReflexElement with proportional sizing support
- Can be nested for complex layouts
- Horizontal and vertical orientations
- Lightweight with no external dependencies
