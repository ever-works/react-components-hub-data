## Overview

Docusaurus is an open-source static site generator designed for building documentation websites. Built with React and Markdown, it provides a streamlined workflow for creating, maintaining, and deploying developer documentation.

## Features

- Markdown-first content authoring with MDX (Markdown + JSX) support
- Built-in versioned documentation for managing multiple release versions
- Internationalization (i18n) with support for multiple languages
- Full-text search powered by Algolia DocSearch
- Themable with built-in Infima design system and custom CSS support
- Blog functionality included out of the box
- Static HTML generation with fast load times and SEO optimization
- Plugin architecture for extending functionality
- React components embedded directly in documentation pages
