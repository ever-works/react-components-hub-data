## Overview

Auth.js (formerly NextAuth.js) is one of the most popular open-source authentication solutions for React applications. It provides a comprehensive authentication system with support for credentials, OAuth, email magic links, and more, with deep integration for full-stack frameworks.

## Features

- Support for 50+ authentication providers
- OAuth 2.0 and OpenID Connect support
- Email/password and magic link authentication
- Session management with JWT or database sessions
- Built-in CSRF protection
- Callbacks for custom authentication logic
- Works with Next.js, other frameworks, and standalone
