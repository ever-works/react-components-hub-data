## Overview

React Placeholder Shimmer provides placeholder components with shimmer loading effects. The library includes pre-built elements and containers for creating engaging loading states.

## Components

- RoundImage: Circular image placeholder
- SquareImage: Square image placeholder
- TextBlock: Text line placeholders
- Button: Button placeholder
- MediaBlock: Container for layout alignment

## Features

- Shimmer animation effects
- Pre-built placeholder shapes
- Customizable sizes
- Color customization
- Animation speed control
- Multiple placeholders support
- Responsive design
- Zero dependencies
- Lightweight
- Easy integration

## Shimmer Effect

- Smooth gradient animation
- Customizable colors
- Direction control
- Speed adjustment
- Pause on hover option

## Customization

- Width and height
- Border radius
- Background color
- Shimmer color
- Animation duration
- Gap between elements

## Layout

- Flex container support
- Grid layouts
- Vertical stacking
- Horizontal alignment
- Custom spacing

## Use Cases

- Image loading states
- Text content loading
- Card skeletons
- List placeholders
- Profile loading
- Feed skeletons
- Dashboard loading
