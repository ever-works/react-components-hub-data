## Overview

React Datatable Component (react-data-table-component) is a simple, declarative React data table library. It provides an intuitive API for rendering tabular data with core features like sorting, filtering, selection, and pagination.

## Features

- Declarative column and data configuration
- Column sorting with custom sort functions
- Filtering and search functionality
- Pagination with customizable options
- Row selection with checkboxes
- Expandable rows (expandable content panels)
- Custom cell formatting and conditional styling
- Responsive and accessible layout
- Sub-header with custom components
- Context menu and action support
- No external dependencies
- TypeScript definitions included
