## Overview

react-router-sitemap is a module for generating sitemap.xml files using your React Router configuration. It automates sitemap creation for better SEO by parsing your routes.

## Features

- Generates sitemap.xml from routes
- React Router integration
- Path filtering
- Dynamic param replacement
- Custom priorities
- Change frequency settings
- Last modification dates
- Multiple sitemap support
- Custom transformations
- Route exclusion
- TypeScript support
- CLI support
- Easy integration

## Configuration Options

- Priority levels
- Change frequency (always, hourly, daily, etc.)
- Last modified dates
- Hostname
- Path filters
- Route transformers

## Use Cases

- SEO optimization
- Search engine indexing
- Website crawling
- Route documentation
- Site structure mapping
- Automated sitemap updates
