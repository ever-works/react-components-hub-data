## Overview

tRPC is a TypeScript-first API framework that enables end-to-end type safety between clients and servers. It allows you to call your backend from your frontend using typed functions, providing a tremendous productivity boost and developer experience for tightly coupled client-server architectures.

## Features

- End-to-end type safety with zero code generation
- Works seamlessly with React
- Can be combined with TanStack Query for data fetching benefits
- Requires TypeScript on both frontend and backend
- Supports procedures with input validation
- Built-in error handling
- Lightweight and fast
