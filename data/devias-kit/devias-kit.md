## Overview

Devias Kit is a free, open-source admin dashboard starter template built with Material UI. It provides a minimal set of pages and components to get a project started, with an option to upgrade to a paid version for access to 80+ user pages.

## Key Features

- **Built on Material UI:** Uses the Material UI component library
- **Minimalist Starter:** Includes a dashboard, user list, settings, and login pages
- **Free and paid versions:** Try out the free version before opting into a paid version
- **Authentication:** Includes authentication logic with Clerk, Auth0, AWS Cognito/Amplify, and Supabase
- **Form validation:** Forms built with React Hook Form and validated via Zod schemas

## Pros

- Functional authentication with multiple providers
- Form validation with React Hook Form and Zod
- Free version available for evaluation

## Cons

- Dated Material Design look compared to modern standards
- Customizing built-in components can be challenging due to Material UI lock-in

## Pricing

- Free version available
- Paid version from $69 with 80+ additional user pages
