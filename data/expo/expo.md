## Overview

Expo is the most popular framework for building React Native applications. It provides a managed workflow that simplifies native module setup while still offering access to native APIs through its extensive SDK and plugin system.

## Features

- Managed workflow with minimal native configuration
- Over-the-air updates without app store review
- Expo SDK with pre-built native modules
- Expo Go for testing on physical devices
- EAS Build for native builds in the cloud
- Built-in support for push notifications, camera, location, and more
- Development tools and debugging via Expo DevTools

## Pricing

Core Expo SDK is free and open-source. EAS Cloud Build and additional hosting services have free tiers and paid plans.
