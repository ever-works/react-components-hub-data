## Overview

Mobiscroll is a feature-rich UI component library focused on advanced scheduling, calendar, and date/time picking functionality. It provides components that work seamlessly across desktop and mobile devices with native-feeling interactions.

## Core Components

- Event Calendar with multiple views
- Scheduler with resource management
- Date & Time Pickers
- Select dropdowns
- Form components
- Popup and notifications
- Range selectors
- Color picker
- Calendar sync (Google, Outlook)

## Calendar & Scheduler Features

- Day, Week, Month, Year views
- Agenda and Timeline views
- Resource scheduling
- Drag and drop events
- Recurring events
- Timezone support
- Event templates
- Touch gestures
- Responsive design
- Dark mode
