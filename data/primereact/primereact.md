## Overview

PrimeReact is a rich set of open source UI components for React, featuring a wide range of highly configurable components suitable for creating applications that span a spectrum of use cases.

## Key Features

- 100+ customizable components
- Built-in themes (30+ themes)
- Dark mode support
- Complete form components
- Data grid and table components
- Advanced data visualization
- Accessibility support
- TypeScript support

## Component Categories

- Input: InputText, InputNumber, Dropdown, Autocomplete
- Button: Button, SplitButton, Speed Dial
- Panel: Panel, Fieldset, Card, Splitter
- Data: DataTable, TreeTable, Tree, DataView
- Overlay: Dialog, Sidebar, OverlayPanel, Toast
- Menu: Menu, Menubar, ContextMenu, Breadcrumb
- Chart: BarChart, LineChart, PieChart, RadarChart
