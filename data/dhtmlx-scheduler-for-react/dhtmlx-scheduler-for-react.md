## Overview

DHTMLX Scheduler for React is a feature-rich scheduling component built specifically for React 18+. It provides a comprehensive solution for event management with support for complex scheduling scenarios.

## Features

- Multiple view modes (Day, Week, Month, Year, Agenda, Timeline, Units, Grid)
- Drag and drop events
- Resize events
- Recurring events with complex patterns
- Multi-day and multi-section events
- Resource/unit scheduling
- Custom event rendering
- Event templates
- Lightbox for event details
- Quick info popup
- Collision detection
- Keyboard navigation
- Undo/redo functionality
- Export to PDF, Excel, iCal
- Print functionality
- Responsive design
- Touch support
- RTL mode
- Localization (30+ languages)

## View Types

- Day - hourly day view
- Week - 7-day view
- Month - monthly calendar
- Year - yearly overview
- Agenda - list of events
- Timeline - horizontal resource view
- Units - vertical resource view
- Grid - custom grid layouts
- Map - events on map
