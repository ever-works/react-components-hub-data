## Overview

Braintree is a payment platform owned by PayPal that provides comprehensive payment processing for web applications, including React. It offers a full-stack payment solution with support for multiple payment methods and global currencies.

## Features

- Drop-in UI for quick integration
- Custom UI components for React
- Accept credit cards, PayPal, Venmo, Apple Pay, Google Pay
- Fraud protection and risk management
- Recurring billing and subscriptions
- Global payment support with multi-currency
- PCI-compliant payment processing
- GraphQL and REST APIs

## Pricing

Pay-per-transaction pricing: 2.59% + $0.49 per transaction for cards. No monthly fees for standard accounts.
