## Overview

TanStack Virtual provides headless UI for virtualizing only the visible content for massive scrollable DOM nodes at 60FPS while retaining 100% control over markup and styles. It's a modern, flexible alternative to react-window and react-virtualized.

## Features

- Lightweight (10-15kb) with tree-shaking
- Vertical/column virtualization
- Horizontal/row virtualization
- Grid virtualization
- Window scrolling support
- Fixed, variable, and dynamic sizing
- Scrolling utilities
- Sticky items
- Tiny API (literally 1 function)
- Headless - complete styling control
- Framework agnostic core
- TypeScript support
- Highly performant (60+ FPS)
- Composable and extensible
- Smooth scrolling
- Overscan support

## Supported Frameworks

- React
- Vue
- Solid
- Svelte
- Lit
- Angular
- Vanilla JS/TS

## Use Cases

- Large lists and tables
- Infinite scrolling
- Chat applications
- Data grids
- Feed displays
- Any UI with thousands of items
