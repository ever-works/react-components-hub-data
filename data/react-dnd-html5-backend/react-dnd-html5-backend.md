## Overview

React DnD HTML5 Backend is the primary backend for React DnD that uses the native HTML5 drag and drop API. It provides the core functionality for drag and drop operations in web browsers.

## Features

- **Native Browser Support**: Uses HTML5 drag and drop API
- **Cross-Browser**: Works across all modern browsers
- **File Drop Support**: Handle file drops from desktop
- **Preview Customization**: Custom drag previews and ghost images
- **Touch Alternative**: Can be combined with touch backend for mobile
- **Performance**: Efficient native implementation

## Use Cases

- Desktop web applications with drag and drop
- File upload interfaces
- Kanban boards and task managers
- Layout builders and designers
- Data visualization tools
