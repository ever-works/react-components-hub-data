## Overview

Lottie React enables seamless integration of vector animations exported from Adobe After Effects using the Bodymovin plugin. It renders high-quality, scalable animations with small file sizes.

## Features

- After Effects integration
- Vector-based animations
- Small file sizes (JSON)
- High-quality rendering
- Scalable graphics
- Playback controls
- Loop and autoplay
- Speed control
- Event callbacks
- TypeScript support
- Responsive animations
- Cross-platform (web, mobile)
- No dependencies
- Easy integration

## Animation Sources

- Adobe After Effects exports
- LottieFiles marketplace
- Custom JSON animations
- Bodymovin plugin exports

## Use Cases

- Loading animations
- Success/error states
- Onboarding flows
- Micro-interactions
- Illustrations
- Brand animations
- Empty states
