## Overview

Mo.js (Motion.js) is a motion graphics toolbelt designed for the web. It provides a modular set of tools for creating complex, visually rich animations including shape tweens, timeline management, and burst effects.

## Features

- Shape tweens with gradient fills and morphing capabilities
- Burst module for particle-like explosion animations
- Timeline for sequencing and coordinating multiple animations
- HTML module for animating DOM elements
- Custom easing functions including elastic, sine, and more
- Declarative API with chainable methods
- Yoyo, repeat, and delay controls for complex animation patterns
- Works with SVG and HTML elements

## Best Use Case

Creating complex motion graphics, particle effects, and rich visual animations for interactive web experiences.
