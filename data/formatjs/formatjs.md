## Overview

FormatJS is a comprehensive suite of internationalization libraries developed for JavaScript and TypeScript applications. Its primary React integration is through react-intl, which provides robust tools for translations, date/number formatting, and more.

## Features

- react-intl integration for React components
- ICU message syntax for complex translations
- Plural and gender formatting
- Date and number localization
- Relative time formatting
- TypeScript support
- Server-side rendering support
- Runtime and compile-time translation options
