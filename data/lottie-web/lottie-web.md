## Overview

Lottie is a cross-platform animation library originally created by Airbnb that renders After Effects animations in real-time using JSON bodymovin exports. The web version (lottie-web) supports SVG, Canvas, and HTML rendering.

## Features

- Renders After Effects animations natively using Bodymovin JSON exports
- Three rendering modes: SVG, Canvas, and HTML
- Supports play, pause, stop, and seek controls
- Loop and autoplay configuration
- Marker support for jumping to specific animation sections
- Responsive sizing
- Event listeners for animation lifecycle (complete, loop, enterFrame)
- Cross-platform (web, iOS, Android, React Native, Windows)
- Works with LottieFiles ecosystem for discovery and management

## Best Use Case

Integrating high-quality, complex animations designed in After Effects into web applications.
