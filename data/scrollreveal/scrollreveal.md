## Overview

ScrollReveal is a lightweight JavaScript library for animating elements as they scroll into view. It provides a simple API for adding reveal animations to headings, images, cards, and any DOM elements.

## Features

- Automatic viewport detection for triggering animations on scroll
- Configurable origin, distance, duration, easing, and delay
- Supports entering and exiting animations
- ViewFactor and viewPort offset controls
- Reset option for repeated animations on re-scroll
- Mobile-responsive with container and media query support
- Chainable API for applying to multiple elements
- Lightweight — under 5KB gzipped
- No external dependencies

## Usage

```javascript
import ScrollReveal from 'scrollreveal';
ScrollReveal().reveal('.box', { delay: 500 });
```

## Best Use Case

Adding scroll-based reveal animations to landing pages, portfolios, and content-heavy sites.
