## Overview

react-ga4 is a JavaScript module for Google Analytics 4 integration in React applications. It provides a simple API for tracking pageviews, events, e-commerce, and custom dimensions.

## Features

- **GA4 Support**: Latest Google Analytics version
- **Event Tracking**: Custom event tracking
- **E-commerce**: Enhanced e-commerce tracking
- **Custom Dimensions**: Track custom data
- **User ID**: Track authenticated users
- **React Router**: Easy router integration
- **TypeScript**: Full type definitions
- **Debugging**: Built-in debug mode

## Use Cases

- Website analytics
- User behavior tracking
- E-commerce analytics
- Conversion tracking
- Custom event tracking
- Marketing attribution
