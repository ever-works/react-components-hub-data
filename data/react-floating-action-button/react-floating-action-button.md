## Overview

React Floating Action Button is a component that implements the Material Design floating action button (FAB) pattern, providing a customizable button that floats above other content.

## Key Features

- **Material Design**: Follows Material Design spec
- **Expandable**: Can expand to show multiple actions
- **Customizable**: Full styling customization
- **Icon Support**: Custom icon rendering
- **Animation**: Smooth expansion animation
- **Responsive**: Mobile-friendly
- **TypeScript**: Type definitions

## Features

- Primary action button
- Multiple action items
- Expand/collapse animation
- Custom positioning
- Tooltip support
- Keyboard accessibility
- Mobile optimized
- Theme customization

## Configuration

- icon: Button icon
- actions: Array of action items
- color: Button color
- size: Button size
- position: Fixed positioning
- tooltip: Tooltip text
- onClick: Click handler

## Use Cases

- Call-to-Action Buttons
- Quick Actions
- Create/Add Actions
- Navigation

- Settings
