## Overview

react-charts (part of the TanStack ecosystem by Tanner Linsley) is a simple and modern charting library perfect for getting started quickly without heavy configurations. It delivers clean and modern visuals with a user-friendly API.

## Features

- **User-friendly**: Designed for easy setup and use, ideal for quick projects or beginners
- **Contemporary design**: Delivers clean and modern visuals without extra design input
- **Headless architecture**: Built with a composable, headless approach following the TanStack ecosystem patterns
- **TypeScript support**: Full type definitions for a robust developer experience

## Considerations

- **Feature gaps**: Excels at basic visualizations but lacks advanced features such as smooth animations for mixed chart types (e.g., combining line and bar charts)
- **Chart variety constraints**: Offers basic chart types like line and bar charts but does not support specialized types such as radar or waterfall charts
