## Overview

Nivo provides a rich set of dataviz components, built on top of the awesome D3 and React libraries.

## Key Features

- 30+ chart types
- Built on D3.js
- Motion and transitions
- Theming support
- Server-side rendering
- Responsive design
- Interactive charts
- Accessible
- API documentation
- Multiple formats (SVG, Canvas)

## Available Charts

- Bar, Line, Area, Scatter, Bubble
- Pie, Donut, Sunburst, Treemap
- Network, Sankey, Stream
- Heatmap, Calendar, Funnel
- Waffle, Geo
