## Overview

React Datasheet Grid is an Excel-like spreadsheet component that's simpler to use than a standard input element yet powerful enough to handle massive datasets. It's optimized for speed with virtualized rendering.

## Features

- Virtualized rendering for performance
- Handle 100,000+ rows efficiently
- Copy/paste from Excel
- Copy/paste from Google Sheets
- Copy/paste from Notion
- Keyboard navigation
- Cell selection
- Multiple cell selection
- Cut, copy, paste operations
- Delete operations
- Fill handle (drag to fill)
- Undo/redo support
- Custom cell types
- Column configuration
- Cell validation
- TypeScript support

## Built-in Cell Types

- Text cells
- Number cells
- Checkbox cells
- Select dropdown cells
- Date cells
- Custom cell types

## Column Configuration

- title: Column header
- component: Cell component
- disabled: Disable column
- width: Column width
- minWidth: Minimum width
- maxWidth: Maximum width

## Operations

- addRow: Add new row
- deleteRow: Remove row
- duplicateRow: Copy row
- setRowData: Update row
