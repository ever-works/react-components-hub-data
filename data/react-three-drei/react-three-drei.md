## Overview

@react-three/drei is a growing collection of useful helpers and fully functional abstractions for @react-three/fiber. It includes everything from camera controls to ready-made shapes, effects, and much more.

## Features

- 100+ ready-to-use abstractions and helpers
- Camera controls (OrbitControls, FlyControls, etc.)
- Loaders for various 3D formats (GLTF, FBX, OBJ)
- Text rendering components
- Environment and lighting helpers
- Material and shader utilities
- Reflector and portals
- HTML and sprite rendering
- Performance monitoring
- Staging and presentation components
- Post-processing effects
- Physics-ready components
- Animation helpers
- Geometric shapes and primitives
- TypeScript definitions
- Regularly updated with new components

## Popular Components

- OrbitControls - Mouse-based camera controls
- Environment - HDRI environment maps
- Text3D - 3D text rendering
- GizmoHelper - Axis helper gizmo
- Html - Render HTML in 3D space
- useGLTF - Easy GLTF model loading
- Sky - Procedural sky generation
- Stars - Starfield background
