## Overview

A comprehensive toolkit built on ProseMirror, designed specifically for React developers who need to build powerful and customizable rich text editors.

## Features

- Built on top of ProseMirror
- Declarative React API for editor configuration
- Extensible schema system
- Rich plugin ecosystem
- Collaborative editing support
- Interactive playground available
- Comprehensive documentation
- TypeScript support
- Framework-agnostic ProseMirror extensions with React bindings
