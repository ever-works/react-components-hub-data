## Overview

A modern WYSIWYG editor built on the powerful CKEditor engine, featuring a completely redesigned user interface focused on inline editing simplicity.

## Features

- Built on top of CKEditor engine for reliable content editing
- Completely rewritten UI for modern inline editing
- Inline editing toolbar that appears contextually
- Image and link management
- Undo/redo support
- Full-screen editing mode
- Plugin extensibility
