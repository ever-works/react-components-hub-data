## Overview

React Query handles data fetching, caching, and updating in React with a simple and intuitive API.

## Key Features

- **Query Caching**: Intelligent caching reduces API calls and bandwidth usage.
- **Pagination Support**: Built-in methods for paginated data fetching.
- **Background Updates**: Auto-reloads stale data in the background.
- **Optimistic UI**: Update UI before server response with rollback capability.
- **Error Handling**: Unified error boundaries for data operations.
