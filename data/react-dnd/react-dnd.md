## Overview

React DnD provides a declarative API for implementing drag-and-drop interactions in React applications.

## Key Features

- **Backends**: Supports HTML5, Touch, and HTML5 to avoid conflicts.
- **Flexible API**: Customizable drag/drop behavior with hooks and event handlers.
- **High Performance**: Optimized for performance with virtualized components.
- **Community-Driven**: Actively maintained with community-contributed drag layer implementations.
