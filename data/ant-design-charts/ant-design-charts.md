## Overview

Ant Design Charts is a charting library developed by the team behind the popular Ant Design UI framework. It integrates seamlessly with other Ant Design components, making it an excellent choice for developers already using the Ant Design ecosystem.

## Features

- Comprehensive range of chart types: line, bar, pie, funnel, radar, gauge, box plots, and waterfall charts
- Canvas rendering for efficient performance with large datasets
- Built-in tooltips, legends, and responsive layouts
- Robust theming capabilities for customizing look and feel
- Accessibility-focused design
- User experience optimized with responsive layouts that adapt to different screen sizes

## Technical Details

- React component syntax
- Canvas support
