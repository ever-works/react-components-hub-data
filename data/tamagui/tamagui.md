## Overview

Tamagui is a powerful performance-focused UI toolkit for React and React Native. It combines a comprehensive style library with a smart optimizing compiler to deliver fast, lightweight applications.

## Features

- Extracts CSS at build time for optimal performance
- Flattens component trees to boost runtime performance
- Full cross-platform component library for React and React Native
- Excellent TypeScript support
- Smart optimizing compiler
- Build sleek, high-speed apps with less code
