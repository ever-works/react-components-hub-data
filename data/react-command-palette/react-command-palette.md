## Overview

react-command-palette is an accessible, browser-compatible command palette component for React. It includes several themes inspired by popular editors and provides keyboard-driven navigation.

## Features

- Keyboard shortcut (Cmd/Ctrl+Shift+P)
- Built-in themes (Chrome, VS Code, Atom)
- Accessible design
- Fuzzy search
- Command grouping
- Custom commands
- Trigger callbacks
- Keyboard navigation
- Theme customization
- Command descriptions
- Icons support
- TypeScript definitions
- Browser compatible
- Easy integration

## Included Themes

- Atom (default)
- VS Code
- Chrome
- Custom themes

## Use Cases

- Application commands
- Quick navigation
- Editor shortcuts
- Settings access
- Developer tools
- Power user features
- Productivity tools
