## Overview

React Sortable HOC is a set of higher-order components to turn any list into an animated, touch-friendly sortable list. It provides simple, flexible sorting functionality for lists, grids, and other layouts.

## Features

- Horizontal, vertical, or grid sorting
- Touch support and mobile-friendly
- Smooth animations
- Auto-scrolling when dragging near edges
- Keyboard accessibility support
- Lock axis (constrain to horizontal/vertical)
- Nested sortables
- Variable heights/widths
- Custom drag handles
- Minimal dependencies
- Works with existing components
- TypeScript definitions
- Controlled and uncontrolled modes

## Higher-Order Components

- SortableContainer: Wraps the list/grid
- SortableElement: Wraps each sortable item
- SortableHandle: Optional drag handle

## Callbacks

- onSortStart: When sorting begins
- onSortMove: During sorting
- onSortEnd: When sorting completes
- onSortOver: When hovering over items

## Use Cases

- Reorderable todo lists
- Dashboard layouts
- Image galleries
- Playlist management
- Priority lists
