## Overview

react-loading-overlay is a simple, customizable loading overlay component. Wrap your components and toggle the active prop to show/hide a loading state with spinner and optional message.

## Features

- Simple toggle with active prop
- Default spinner included
- Custom spinner support
- Loading text/message
- Fade transitions
- Customizable styles
- Emotion/styled-components support
- Z-index control
- Background color customization
- Spinner size and color
- Wraps any component
- TypeScript support
- Lightweight bundle

## Configuration

- active: Show/hide overlay
- spinner: Custom spinner component
- text: Loading message
- fadeSpeed: Transition speed
- styles: Custom style object
- classNamePrefix: CSS class prefix

## Custom Spinners

- Any React component
- Third-party spinners
- SVG animations
- Custom CSS animations
- Icon libraries

## Styling

- Inline styles object
- CSS-in-JS (Emotion)
- Styled-components
- CSS modules
- Tailwind CSS compatible

## Use Cases

- Form submission loading
- Data fetching states
- File upload progress
- API call indicators
- Page transitions
- Content loading
- Modal loading states
