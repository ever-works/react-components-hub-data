## Overview

Lightweight Charts is a financial charting library built by TradingView using HTML5 canvas for high performance. It provides interactive candlestick, area, bar, and line charts optimized for displaying real-time financial data. The library renders smoothly even with large datasets and supports multiple frameworks.

## Features

- High-performance HTML5 canvas rendering
- Chart types: candlestick, area, bar, line, histogram, and baseline
- Real-time data updates
- Crosshair, tooltips, and price scale labels
- Multiple series on a single chart
- Timezone configuration and localization
- Full width and height adaptability
- TypeScript definitions included
- Works with React, Vue, Angular, and vanilla JS/TS
