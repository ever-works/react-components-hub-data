## Overview

SheetJS is a JavaScript library specializing in reading and writing spreadsheet files across multiple formats. While not a full UI component itself, it is frequently used together with data grids and spreadsheets to handle file operations.

## Features

- XLSX parsing and writing
- CSV import and export
- Supports multiple spreadsheet formats
- Data transformation utilities
- Client-side and server-side usage
- No UI component (focused on data layer)
