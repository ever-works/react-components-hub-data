## Overview

React QR Code is a simple component that generates QR codes from data, making it easy to create shareable, scannable codes in your React applications.

## Key Features

- **Simple API**: One-line QR code generation
- **Customizable**: Size, level, styling
- **SVG Output**: Scalpable vector graphics
- **Error Correction**: Multiple error levels
- **Dark/Light Mode**: Theming support
- **TypeScript**: Type support
- **No External Dependencies**: Lightweight

## Configuration

- value: Data to encode
- size: QR code size
- level: Error correction level (L, M, Q, H)
- includeMargin: Include quiet zone
- fgColor: Foreground color
- bgColor: Background color

## Error Correction Levels

- L: 7% recovery
- M: 15% recovery
- Q: 25% recovery
- H: 30% recovery
