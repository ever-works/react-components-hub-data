## Overview

A heart-shaped toggle switch component for React designed with full themeability and accessibility in mind.

## Features

- Heart-shaped toggle design with smooth animation
- Fully themeable appearance via props
- Accessible with proper ARIA attributes and keyboard navigation
- Small package size with zero runtime dependencies
- TypeScript support

## Pricing

Free and open-source under the MIT license.

## Installation

```bash
npm install @anatoliygatt/heart-switch
# or
yarn add @anatoliygatt/heart-switch
```
