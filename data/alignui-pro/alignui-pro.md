## Overview

AlignUI Pro includes premium React admin dashboard templates designed for professional applications. It is built with Tailwind CSS and offers a sleek, modern design with a wide range of components, making it suitable for high-end business applications.

## Tech Stack

- Tailwind CSS
- Radix UI
- React Aria
- React Day Picker

## Features

- Professional, modern design suitable for commercial projects
- Includes a matching Figma UI kit and design system
- Comprehensive UI components including charts, tables, forms, and application layouts
- Premium product with ongoing support
