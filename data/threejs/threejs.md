## Overview

three.js is the most widely used JavaScript 3D library. It provides an easy-to-use API for creating and displaying animated 3D computer graphics in web browsers, leveraging WebGL and WebGPU. It is commonly integrated with React through projects like React Three Fiber (@react-three/fiber).

## Features

- Full WebGL and WebGPU support
- 3D geometric primitives and custom geometry creation
- Advanced lighting systems (point, directional, spot, ambient, hemisphere)
- Material system with physically-based rendering (PBR)
- Post-processing effects pipeline
- Animation system with keyframe and morph animations
- Built-in loaders for GLTF, OBJ, FBX, and other 3D formats
- Physics and collision detection capabilities
- Large community ecosystem with extensive third-party addons
- TypeScript definitions included
- Works with React (via React Three Fiber), Angular, Vue, and vanilla JS
