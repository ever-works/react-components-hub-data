## Overview

Tiny Slider is a pure JavaScript slider/carousel solution with a tiny footprint of ~10kb gzipped. It works with all browsers and has no dependencies, making it one of the most lightweight carousel options available.

## Features

- **Tiny Size**: Only ~10kb gzipped
- **No Dependencies**: Pure vanilla JavaScript
- **Cross-Browser**: Works on all browsers
- **Touch Support**: Mobile swipe gestures
- **Keyboard Navigation**: Arrow key support
- **Responsive**: Breakpoint configuration
- **Vertical Mode**: Vertical carousel support
- **Auto Height**: Automatic height adjustment
- **Lazy Loading**: Built-in image lazy loading
- **ARIA Support**: Accessibility features

## Slider Modes

- Carousel (loop)
- Gallery (rewind)
- Responsive slides per view
- Center mode
- Edge padding
- Fixed width
- Auto width

## Performance

Optimized for performance with:
- CSS transforms for animations
- RequestAnimationFrame
- Efficient event handling
- Minimal DOM manipulation
- No jQuery overhead

## Browser Support

Supports all modern browsers and IE8+ (with classList polyfill).
