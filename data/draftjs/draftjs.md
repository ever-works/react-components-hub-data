## Overview

A rich text editor framework developed by Facebook that provides the building blocks for creating custom text editors with React.

## Features

- Declarative API built on React components
- Custom content model for representing editor state
- Supports rich text formatting, mentions, embeds, and more
- Immutable data structures for state management
- Extensible with custom block types and entity types
- Support for inline styles and custom rendering
- Entity system for complex content (links, mentions, images)
- Used as the foundation for many other editors
