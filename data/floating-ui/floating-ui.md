## Overview

Floating UI (formerly PopperJS) is a low-level positioning library for creating floating elements. It abstracts away the popper positioning logic and provides a simple API.

## Key Features

- Precise positioning
- Collision detection
- Automatic placement adjustment
- Offset control
- Shift, flip, slide modifiers
- Size modifiers
- TypeScript support
- Framework agnostic
- Lightweight
- Accessible by default

## Use Cases

- Tooltips
- Popovers
- Dropdowns
- Context menus
- Select menus
- Autocomplete lists
