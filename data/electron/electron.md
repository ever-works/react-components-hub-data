## Overview

Electron is a battle-tested framework for building cross-platform desktop applications using HTML, CSS, and JavaScript. It bundles a Chromium renderer and Node.js runtime, allowing React developers to create desktop apps with full access to native operating system APIs.

## Features

- Cross-platform support (Windows, macOS, Linux)
- Chromium-based rendering for consistent display
- Full Node.js API access
- Native OS integrations (menus, dialogs, notifications)
- Auto-updater built-in
- Developer tools and debugging
- Large ecosystem of existing apps (VS Code, Slack, Discord)
