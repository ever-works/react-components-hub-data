## Overview

React Flip Move uses the FLIP (First, Last, Invert, Play) animation technique to create smooth transitions when elements change position in the DOM. It is particularly well-suited for list reordering, filtering, and sorting animations.

## Features

- FLIP animation technique for smooth transitions
- Ideal for list reordering animations
- Automatic position tracking
- Works with Enter/Exit animations
- Customizable animation duration and easing
- Zero-config for common use cases
- Preserves component state during animations
