## Overview

react-image-crop is a responsive image cropping tool for React with no dependencies and a small footprint of less than 5KB gzipped. It supports both pixel and percentage-based cropping with full accessibility.

## Features

- No dependencies
- Lightweight (<5KB gzip)
- Responsive design
- Touch device support
- Keyboard accessible (a11y)
- Free-form cropping
- Fixed aspect ratio crops
- Min/max crop dimensions
- Circular crop areas
- Percentage or pixel units
- Image rotation support
- Crop change callbacks
- Complete crop data
- Custom styling
- TypeScript definitions

## Crop Configuration

- x, y: Crop position
- width, height: Crop dimensions
- unit: 'px' or '%'
- aspect: Fixed aspect ratio
- minWidth, maxWidth: Constraints
- minHeight, maxHeight: Constraints

## Callbacks

- onChange: Crop area change
- onComplete: Crop finalized
- onImageLoaded: Image ready
- onDragStart: Drag begins
- onDragEnd: Drag ends

## Use Cases

- Profile picture cropping
- Avatar editing
- Product image adjustment
- Photo editors
- Social media uploads
- ID document cropping
- Thumbnail generation
