## Overview

react-pdf allows you to display PDF files in your React application. It uses PDF.js under the hood to render PDFs in the browser with support for various input formats and rendering options.

## Features

- Display PDFs from various sources (URL, file, base64, Uint8Array)
- Render specific pages or all pages
- Text selection support
- Annotations rendering
- Custom page rendering
- Password-protected PDF support
- Print functionality
- Zoom and scaling controls
- TypeScript support
- SSR compatible
- Customizable loading and error states

## Components

- Document: Main wrapper component for PDF
- Page: Renders individual PDF pages
- Outline: Displays PDF document outline
- Thumbnail: Renders page thumbnails

## Use Cases

- Document management systems
- Report viewers
- E-book readers
- Invoice display
- Contract review applications
