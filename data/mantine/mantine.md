## Overview

Mantine is a fully featured React components library with 100+ components that support dark mode out of the box, created with focus on accessibility, developer experience, and customization.

## Key Features

- 100+ customizable components
- 50+ React hooks
- Full TypeScript support
- Dark mode by default
- Accessibility-first approach
- Composable component system
- Fully themeable
- No external styling dependencies

## Component Categories

- Form Inputs: TextInput, PasswordInput, Checkbox, Radio, Select
- Layout: Container, Grid, Stack, Flex, Center
- Navigation: Tabs, Breadcrumbs, Pagination, Navbar
- Overlays: Modal, Popover, Tooltip, Drawer
- Feedback: Alert, Notification, Progress, LoadingOverlay
- Data: Table, Avatar, Badge, Badge Group
