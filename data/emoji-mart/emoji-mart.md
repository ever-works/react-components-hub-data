## Overview

Emoji Mart is a highly customizable emoji picker component for React and other frameworks. It supports custom emojis, multiple skin tones, and can work with GIFs or SVGs, providing flexibility for various use cases.

## Features

- Custom emoji support
- Multiple skin tone variations
- GIF and SVG emoji support
- Category organization
- Search functionality
- Frequently used section
- Native emoji rendering
- Custom emoji datasets
- Internationalization support
- Keyboard navigation
- Preview on hover
- Click handlers
- Custom styles and themes
- Per-line height control
- Emoji size customization

## Custom Emojis

- Provide array of categories
- Define custom emoji objects
- Add GIF or SVG emojis
- Custom emoji IDs and names

## Data Options

- Multiple data sources
- CDN or local data
- Reduced dataset for performance
- Custom emoji sets

## Components

- Picker: Main emoji picker
- EmojiData: Data provider
- Custom category support

## Use Cases

- Corporate chat applications
- Branded emoji sets
- Custom reaction systems
- Multi-language platforms
- Accessibility-focused apps
- Gaming platforms
