## Overview

JS-Lingui is a readable, automated, and optimized (5 kb) internationalization for JavaScript, with excellent React integration.

## Features

- Lightweight runtime (~5 kb minified + gzipped)
- Readable ICU MessageFormat syntax
- Support for pluralization and message formatting
- Extract messages from JSX/JS sources
- Babel and Vite integrations
- TypeScript support
- React context provider for translations
- Pluralization and select formatting
- Message compilation and translation management
- CLI tools for message extraction and compilation
- Framework-agnostic with React bindings
