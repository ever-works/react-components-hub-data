## Overview

BlockNote is a React rich text editor that features a familiar block-based editing experience, similar to Notion. Built on top of ProseMirror and TipTap, it provides a powerful yet user-friendly interface for content creation.

## Features

- Block-based editing with drag-and-drop reordering
- Slash commands for quick formatting
- Real-time collaboration via Yjs
- AI integration capabilities
- Customizable blocks and extensions
- TypeScript support
- Fully accessible (WAI-ARIA)
- Light and dark themes
- Markdown shortcuts
- Inline formatting options
- Image and media embedding
- Export to Markdown and HTML

## Key Benefits

- Modern, intuitive editing experience
- Extensive customization options
- Strong TypeScript support
- Active development and community
- Built on proven technologies (ProseMirror, TipTap)
