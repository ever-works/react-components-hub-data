## Overview

react-simple-keyboard is a modern virtual keyboard component for React that's customizable, responsive, and lightweight. It's compatible with various frameworks and ideal for touch interfaces and kiosks.

## Features

- Fully customizable layouts
- Multiple language support
- Numeric keyboard
- Theme options
- Touch-friendly
- Mouse support
- Custom button actions
- Input masking
- Keyboard layouts (QWERTY, AZERTY, etc.)
- Shift and caps lock
- Special characters
- TypeScript support
- Lightweight bundle
- No dependencies
- Responsive design

## Keyboard Types

- Standard QWERTY
- Numeric pad
- Phone keypad
- Custom layouts
- Multi-language

## Use Cases

- Kiosk applications
- Touch screen interfaces
- Mobile web apps
- ATM interfaces
- Point of sale systems
- Public terminals
- Accessibility features
