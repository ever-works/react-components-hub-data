## Overview

LightGallery React is a comprehensive lightbox gallery component that supports images, videos, and various interactive features. It's the official React implementation of the popular LightGallery library.

## Features

- **Image Support**: Full-featured image lightbox
- **Video Support**: HTML5 video, YouTube, Vimeo, Wistia
- **Pinch to Zoom**: Touch zoom on mobile
- **Rotate & Flip**: Image manipulation
- **Social Sharing**: Share to social networks
- **Animated Thumbnails**: Smooth thumbnail transitions
- **Fullscreen**: Native fullscreen support
- **Autoplay**: Slideshow functionality
- **Responsive**: Mobile-optimized

## Media Types Supported

- Images (JPEG, PNG, GIF, WebP)
- HTML5 videos
- YouTube videos
- Vimeo videos
- Wistia videos
- Dailymotion
- iframes
- Inline HTML content

## Interactive Features

- Pinch and double-tap zoom
- Rotate images (90° increments)
- Flip horizontally/vertically
- Pan zoomed images
- Mouse wheel zoom
- Keyboard shortcuts
- Swipe navigation

## Customization
