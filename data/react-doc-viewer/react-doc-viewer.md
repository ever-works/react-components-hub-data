## Overview

react-doc-viewer is a file viewer component for React that supports viewing multiple document formats. It provides built-in renderers and accepts an array of document objects for display.

## Features

- Multiple format support
- Built-in renderers (PDF, PNG, JPEG, etc.)
- Document array input
- URI-based loading (URL or local)
- Individual renderer imports
- Custom renderer support
- Navigation between documents
- TypeScript support
- Responsive design
- Easy integration
- Lightweight package
- Error handling
- Loading states

## Supported Formats

- PDF documents
- Images (PNG, JPEG, GIF)
- Office documents
- Text files
- Custom formats via renderers

## Use Cases

- Document management systems
- File preview features
- Multi-format viewers
- Content management
- Document libraries
- File browsers
