## Overview

Enzyme is a JavaScript Testing utility for React that makes it easier to assert, manipulate, and traverse your React Components' output.

## Features

- Shallow rendering
- Full DOM rendering
- Static rendered markup
- Component instance methods
- Selector API (CSS, prop)
- Event simulation
- State and prop manipulation
- Children and parent traversal
- Works with Jest, Mocha, and other test runners

## Pricing

Free and open-source.

## Note

Enzyme is currently in maintenance mode and does not fully support React 18. The Airbnb team has not actively developed it since 2020. For React 18+ projects, consider React Testing Library instead.
