## Overview

Mosaic is a lightweight, customizable analytics dashboard built for real-time updates and modern UI experiences. Its integration with Chart.js, Radix UI, and React Day Picker ensures engaging, interactive data visualization.

## Features

- Optimized for speed with Vite 6
- Minimalistic and highly customizable design
- Chart.js for data visualization
- Radix UI primitives for accessible components
- React Day Picker for date selection
- Perfect for SaaS platforms and admin panels
- Tech stack: React 19, Tailwind CSS, Vite 6
