## Overview

React Window is a small library for rendering large lists efficiently. By automatically scrolling and rendering only visible elements, it dramatically improves performance when dealing with large datasets.

## Key Features

- **Virtualization**: Only render visible items
- **Performance**: Handle millions of items smoothly
- **Small Size**: Minimal bundle impact
- **Simple API**: Easy to use interface
- **TypeScript**: Full type support
- **Horizontal & Vertical**: Support for both orientations
- **Variable Sizes**: Items with different heights/widths
- **Fixed Size**: Optimized for uniform item sizes

## Components

- **FixedSizeList**: Lists with uniform item height
- **FixedSizeGrid**: Grids with uniform cell size
- **VariableSizeList**: Lists with variable item heights
- **VariableSizeGrid**: Grids with variable cell sizes
