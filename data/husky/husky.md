## Overview

Husky is a tool that makes Git hooks easy to use. It allows you to run scripts before commits, pushes, and other Git operations, ensuring code quality and preventing bad commits.

## Features

- **Git Hooks**: Pre-commit, pre-push, and more
- **Zero Config**: Simple setup
- **Fast**: Native Git hooks
- **Shareable**: Team configuration
- **Cross-platform**: Works on all OSs
- **Lint-staged Integration**: Perfect with lint-staged
- **CI Skip**: Skip hooks in CI environments
- **TypeScript**: Type definitions included

## Use Cases

- Running linters before commits
- Executing tests before push
- Code formatting enforcement
- Commit message validation
- Preventing bad commits
- Team code quality standards
