## Overview

React Complex Tree is a performant, accessible tree view component for React with advanced features like drag-and-drop, keyboard navigation, and multi-item selection.

## Features

- Zero dependencies (only React required)
- Drag-and-drop support for moving nodes
- Multi-item selection with Ctrl/Shift modifiers
- Inline renaming of nodes
- Full keyboard navigation
- Screen reader support and ARIA compliance
- Programmable expand/collapse state
- Search and filtering
- Custom renderers for complete visual control
- Virtualization for large trees
- Programmatic tree manipulation API
- Shared state across multiple tree instances
- TypeScript support
- Customizable styling
- Performance optimized for thousands of nodes

## Use Cases

- File explorers and file system UIs
- Organization hierarchies
- Category and taxonomy management
- Navigation menus
- Decision trees
- Mind mapping tools
