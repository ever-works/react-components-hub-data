## Overview

React Confetti Explosion is a lightweight confetti component that uses only CSS for animations. Unlike canvas-based solutions, it's designed as an explosion effect rather than continuous raining confetti.

## Features

- Pure CSS animations (no canvas)
- Super lightweight
- Explosion effect
- Customizable particle count
- Custom colors
- Size configuration
- Duration control
- Force control
- Height and width settings
- No external dependencies
- TypeScript support
- Performance optimized
- Easy integration

## Configuration

- particleCount: Number of confetti pieces
- particleSize: Size of particles
- duration: Animation duration
- colors: Custom color array
- force: Explosion strength
- height: Explosion height
- width: Explosion width
- zIndex: Stacking context

## Animation

- CSS keyframe animations
- Hardware accelerated
- Smooth 60fps performance
- Automatic cleanup
