## Overview

Yet Another React Lightbox is a modern, performant, and feature-rich lightbox component for React. It's designed to be easy to use, highly customizable, and extendable through a plugin-based architecture.

## Features

- Responsive image resolution switching
- Keyboard, mouse, touchpad, and touchscreen navigation
- Video slides via plugins
- Zoom functionality via plugins
- Captions and descriptions
- Thumbnails navigation
- Fullscreen mode
- Slideshow autoplay
- Inline gallery mode
- Server-side rendering support
- TypeScript support
- Accessibility features
- Custom slide types
- Animation controls
- Lazy loading

## Plugins

- Captions: Add image captions
- Thumbnails: Thumbnail navigation
- Fullscreen: Fullscreen mode
- Slideshow: Auto-advance slides
- Zoom: Image zoom functionality
- Video: Video slide support
- Inline: Inline gallery mode
- Counter: Slide counter display
- Download: Download button
- Share: Social sharing

## Navigation

- Arrow keys for prev/next
- Escape to close
- Mouse wheel support
- Touch gestures
- Click outside to close

## Use Cases

- Photo galleries
- Portfolio displays
- Product images
- Before/after comparisons
- Image documentation
- Media presentations
