## Overview

Extensive React Boilerplate is a comprehensive starter kit featuring Next.js, authentication flows, Material UI, React Hook Form, internationalization, file uploads (local and Amazon S3), tests, and CI/CD configuration.

## Features

- Next.js framework
- Complete auth (Sign in, Sign up, Reset password, Confirm email, Refresh Token)
- Material UI integration
- React Hook Form
- I18N support
- File uploads (local and Amazon S3 drivers)
- Testing setup
- CI/CD configuration
