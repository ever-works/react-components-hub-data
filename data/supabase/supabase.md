## Overview

Supabase is an open-source Firebase alternative built on PostgreSQL. It provides a comprehensive suite of backend services including a managed PostgreSQL database, authentication, real-time subscriptions, storage, and Edge Functions, making it a popular choice for full-stack React applications.

## Features

- Managed PostgreSQL database with real-time capabilities
- Built-in authentication with email/password, OAuth, and SSO
- Row-level security (RLS) for data protection
- Auto-generated REST and GraphQL APIs
- Real-time subscriptions via WebSockets
- File storage service
- Edge Functions for serverless computing
- Can be self-hosted or used as a paid cloud service

## Pricing

Open-source and self-hostable for free. Cloud offering has a generous free tier and paid plans starting at paid tiers for larger projects.
