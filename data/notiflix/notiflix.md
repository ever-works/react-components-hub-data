## Overview

Notiflix is a comprehensive notification library designed for modern web applications, offering a wide range of notification types from success messages to loading indicators and error alerts. It emphasizes simplicity and customization while maintaining a feature-rich offering.

## Features

- Multiple notification types: success, error, warning, info
- Loading indicators and spinners
- Confirmation dialogs
- Report/notification customization options
- Pure JavaScript library, framework-agnostic with React support
- Multiple positions for displaying notifications
- Customizable appearance and behavior
- Queue management for multiple notifications
- SVG-based icons

## Installation

```bash
npm i notiflix
# or
yarn add notiflix
```
