## Overview

React Masked Input provides real-time input masking for React forms. Supports phone numbers, dates, and custom validation patterns.

## Features

- Pre-built mask types (currency, phone, date)
- Custom regex-based masking
- Keyboard navigation support
- Accessibility attributes
- Integration with Formik and React Hook Form

## Installation

`npm install text-mask-core text-mask-react`
