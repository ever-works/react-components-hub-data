## Overview

React Voodoo is an additive animation engine designed to replicate native Android/iOS-like animations in React. It supports server-side rendering, predictive inertia, multitouch gestures, and smooth slider implementations.

## Features

- Additive animation engine
- Native-like smooth animations
- Server-side rendering compatible sliders
- Predictive inertia for scroll interactions
- Multitouch gesture support
- Complex spring-based physics
