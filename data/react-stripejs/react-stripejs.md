## Overview

React Stripe.js is the official React library for Stripe. It provides React components that wrap Stripe.js and Stripe Elements, making it easy to build secure payment forms in React applications.

## Features

- **Pre-built Elements**: Card, Payment, IBAN elements
- **Payment Intents**: Modern payment flow
- **Subscriptions**: Recurring payment support
- **TypeScript**: Full type definitions
- **Customizable**: Style elements to match your brand
- **Secure**: PCI compliant by default
- **All Payment Methods**: Cards, wallets, bank transfers
- **Hooks API**: useStripe and useElements hooks

## Use Cases

- E-commerce checkout
- Subscription payments
- Donation forms
- Marketplace payments
- SaaS billing
- One-time payments

## Pricing

Free to use. Stripe charges transaction fees.
