## Overview

Azia Admin React is a React-based analytics dashboard built with Tailwind CSS, delivering real-time data visualization. It features modern UI and structured components with a lightweight build and flexible architecture.

## Features

- Modern UI and structured components
- Lightweight build and flexible architecture
- Real-time data visualization
- Tailwind CSS styling
- Built with React and Tailwind CSS
