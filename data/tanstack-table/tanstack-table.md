## Overview

Headless UI for building powerful tables & datagrids for TS/JS - React-Table, Vue-Table, Solid-Table, Svelte-Table and more.

## Key Features

- Lightweight (10-15kb)
- Tree-shakable
- Sorting and filtering
- Multi-sort support
- Pagination
- Row selection
- Row expansion
- Column visibility
- Column resizing
- Column ordering
- Row grouping
- Cell formatting
- SSR support

## Core Concepts

- Columns: Define table structure
- Data: Source data
- Column Helpers: Type-safe column definition
- useReactTable: Main hook
