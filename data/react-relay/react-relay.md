## Overview

Relay is a JavaScript framework developed by Meta for building data-driven React applications. It uses GraphQL to declaratively describe the data requirements of components and provides optimizations like fragment colocation and automatic batching.

## Features

- Fragment colocation
- Automatic query batching
- Data consistency
- Predictable mutations
- Built-in caching
- Streaming SSR
- Pagination helpers
- TypeScript support
- Optimistic updates
