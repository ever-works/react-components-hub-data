## Overview

@uiw/react-md-editor is a lightweight markdown editor component for React that provides syntax highlighting and live preview functionality. It aims to offer a simple, straightforward markdown editing experience.

## Features

- Live preview mode
- Syntax highlighting for markdown
- Split-view editing (edit and preview side-by-side)
- Full-screen mode
- Toolbar with common formatting options
- Keyboard shortcuts
- Dark mode support
- Customizable toolbar
- Custom commands support
- Lightweight bundle (~4.6 kB gzipped)
- TypeScript support
- Copy code button
- Line numbers
- Drag and drop image upload
- Custom preview rendering
- Accessible design

## Editor Modes

- **Edit**: Pure editing mode
- **Live**: Split view with live preview
- **Preview**: Preview-only mode

## Use Cases

- Comment sections
- Note-taking apps
- Documentation tools
- Blog editors
- README editors
- Any markdown input needs
