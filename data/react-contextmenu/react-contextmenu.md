## Overview

React Context Menu provides a flexible and accessible context menu component for React applications. It handles right-click events to display custom menus with full keyboard navigation and ARIA support.

## Features

- Right-click menu support
- Keyboard navigation
- Accessibility (ARIA attributes)
- Custom menu items
- Nested submenus
- Menu dividers
- Disabled items
- Custom styling and themes
- Event handlers for menu actions
- Dynamic menu generation
- Touch device support (long press)
- Portal rendering
- Custom positioning
- Multiple context menus per page

## Components

- ContextMenu: Menu container
- MenuItem: Individual menu item
- SubMenu: Nested submenu
- ContextMenuTrigger: Trigger element

## Events

- onClick: Item click handler
- onShow: Menu shown callback
- onHide: Menu hidden callback
- onMouseEnter: Mouse enter item
- onMouseLeave: Mouse leave item

## Use Cases

- File manager applications
- Canvas/drawing tools
- Data grid actions
- Text editor operations
- Node graph interactions
- Dashboard widgets
