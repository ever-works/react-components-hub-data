## Overview

react-range is a small, accessible range input component where you bring your own styles and markup. With a footprint of less than 4KB gzipped and zero dependencies, it provides full flexibility for customization.

## Features

- Accessible (ARIA patterns)
- Bring your own styles and markup
- Single and dual thumb support
- Vertical and horizontal orientation
- Custom step values
- Min/max constraints
- Disabled state
- RTL support
- Keyboard navigation
- Touch device support
- No dependencies
- TypeScript support
- SSR compatible
- Controlled component

## Customization

- Custom track rendering
- Custom thumb rendering
- Custom labels
- CSS-in-JS or styled-components
- Tailwind CSS compatible
- Complete style control

## Accessibility

- ARIA attributes
- Keyboard navigation (arrows, home, end)
- Screen reader support
- Focus management

## API

- values: Current values array
- onChange: Value change handler
- min, max: Range bounds
- step: Increment step
- direction: Horizontal/vertical
- disabled: Disable interaction

## Use Cases

- Price range filters
- Volume controls
- Brightness adjusters
- Time range selection
- Custom form inputs
- Dashboard controls
