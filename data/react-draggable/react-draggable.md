## Overview

React Draggable provides a powerful drag-and-drop API for React applications. It supports custom drag handles, movement constraints, and collision detection between elements.

## Features

- Drag-and-drop with customizable backends (HTML5, Touch)
- Multiple drag sources and drop targets
- Drag preview and feedback
- Scroll container support
- Accessible keyboard drag operations

## Installation

`npm install react-dnd`
