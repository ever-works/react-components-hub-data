## Overview

React Select is a flexible and beautiful Select Input control for React with intuitive multiselect, autocomplete and clear functionality.

## Key Features

- Single/multi-select
- Customizable options
- Async options loading
- Creatable select
- Clear functionality
- Search/filter options
- Keyboard support
- Accessible
- TypeScript support
- Themeable
- Mobile-friendly

## Variants

- Select: Basic select
- AsyncSelect: Options from API
- CreatableSelect: Create new options
- AsyncCreatableSelect: Both async and creatable
