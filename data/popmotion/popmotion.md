## Overview

Popmotion is a functional, reactive animation library that is highly customizable. It operates as a lower-level tool but offers incredible flexibility for building custom animation sequences.

## Features

- Functional and reactive programming model
- Highly customizable animation primitives
- Support for springs, physics, and keyframe animations
- Works with any rendering engine
- Lower-level API for fine-grained control
- Lightweight and composable
