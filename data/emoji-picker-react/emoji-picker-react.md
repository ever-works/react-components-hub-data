## Overview

emoji-picker-react is the most popular emoji picker component for React applications. It works immediately with no required props and offers extensive customization options with lazy loading for optimal performance.

## Features

- Zero required props (works out of the box)
- Lazy loading of emoji images
- Multiple emoji sets (Apple, Google, Facebook, Twitter, Native)
- Light, dark, and auto themes
- Search functionality
- Category filtering
- Skin tone selection
- Recent emojis tracking
- Custom emoji support
- Emoji variations
- Click and hover events
- Customizable via CSS variables
- TypeScript support
- Accessibility features
- Responsive design

## Emoji Sets

- apple: Apple emoji style
- google: Google emoji style
- facebook: Facebook emoji style
- twitter: Twitter emoji style
- native: System native emojis

## Themes

- light: Light theme
- dark: Dark theme
- auto: System preference

## Customization

- Custom categories
- Search placeholder text
- Category labels
- Preview configuration
- Height and width
- CSS variable overrides

## Use Cases

- Chat applications

- Comment systems
- Messaging apps
- Feedback forms
- Reaction systems
