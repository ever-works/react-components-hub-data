## Overview

ReactPlayer is a simple React component for playing a variety of media sources. It supports multiple video hosting platforms, local files, and streaming protocols, making it ideal for building custom video players.

## Supported Sources

- YouTube
- Vimeo
- Wistia
- Twitch
- SoundCloud
- Streamable
- Facebook
- DailyMotion
- HTML5 video files
- HLS streams
- And more

## Features

- Single component for multiple media types
- Play, pause, seek controls
- Volume and speed control
- Fullscreen support
- Progress bar with preview
- Light player mode for performance
- Responsive design
- TypeScript support
- Extensive event callbacks
- No external dependencies for HTML5

## Installation

```bash
npm install react-player
```

## Basic Usage

```jsx
import ReactPlayer from 'react-player'

<ReactPlayer url='https://www.youtube.com/watch?v=dQw4w9WgXcQ' />
```
