## Overview

React Intl Universal provides internationalization for React applications. Unlike react-intl, it works not only for React.Component but also for Vanilla JS, making it versatile for various JavaScript applications.

## Features

- Works with React.Component
- Works with Vanilla JS
- Server-side rendering support
- Message formatting
- Pluralization support
- Developed by Alibaba
