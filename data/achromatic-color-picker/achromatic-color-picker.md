## Overview

Achromatic Color Picker is a dependency-free, lightweight color picker component designed for web applications in the React/Next.js + Tailwind + shadcn/ui ecosystem. It provides essential color selection features without requiring any external libraries.

## Features

- Hue slider support for color selection
- Alpha channel (transparency) control
- Eyedropper tool integration using the browser EyeDropper API
- Color preview display
- Direct color input fields (HEX, RGB, HSL)
- No external dependencies
- Built for React/Next.js with Tailwind CSS and shadcn/ui compatibility

## Props

- `value` — The current color value
- `onChange` — Callback fired when color changes
- `showAlpha` — Enable alpha/transparency slider
- `showHue` — Enable hue slider
- `showColorPreview` — Show current color preview
- `showInputs` — Show color input fields (HEX, RGB, HSL)
- `showEyeDropper` — Show eyedropper button

## Use Cases

- Theme customization interfaces in React/Next.js applications
- Dashboard and admin panel color configuration
- Design system configuration tools
- Real-time brand color customization
