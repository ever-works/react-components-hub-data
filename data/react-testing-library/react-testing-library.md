## Overview

React Testing Library is a testing utility that provides simple and complete React component testing APIs that encourage good testing practices.

## Key Features

- **User-Centric**: Test like users interact
- **DOM Queries**: Query by role, text, etc.
- **Accessibility Focus**: Promote accessibility
- **Simple API**: Minimal learning curve
- **Async Utilities**: Handle async operations
- **Hooks Support**: Test custom hooks
- **TypeScript**: Full type support
- **Best Practices**: Guide correct testing

## Query Methods

- getByRole: Query by accessibility role
- getByText: Query by displayed text
- getByPlaceholderText: Query by placeholder
- getByDisplayValue: Query by form value
- getByAltText: Query by alt text
- getByTestId: Query by test ID
- queryBy*: Return null if not found
- findBy*: Async query methods

## Testing Utilities

- render: Render component
- fireEvent: Trigger events
- userEvent: Simulate user interactions
- waitFor: Wait for async changes
- screen: Access rendered elements
- within: Scoped queries
