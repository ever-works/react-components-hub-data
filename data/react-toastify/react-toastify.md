## Overview

React-Toastify is an easy-to-use library for adding toast notifications to your React applications. It handles positioning, animations, and customization seamlessly.

## Key Features

- Multiple notification types (success, error, warning, info)
- Customizable positioning
- Auto-dismiss timers
- Progress bar
- Pause on hover
- Stack control
- Theme support (light/dark)
- Accessible notifications
- Mobile-friendly
- TypeScript support

## Main Components

- ToastContainer: Main container for toasts
- toast: Function to display notifications
- useToast: Hook-based API
