## Overview

Yet another Kanban/Trello board library for React with flexible architecture and straightforward API. It supports both controlled and uncontrolled component modes.

## Features

- Uncontrolled and controlled board modes
- Drag-and-drop card movement
- Lane management
- TypeScript support
- Customizable card and lane rendering
- Event callbacks for board changes
- Mobile-friendly interactions
- Responsive grid layout

## Architecture

- Board contains Lanes
- Lanes contain Cards
- Flexible data structure
- Easy integration with state management libraries
