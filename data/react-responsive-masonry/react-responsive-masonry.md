## Overview

react-responsive-masonry is a lightweight masonry layout component for React that creates responsive layouts where the number of columns changes based on the viewport width. Built with CSS flexbox for performance and reliability.

## Features

- Responsive column count based on breakpoints
- CSS flexbox-based (no absolute positioning)
- Configurable column gap/gutter
- Customizable breakpoints
- Lightweight and performant
- No JavaScript calculations for positioning
- SSR compatible
- TypeScript support
- Easy to implement
- Works with any content type
- Smooth responsive transitions

## Key Advantages

- Uses flexbox instead of absolute positioning
- Better performance than JavaScript-based solutions
- Native browser reflow handling
- Simpler implementation
- More maintainable code

## Use Cases

- Photo galleries
- Pinterest-style layouts
- Blog post grids
- Product showcases
- Portfolio displays
- Card-based interfaces
- Image walls
