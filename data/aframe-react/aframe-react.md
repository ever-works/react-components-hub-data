## Overview

A-Frame React bridges A-Frame's WebVR framework with React's component model, enabling developers to build immersive virtual reality experiences using familiar React patterns and JSX.

## Features

- Virtual reality and WebXR experiences
- Declarative 3D scene composition via React components
- A-Frame entity-component system integration
- WebVR and WebXR support
- React lifecycle methods mapped to A-Frame events
- Support for VR headsets and mobile AR
- TypeScript definitions available
