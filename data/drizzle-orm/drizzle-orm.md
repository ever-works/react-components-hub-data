## Overview

Drizzle ORM is a rising star in the TypeScript ORM space. It provides a lightweight, fast alternative to Prisma with a headless design that gives developers more control over their queries while maintaining full type safety.

## Features

- TypeScript-first with full type inference
- Supports PostgreSQL, MySQL, and SQLite
- No heavy runtime overhead
- SQL-like query syntax
- Auto-migrations with Drizzle Kit
- Works with serverless environments
- Compatible with Neon, Turso, PlanetScale, and other serverless databases
