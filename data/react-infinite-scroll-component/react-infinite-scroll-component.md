## Overview

React Infinite Scroll Component is a lightweight library (just 4.15 kB) that provides a faster and streamlined implementation for infinite scrolling in React applications. It automatically loads more content as users scroll.

## Features

- Infinite scroll functionality
- Pull-down to refresh
- Vertical and horizontal scrolling
- Custom scroll height
- Scroll overflow configuration
- Loading indicator customization
- End message customization
- Reverse scrolling support
- Initial scroll position
- Scroll event handling
- TypeScript support
- No external dependencies
- SSR compatible
- Mobile optimized

## Props

- dataLength: Number of items loaded
- next: Function to load more items
- hasMore: Boolean for more data
- loader: Loading component
- endMessage: No more data component
- refreshFunction: Pull to refresh handler
- pullDownToRefresh: Enable pull to refresh
- scrollThreshold: Trigger threshold

## Use Cases

- Social media feeds
- Product listings
- News articles
- Image galleries
- Search results
- Comment sections
- Chat message history
