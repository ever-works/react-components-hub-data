## Overview

A React wrapper for the Trumbowyg WYSIWYG editor, providing a lightweight rich text editing experience.

## Features

- Wraps the Trumbowyg lightweight WYSIWYG editor
- Produces semantic and clean HTML
- Customizable toolbar with plugins
- Keyboard shortcuts support
- RTL mode support
- SVG icons for toolbar buttons
- Easy configuration and integration
