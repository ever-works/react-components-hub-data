## Overview

react-use-measure is a React hook that uses ResizeObserver to measure element dimensions and position. It provides accurate measurements with optimal performance for creating responsive components.

## Features

- **ResizeObserver**: Modern, performant resize detection
- **Debouncing**: Built-in debounce support
- **Polyfill Support**: Works in older browsers
- **TypeScript**: Full type definitions
- **Scroll Detection**: Detects position changes
- **Lightweight**: Minimal bundle size

## Use Cases

- Responsive components that adapt to size changes
- Animation triggers based on element size
- Chart and visualization sizing
- Dynamic layout calculations
- Virtual scrolling and windowing
