## Overview

Pragmatic Drag and Drop is Atlassian's next-generation drag-and-drop library, built to replace react-beautiful-dnd. It's framework-agnostic, performant, and built on top of the native HTML5 drag-and-drop API.

## Features

- Framework agnostic - works with React, Vue, Svelte, Angular, or vanilla JS
- Built on native HTML5 drag-and-drop API
- Extremely lightweight with tree-shakeable architecture
- Bundle only the features you need
- Supports mouse, touch, and keyboard interactions
- Accessibility built-in with screen reader support
- Drag handles and conditional dragging
- Auto-scrolling during drag
- Drop indicators and visual feedback
- Nested drop targets
- Virtual scrolling support
- Multiple simultaneous drags
- File drag-and-drop from OS

## Performance Benefits

- Minimal JavaScript execution during drag
- Leverages browser's native drag behavior
- No React-specific coupling
- Optimized for large lists

## Migration from react-beautiful-dnd

Atlassian provides migration guides and codemods to help teams transition from react-beautiful-dnd to Pragmatic Drag and Drop.
