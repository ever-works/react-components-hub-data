## Overview

Hugeicons is a comprehensive icon library offering 46,000+ icons across 10 styles. It provides 4,600+ free open-source icons in its stroke-rounded style, with the full collection available under a premium license.

## Features

- **46,000+ icons** across 10 different visual styles
- **4,600+ free icons** in stroke-rounded style (open source)
- Production-ready React components
- Built for visual consistency and polish
- Covers a wide range of use cases without needing to mix multiple libraries

## Use Cases

- Modern SaaS products
- Professional dashboards and admin panels
- AI-driven interfaces
- Design-driven web and mobile applications

## Trusted By

Used by teams including Y Combinator (YC), Fiverr, the Saudi Government, and developers worldwide.

## Pricing

- **Free tier**: 4,600+ icons in stroke-rounded style (open source)
- **Premium**: Full library of 46,000+ icons across 10 styles (paid license)
