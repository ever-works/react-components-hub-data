## Overview

Ember is a healthcare-focused React admin dashboard designed specifically for medical workflows, where every page maps to a real clinical or administrative function.

## Features

- 50+ pages covering healthcare-specific use cases
- Patient Management
- Appointments scheduling
- Vitals Monitor
- Prescriptions management
- Lab Results viewing
- Health Records management
- Bed Management
- Telemedicine integration
- Billing module
- TanStack Table integration
- Leaflet maps integration
- TipTap editor integration
- Recharts 3 for charting
- Authentication with 2FA
- Storybook documentation
- Vitest + Playwright testing
- Dark/light theme support
