## Overview

Radix UI provides an unstyled, accessible component system that serves as a foundation for building your own design system. Each component is built with complete keyboard navigation, focus management, and ARIA attributes.

## Key Features

- Unstyled, fully customizable components
- Complete keyboard and screen reader support
- WAI-ARIA compliance
- Composable primitives
- Zero external dependencies
- TypeScript support
- Excellent documentation

## Available Primitives

- Accessibility: Accessible Menu, Accessible Dropdown
- Dialog & Modals: Dialog, AlertDialog, Popover
- Navigation: NavigationMenu, Tabs
- Overlays: Tooltip, DropdownMenu, ContextMenu
- Forms: Label, RadioGroup, Checkbox, Switch
- Data: Collapsible, Accordion
