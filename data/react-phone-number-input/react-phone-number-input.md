## Overview

React Phone Number Input provides a comprehensive solution for international phone number input with automatic formatting, country detection, and validation based on Google's libphonenumber library.

## Features

- International phone number support
- Country selector dropdown
- Automatic formatting
- Phone number validation
- E.164 format output
- Country flag icons
- Country code prefix
- Auto-detect country from number
- Search countries
- Preferred countries
- Excluded countries
- Default country
- Placeholder formatting
- Custom styling
- RTL support
- TypeScript support
