## Overview

react-audio-visualize is an audio visualization library for React that provides components for visualizing both live audio (during recording) and pre-recorded audio blobs with waveform displays.

## Features

- Live audio visualization during recording
- Audio blob visualization for playback
- Waveform rendering
- Customizable visualizer appearance
- Bar-based and waveform displays
- Color customization
- Size configuration
- React component-based
- TypeScript support
- Web Audio API based
- Performance optimized
- Canvas rendering

## Visualization Types

- Live audio bars (recording)
- Audio waveform (playback)
- Real-time frequency analysis
- Amplitude visualization

## Use Cases

- Voice recording apps
- Audio players
- Music applications
- Podcast editors
- Voice note features
- Audio messaging
- Sound recording tools
