## Overview

A comprehensive date picker component for React applications that provides a clean, intuitive interface for selecting dates. Part of the react-calendar suite by the same author.

## Features

- Supports single date selection
- Configurable date formats and ranges
- Keyboard accessible navigation
- Customizable styling to match any design system
- Supports controlled and uncontrolled modes
- Internationalization support
- Calendar navigation with month and year views
