## Overview

Kysely is a type-safe SQL query builder designed specifically for TypeScript. Unlike traditional ORMs, it provides a query builder that generates full SQL with complete type safety at compile time, giving developers control over SQL while maintaining TypeScript's type checking.

## Features

- Full type safety at compile time
- Works with PostgreSQL, MySQL, and SQLite
- Query builder API with full SQL coverage
- No hidden queries — you write the SQL
- Migration system
- Plugin system for extending functionality
- Excellent autocompletion in editors
