## Overview

React Scroll Parallax provides React hooks and components to create parallax scroll effects. It's optimized to reduce jank during scroll and works seamlessly with SSR and SSG rendered React apps.

## Features

- Vertical and horizontal parallax effects
- Speed-based parallax
- Translate, scale, rotate effects
- Opacity transitions
- Custom easing functions
- Progress-based animations
- SSR/SSG compatible
- Intersection observer for performance
- Disabled state for accessibility
- TypeScript support
- Works with any element type
- Banner component for hero sections
- Multiple layers support
- Custom scroll containers

## Components

- ParallaxProvider: Context provider
- Parallax: Individual parallax element
- ParallaxBanner: Multi-layer parallax sections
- useParallax: Hook for custom implementations

## Effects

- translateX/translateY: Movement
- scale: Zoom effects
- rotate: Rotation
- opacity: Fade effects
- Custom CSS transformations

## Configuration

- speed: Parallax speed (-100 to 100)
- easing: Timing function
- disabled: Conditional rendering
- rootMargin: Trigger offset

- Product showcases
