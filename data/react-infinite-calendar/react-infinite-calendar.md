## Overview

React Infinite Calendar is a modern, lightweight, and customizable datepicker/Calendar component for React with built-in infinite month/year scrolling. It provides a sleek user experience particularly well-suited for mobile devices where infinite scroll greatly aids navigation.

## Features

- Infinite scrolling across months and years for seamless date navigation
- Modern, sleek design out of the box
- Keyboard navigation support
- Multiple selection modes: single date, date range, and multiple date selection
- Customizable date cells and month headers
- Localization support for internationalization
- Optimized for mobile device interaction
- Themable via CSS and component props
- TypeScript support
