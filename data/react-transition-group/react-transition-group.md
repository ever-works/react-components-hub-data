## Overview

React Transition Group is a component library that provides simple utilities for managing animations and transitions when React components mount or unmount.

## Key Features

- **Component Lifecycle**: Animate on mount/unmount
- **State Transitions**: Animate between states
- **CSS Transitions**: CSS-based animations
- **Hooks Support**: useTransition hook
- **Callback Hooks**: onEnter, onExit callbacks
- **Multiple Components**: Transition multiple items
- **Teleporting**: Portal support
- **TypeScript**: Type support

## Components

- **Transition**: Base transition component
- **CSSTransition**: CSS-based transitions
- **TransitionGroup**: Multiple transitions
- **SwitchTransition**: Switch between transitions

## Features

- Enter animations
- Exit animations
- Transition timing
- CSS classes management
- Callback hooks
- Appear animations
- Timeout control
- In prop control
- Multiple children
- Nested transitions

## Configuration

- in: Transition state
- timeout: Animation duration
- classNames: CSS class prefix
- onEnter: Enter callback
- onExit: Exit callback
- onExited: Exited callback
- appear: Show on mount
- unmountOnExit: Remove from DOM

## CSS Classes

- enter: Entering state
- enterActive: Active entering
- exit: Exiting state
- exitActive: Active exiting
- appear: Initial render
- appearActive: Active appearing
