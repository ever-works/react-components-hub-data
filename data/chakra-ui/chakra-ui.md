## Overview

Chakra UI is a simple, modular and accessible component library that gives you the building blocks you need to build your React applications.

## Key Features

- 40+ accessible components
- Style props system
- Easy dark mode
- Responsive design with ease
- Keyboard accessibility built-in
- Themeable components
- TypeScript support
- Emotion CSS-in-JS

## Component Categories

- Form Inputs: Button, Input, Checkbox, Radio, Select, Textarea
- Layout: Box, Flex, Grid, Stack, Container
- Navigation: Breadcrumb, Menu, Tabs
- Overlay: Alert, Dialog, Drawer, Popover, Tooltip
- Feedback: Progress, Spinner, Toast
- Data: Avatar, Badge, List, Table
