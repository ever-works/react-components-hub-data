## Overview

useCountdown is a lightweight React hook that provides countdown timer functionality. It offers a simple API for creating and managing countdown timers with pause, resume, and reset controls.

## Features

- **Simple API**: Easy to use countdown hook
- **Pause/Resume**: Control timer execution
- **Reset**: Reset countdown to initial value
- **Callbacks**: onComplete callback support
- **Interval Control**: Customizable update intervals
- **TypeScript**: Full type definitions
- **Lightweight**: Minimal bundle size

## Use Cases

- Countdown timers
- Flash sales and limited offers
- Event countdowns
- Quiz and exam timers
- Session timeouts
- Auction countdowns
