## Overview

React Dropzone is a React component for creating drag-and-drop file upload zones with full accessibility support and extensive customization options.

## Key Features

- **Drag & Drop**: Native file drag-and-drop
- **Click to Upload**: Alternative upload method
- **Multiple Files**: Accept multiple files
- **File Filtering**: Accept specific file types
- **File Size**: Size validation
- **Preview**: Display file previews
- **Accessibility**: WCAG compliant
- **Disabled State**: Disable dropzone
- **Custom Styling**: Full customization
- **TypeScript**: Type support

## Configuration

- accept: Accepted file types
- maxSize: Maximum file size
- minSize: Minimum file size
- maxFiles: Maximum files
- multiple: Allow multiple
- disabled: Disable dropzone
- preventDropOnDocument: Prevent default
- noClick: Disable click
- noDrag: Disable drag
- noKeyboard: Disable keyboard

## Methods

- getRootProps: Dropzone props
- getInputProps: Input props
- open: Open file dialog
