## Overview

FusionCharts is a comprehensive JavaScript charting suite that provides over 100+ charts and 2000+ choropleth maps. It offers wide-ranging chart types and deep customization options for building interactive data visualizations across web and mobile platforms.

## Features

- 100+ chart types and 2000+ choropleth maps
- Interactive features: tooltips, zooming, panning, and drill-down
- Export charts as PNG, JPEG, SVG, and PDF
- Responsive design across devices
- Live charting and real-time updates
- Custom themes and branding options
- TypeScript definitions included
- Works with React, Angular, Vue, Svelte, and vanilla JS
