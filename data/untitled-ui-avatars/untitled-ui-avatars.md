## Overview

Avatar components from Untitled UI React for displaying user images and initials.

## Features

- 3 components with 11 variants
- Built with Tailwind CSS v4.2 and React Aria v1.16
- Support for profile images and initials fallback
- Status indicators
- Responsive sizing
- Copy, paste, and build workflow

## Pricing

Open-source. Part of the Untitled UI React collection.
