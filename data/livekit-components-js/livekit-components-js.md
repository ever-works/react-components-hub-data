## Overview

LiveKit Components provides React UI components for building real-time communication features like voice agents, video conferencing, and chat. The components integrate with LiveKit's cloud or self-hosted servers for low-latency communication.

## Key Features

- Pre-built components for agent sessions
- Audio visualization UI
- Voice control bars
- Chat transcript components
- Compatible with Shadcn UI

## Installation

```bash
$ npm install @livekit/components-js
``
