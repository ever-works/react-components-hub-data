## Overview

Sonner is a modern notification/toast library built specifically for React 18+ with TypeScript support and a beautiful default UI.

## Key Features

- React 18+ optimized
- TypeScript first
- Minimal bundle size
- Beautiful default UI
- Customizable styling
- Multiple notification types
- Position control
- Auto-dismiss
- Pause on hover
- Action buttons support
- Accessible

## Notification Types

- Success
- Error
- Warning
- Info
- Custom
