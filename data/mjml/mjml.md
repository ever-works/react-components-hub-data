## Overview

MJML is a markup language that simplifies the process of creating responsive HTML emails. Instead of dealing with complex email HTML quirks and table-based layouts, developers use MJML's component-based syntax which compiles down to responsive, cross-client compatible email HTML.

## Features

- Simple, semantic markup for email components
- Responsive design out of the box
- Compiles to cross-client compatible HTML
- Built-in components (button, column, section, text, image, etc.)
- Live preview and validation
- Programmatic API for integration
- Can be integrated with React via third-party bindings
