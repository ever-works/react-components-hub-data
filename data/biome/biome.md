## Overview

Biome (formerly Rome) is an emerging all-in-one toolchain for JavaScript and TypeScript that aims to replace the ESLint + Prettier combination. Written in Rust, it provides blazing-fast linting and formatting with a unified configuration, eliminating the complexity of managing multiple tools and their interplay.

## Features

- Rust-based for exceptional performance
- Combines linting and formatting in one tool
- Supports JavaScript, TypeScript, JSX, and TSX
- Minimal configuration required
- Drop-in replacement for ESLint and Prettier
- Consistent output across all file types
- Built for monorepos with workspace support
