## Overview

Saasable is a React starter kit for building enterprise applications and web apps. It is built with Material UI v7 following the Material 3 guideline and provides an excellent foundation for building React applications and analytics dashboards.

## Tech Stack

- Material UI v7
- Next.js 14 (App Router)
- TypeScript

## Features

- Powered by Material UI v7, built with Next.js 14 and TypeScript
- Pre-built SaaS features including Supabase authentication
- Includes a matching Figma UI kit design file to speed up the design-to-development process
- Clean and minimalist design that is neutral enough for most web applications and brands
- SEO-friendly features and best practices
- Available in both free and paid versions

## Pros

- Excellent modern foundation for a new project
- Includes a matching Figma UI kit
- Minimalist, neutral design suitable for most brands
- SEO-friendly out of the box

## Cons

- The free version can be quite limited, with most pre-made components and React templates reserved for the paid version
- More of a starter kit than a dedicated, feature-packed React dashboard template
