## Overview

React Query DevTools is a browser-based developer tools companion for React Query, providing real-time visibility into query state, cache, and detailed debugging capabilities.

## Key Features

- **Query Inspector**: View all active queries
- **Cache Viewer**: See cached data
- **Timeline**: Track query lifecycle
- **Network Monitor**: Monitor API calls
- **Error Display**: Show query errors
- **Refetch Trigger**: Manual refetch
- **Toggle UI**: Hide/show tools
- **Floating Panel**: Always accessible
- **Keyboard Shortcut**: Quick toggle
