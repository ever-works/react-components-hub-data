## Overview

React Experiments provides components and utilities for running A/B tests and UI experiments in React applications. Developed by HubSpot, it supports experiment configuration, user bucketing, and result tracking.

## Features

- Declarative experiment wrapper components
- Automatic user bucketing and consistent variant assignment
- Integration with analytics platforms for metric tracking
- Support for nested experiments
- Server-side rendering compatibility
- Feature flag toggles for gradual rollouts
