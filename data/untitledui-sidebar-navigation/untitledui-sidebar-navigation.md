## Overview

A collection of sidebar navigation components for React applications, built using React Aria for accessibility and styled with Tailwind CSS. Base components are free and open-source, while featured card add-ons are available with a PRO license.

## Installation

```bash
npx untitledui@latest add sidebar-navigation-base
```
