## Overview

Material React Table combines the power of TanStack Table with the elegance of Material UI to provide a comprehensive, highly customizable data table for React applications.

## Features

- Built on TanStack Table (headless, framework-agnostic)
- Material UI component styling and theming
- Inline cell and row editing
- Advanced filtering with custom filter variants
- Column grouping and pivoting
- Column pinning and resizing
- Row virtualization for large datasets
- Full-screen mode
- Density toggle (comfortable, compact)
- Column visibility management
- CSV and clipboard export
- Custom toolbar actions
- Global search with debouncing
- Multi-sort support
- Pagination (client and server-side)
- Row selection with checkboxes
- Expandable/collapsible sub-rows
- Drag and drop column reordering
- Built-in accessibility features
- Server-side data support
- Responsive column hiding
