## Overview

MDXEditor is a rich-text editor React component that allows users to author markdown documents naturally. Unlike traditional markdown editors with separate edit and preview modes, MDXEditor provides true WYSIWYG inline editing powered by Lexical.

## Features

- True WYSIWYG editing - see formatted content as you type
- Full MDX support with JSX components
- Built on Lexical framework for extensibility
- Markdown shortcuts (e.g., type ** for bold)
- Toolbar with formatting controls
- Code blocks with syntax highlighting
- Image uploads and embedding
- Tables with visual editing
- Headings, lists, quotes, and more
- Link editing
- Diff view for comparing changes
- Frontmatter editing
- Custom plugin system
- TypeScript support
- Theming capabilities
- Accessibility features
- Copy/paste from various sources

## Use Cases

- Documentation editors
- Blog post authoring
- Content management systems
- Note-taking applications
- Technical writing tools
- Knowledge bases
