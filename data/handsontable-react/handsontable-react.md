## Overview

Handsontable is a JavaScript data grid that looks and feels like a spreadsheet. It offers Excel-like functionality including formulas, data validation, and extensive cell formatting options.

## Spreadsheet Features

- Excel-like interface
- Formula support (400+ functions)
- Cell formatting
- Data validation
- Conditional formatting
- Merge cells
- Freeze rows and columns
- Comments
- Context menu
- Keyboard shortcuts
- Copy/paste from Excel
- Undo/redo
