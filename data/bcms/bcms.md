## Overview

BCMS is a headless, API-first CMS designed for Jamstack frameworks. It provides content management through GraphQL and REST APIs for Gatsby, Next.js, and Nuxt applications.

## Features

- Headless architecture with REST and GraphQL APIs
- Visual content editor for non-technical users
- Custom content type definitions
- Media library with asset management
- User roles and permissions
- Multi-language content support
- Self-hosted or Docker deployment
- Framework integrations for Gatsby, Next.js, Nuxt
