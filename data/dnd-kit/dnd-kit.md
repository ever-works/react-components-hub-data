## Overview

dnd kit is a modern drag and drop toolkit for React that replaces complex drag-and-drop implementations with a clean, accessible API. It's built from the ground up for React and provides modular, performant drag and drop functionality.

## Features

- Modular architecture with sortable list, resizable, and touch support presets
- Sensor system for detecting interactions (pointer, mouse, touch, keyboard)
- Minimal learning curve with hooks-based API
- First-class TypeScript support
- Built-in keyboard navigation and screen reader support
- Smooth animations and visual feedback
- Works with both web and React Native (experimental)
- Zero dependencies

## Use Cases

- Sortable lists and boards
- Kanban-style applications
- Reorderable grids
- Multi-column drag and drop
- Tree-based structures

## Installation

```bash
npm install @dnd-kit/core @dnd-kit/utilities @dnd-kit/sortable
```
