## Overview

React Virtualized renders large datasets efficiently by only rendering visible cells and elements, enabling smooth scrolling with hundreds of thousands of rows.

## Features
- Virtualized cell list rendering with windowing
- Grid, List, Table, and Collection components
- Scroll position persistence and synchronization
- CellMeasurer for dynamic content measurement
- InfiniteLoader for pagination and lazy loading
- AutoSizer for responsive container adaptation
- High-performance 60fps scrolling
