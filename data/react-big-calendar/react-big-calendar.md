## Overview

React Big Calendar is a mature calendar component that provides a full-featured calendar experience with multiple view modes, event management, and drag-and-drop support.

## Key Features

- **Multiple Views**: Month, week, day, agenda views
- **Drag & Drop**: Drag events to reschedule
- **Resizing**: Resize events to change duration
- **Custom Events**: Custom event rendering
- **Time Zone Support**: Handle multiple time zones
- **Keyboard Navigation**: Full keyboard support
- **Accessible**: WCAG compliant
- **TypeScript**: Type definitions included

## View Modes

- **Month View**: Full month calendar
- **Week View**: 7-day week view
- **Work Week View**: 5-day week (Mon-Fri)
- **Day View**: Single day view
- **Agenda View**: List of upcoming events
