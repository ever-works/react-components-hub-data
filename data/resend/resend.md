## Overview

Resend is a modern email API designed for developers who need to send transactional and marketing emails at scale. It provides a React-compatible SDK and works seamlessly with React Email templates, making it a natural choice for full-stack React applications.

## Features

- Simple email API with React SDK
- Excellent deliverability rates
- Built-in support for React Email templates
- Email tracking and analytics
- Domain verification and DKIM signing
- Webhooks for email events
- Regional email sending
- DMARC, SPF compliance

## Pricing

Free tier includes 3,000 emails/month. Paid plans start at $20/month for higher volumes.
