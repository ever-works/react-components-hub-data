## Overview

A community-driven open-source icon toolkit that serves as a fork of the popular Feather Icons project, offering a larger and actively maintained collection of icons.

## Features

- Large collection of beautiful, consistent icons
- Fork of the popular Feather Icons project
- SVG-based for crisp rendering at any size
- Available for multiple frameworks (React, Vue, Angular, Svelte, etc.)
- Regularly updated with new icons
- Consistent design language across all icons
- Tree-shakable imports
- Well-documented API
