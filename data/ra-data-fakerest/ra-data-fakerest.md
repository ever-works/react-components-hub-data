## Overview

ra-data-fakerest is a data provider for React Admin that simulates a REST API entirely in the browser's memory. It accepts a plain JavaScript object as an initial dataset and provides all standard React Admin data provider methods (getList, getOne, getMany, create, update, delete, etc.).

## Features

- Simulates a full REST API in-memory without network requests
- Accepts initial dataset as a plain JavaScript object
- Supports standard CRUD operations (Create, Read, Update, Delete)
- Compatible with all React Admin components (List, Show, Edit, Create)
- Auto-generates IDs when the `generateId` option is enabled
- Supports filtering, sorting, and pagination on in-memory data
- Ideal for prototyping, demos, and testing admin interfaces

## Pricing

Free and open-source as part of the React Admin ecosystem.
