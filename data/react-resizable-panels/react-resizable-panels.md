## Overview

React Resizable Panels provides resizable panel group components for React applications. Panels can be arranged horizontally or vertically with draggable handles to adjust sizes. Built by Brian Vaughn, creator of React DevTools.

## Features

- Horizontal and vertical layouts
- Collapsible panels
- Minimum and maximum size constraints
- Default size configuration
- Persistence (localStorage/sessionStorage)
- Controlled and uncontrolled modes
- Keyboard navigation
- Custom resize handles
- Auto-save layout state
- Conditional rendering
- Nested panel groups
- TypeScript support
- SSR compatible
- Accessibility support
- Smooth animations

## Components

- PanelGroup: Container for panels
- Panel: Resizable content area
- PanelResizeHandle: Drag handle between panels

## API

- Imperative panel API (resize, collapse, expand)
- onLayout callback for size changes
- Storage adapter for persistence
- getId for multiple instances

## Use Cases

- Code editors (file tree + editor)
- Email clients (sidebar + content)
- Admin dashboards
- Documentation sites
- IDE layouts
- Multi-pane viewers
