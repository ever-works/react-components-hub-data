## Overview

Frimousse is a lightweight, unstyled, and composable emoji picker component for React. Originally created for Liveblocks Comments, it's particularly well-suited for projects using shadcn/ui and offers complete control over styling.

## Features

- Completely unstyled (headless)
- Composable architecture
- Lightweight bundle
- TypeScript support
- Emoji search functionality
- Category navigation
- Skin tone selection
- Recent emojis tracking
- Keyboard navigation
- Accessible by default
- Works with shadcn/ui
- Custom styling freedom
- Modern emoji support
- Performant rendering

## Architecture

- Modular component design
- Bring your own styles
- Flexible composition
- React hooks integration

## Use Cases

- Comment systems
- Chat applications
- Social media apps
- Messaging platforms
- Feedback widgets
- Content editors
