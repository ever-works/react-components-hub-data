## Overview

PBandJ is a zero-configuration framework for building reusable React components with simplified build tooling.

## Features

- Zero-configuration setup for component libraries
- Automatic bundling with tree-shaking support
- Built-in Storybook integration
- TypeScript support with type definitions
- Multiple output formats (CJS, ESM, UMD)
- Hot module reloading during development
- No webpack or Babel configuration needed
