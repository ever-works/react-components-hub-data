## Overview

React Spaces provides a collection of layout components that create spaces with unique capabilities for building complex user interfaces.

## Features

- Scroll space for scrollable regions
- Resize space for resizable regions (drag to resize)
- Fill space for filling available space within a container
- Anchor space for anchoring elements to edges
- Center space for centering content
- View space for sticky or fixed position elements
- Nestable spaces for complex layouts
- No CSS frameworks required
