## Overview

Unovis is a modular, framework-agnostic data visualization framework that works with React, Angular, Svelte, and vanilla TypeScript or JavaScript. It provides a set of composable visualization primitives powered by D3 to build custom, data-driven visualizations. The framework is designed for flexibility and performance across different web frameworks.

## Features

- Works with React, Angular, Svelte, and vanilla TS/JS
- Modular architecture for building custom visualizations
- Powered by D3 for accurate data-driven rendering
- Reusable chart components (axes, series, legends, annotations)
- Responsive design
- Interactive features: tooltips, selection, hover
- Composable primitives for custom layouts
- TypeScript-first development
