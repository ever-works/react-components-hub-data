## Overview

SweetAlert2 is a beautiful and feature-rich replacement for JavaScript's alert, confirm, and prompt dialogs. It provides an elegant way to show notifications, confirmations, and input prompts with extensive customization options.

## Features

- Beautiful default styling
- Fully responsive design
- Accessible (WAI-ARIA compliant)
- Customizable appearance
- Multiple input types
- Async/await support
- Promise-based API
- Queue system for multiple alerts
- Toast notifications
- Timer-based auto-close
- Confirmation dialogs
- Input validation
- Custom HTML content
- Icon types (success, error, warning, info, question)
- Progress steps
- Custom animations
- Keyboard navigation
- React integration with sweetalert2-react-content
