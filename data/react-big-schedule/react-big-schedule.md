## Overview

React Big Schedule is a comprehensive scheduling component that provides calendar and timeline views for managing events and resources. It's designed for applications that need powerful scheduling capabilities like booking systems and resource management.

## Features

- Multiple view types (Day, Week, Month, Year, Quarter, Custom)
- Timeline view for resource scheduling
- Drag and drop events
- Resize events
- Create events by selection
- Resource scheduling
- Event grouping by resources
- Recurring events support
- Custom event rendering
- Custom cell rendering
- Conflict detection
- Event overlap handling
- Timezone support
- Non-working hours
- Custom time ranges
- Event popover/tooltip
- Responsive design
- Keyboard navigation
- Internationalization

## View Types

- Day view - hourly schedule
- Week view - weekly calendar
- Month view - monthly calendar
- Custom views - define your own ranges
- Timeline view - resource scheduling

## Resource Management

- Custom resource rendering
